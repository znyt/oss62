　　苏重艰难睁开眼皮，一阵浑身酸痛传来。
<br/>　　“又是个病秧子？胖猫这是不打算混了吧，怎么又找了这么个倒霉寄主！”苏重使劲摇了摇头，让自己更清醒一些。
<br/>　　现实世界里，他搬到小院内后，山上别墅就开始建造。本来工程量就不小，加上苗家村地处偏僻运输困难。等彻底完工，苏重再布置好，已经过去一年多。
<br/>　　苏重辛苦赚来的本源，已经没剩多少。他不得不再次开始穿越之旅。
<br/>　　剩余的本源少，苏重只能节省着用，没做仔细筛选，找了个比较容易破界的位面，就直接过来了。
<br/>　　“果然，便宜没好货啊。”来都已经来了，他只能接受这个开局，从床上爬起来，苏重打量四周。
<br/>　　这是一间五十平米左右的单身公寓，家具简单，摆放整齐，房间内颇为干净整洁。原主人生活习惯不错。
<br/>　　走到镜子前，一个头发乱糟糟眼眶发黑，脸色苍白的小伙出现在眼前。
<br/>　　“嗯，普通人一枚。”
<br/>　　“浑身酸痛发热，发高烧把自己烧死啦？”也算是奇葩了。
<br/>　　“等等……浣熊市是什么鬼？”残留记忆在苏重脑海中一闪而过。
<br/>　　他有种不祥的预感。
<br/>　　打开电视，连续换了几个台，然后果然发现了本地浣熊电视台。然后又在一些节目中发现了安布雷拉公司的踪迹。
<br/>　　“日前安布雷拉公司发出公告，公司在生物化学领域取得新成就，从基因层面着手，有望治愈癌症、艾滋等世界顽疾……”报道中的记者满脸潮红，眼里全是兴奋。
<br/>　　苏重却浑身发凉。
<br/>　　保护伞公司确实能治疗顽疾，把人杀了，不就没有顽疾了！这特么一死死一球的生化危机啊！
<br/>　　“他们这是打算用生化病毒进行灭世计划啊！”他看过生化危机所有系列，当然知道幕后黑手的目的。
<br/>　　也不知道怎么搞得，好多西方电影，总是喜欢灭世计划。把普通人杀掉一批，然后留下精英们重新分配资源统治这个世界，简直神经病。
<br/>　　“原主人不是发高烧烧死的，而是被病毒杀死的！”苏重惊觉。
<br/>　　身在浣熊市，死前又发着高烧，这不明摆着被病毒感染了吗。
<br/>　　“底下蜂巢，估计也已经死干净了。”苏重眉头紧皱。
<br/>　　他不是本世界的人，死了也没什么，大不了灵魂跑回去就好。
<br/>　　但他本源不多，必须在这个世界有所收获。
<br/>　　“不管是想让原主人活下去，还是掠夺本源，都必须去一趟蜂巢。”苏重下定决心。
<br/>　　“蜂巢里充满丧尸，空手去肯定不行，要带着武器。”
<br/>　　熟练从床头柜子里摸出一把手枪，把玩了一会儿他又放了回去。
<br/>　　“这玩意子弹有限，而且声音还不小。杀不了几个丧尸，就会引来更多，划不来。”
<br/>　　围着屋子转了几圈，从床底下拉出一个工具箱，看到箱子里手指长的钢钉，苏重眼睛一亮。
<br/>　　苏重不仅仅精通枪械，他对于投掷暗器同样在行啊。
<br/>　　而且钢钉容易携带，找个背包就能背上数百根，足够苏重使用。
<br/>　　“只有二十多根，不够用，必须再找一些。”
<br/>　　找了个背包把钢钉转进去，想了想又把床头手枪别进裤腰带里，带上一顶鸭舌帽，苏重大步走出家门。
<br/>　　五金店，苏重把枪啪的一下拍在桌子上。
<br/>　　“把你们店里所有的钢钉都给我搬过来，快点儿！”
<br/>　　五金店老板被苏重搞懵了，还从没见过抢劫钢钉的。
<br/>　　老板顺从的抱出五大盒子钢钉，怎么看也得有一千四五百根。吃力的办到柜台前，噗通一声扔在桌子上，然后举着双手老老实实的靠墙站，动作熟练地很。
<br/>　　“小王八羔子，这么一大坨钢钉，我看你怎么拿得动！”老板脸上笑眯眯，心里mmp。他一个五大三粗都拿着费劲。苏重脸色苍白，一副瘾君子模样，他就等着苏重出丑。
<br/>　　然后就看到苏重把沉重钢钉扫进帆布背包，一手提着背包，一手拿着枪，轻轻松松风一般的冲出了店门，微风吹起牛仔夹克，酷的一塌糊涂。
<br/>　　店长：“……”至于吗，抢个钢钉而已，怎么弄的和拍电影的似的。
<br/>　　生化危机苏重已经看了好多年，早就不记得浣熊市蜂巢具体位置。
<br/>　　蜂巢底下虽然庞大，地上入口却只是一座别墅。
<br/>　　好在浣熊市并不大，豪华住宅区都在同一片街区。转一转找找，总能找到蛛丝马迹。
<br/>　　打车赶往目的地，苏重下了车，斜挎着单肩包就开始溜达。
<br/>　　这种私人住宅区是不让外人进入的，门禁森严，四周不时有训练有素保镖巡逻。
<br/>　　但这难不倒苏重，三两下翻过围栏，苏重就开始大摇大摆的在内溜达。慢慢打量这些风格迥异别墅，猜测那一座才是蜂巢入口。
<br/>　　沿途巡逻的保镖，不等他们出现，苏重就提前发现，照面都没打就躲了过去。
<br/>　　至于监控，苏重一副悠闲模样，监控室里的人，哪能确定他是不是原住民。
<br/>　　转了几圈没找到目标，不过苏重却发现了一个有意思的人。
<br/>　　一个和他一样翻墙的家伙。
<br/>　　苏重站在围栏下边，静静地看着对方坑坑吃吃的翻墙。
<br/>　　马克已经不是第一次翻墙进入这里。以前也被当做小偷抓过几次，但他仍然乐此不疲。
<br/>　　他是一个公益组织成员，接到任务，调查安布雷拉公司秘密基地，找些黑料给他们捅出来。
<br/>　　他的妹妹就是保护伞公司雇员，所以他才能准确知道保护伞底下基地入口。
<br/>　　但保护伞基地入口伪装太好，就是这个住宅区，他都不容易进来。
<br/>　　这里住的人非富即贵，安保力量强悍的一塌糊涂。
<br/>　　好不容易翻墙进来，吗深吸一口气。一转身，那口气还没吐出来就给憋了回去。
<br/>　　不知道什么时候，他身后竟默默地站了个亚裔青年！
<br/>　　“不会这么倒霉吧，刚进来就被人给发现啦！难道这次又要无功而返？”
<br/>　　“姓名！”对面青年冷不丁问。
<br/>　　“马特，马特·艾迪森。”
<br/>　　“这片儿地方是老子先盯上的，拿了东西，必须分我一半！”
<br/>　　马特：“……”这是把自己当盗窃同行啦？
<br/>　　“好的，给你五成！”马特答应的毫无压力，他又没打算偷。
<br/>　　“不行，你答应的没诚意，我要跟着你，看你偷！”苏重面无表情。
<br/>　　马特：“……”入室盗窃敲诈勒索都这么明目张胆的嘛？
