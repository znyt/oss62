　　方牧南跟云悠说话的时候，眉宇间都透露着慈祥，此时的方牧南就像是彻底变了一个人一样，完全没有“琴魔”的凌厉，更像是看到孙女时，慈祥的爷爷。
<br/>　　“因为、因为……”
<br/>　　云悠支吾了一下，随后声音弱弱的说道：“因为我不喜欢这里……
<br/>　　果然！
<br/>　　这件事情还是在云悠心中留下了阴影，让她对红旗学校产生了排斥。
<br/>　　沈琛眉头紧皱。
<br/>　　云悠这种从小就有强大空间思维能力天赋的孩子，实在太少了，如果能够培养起来，她所代表的价值几乎可以用美国人当年对钱学森的评价，“价值五个师”！这样的人才绝对是千金不换的！
<br/>　　如果让云悠离开红旗学校，对她的未来是不会有什么影响，但是对这个国家而言，绝对是一个巨大的损失。
<br/>　　云悠平时身上都会带一个魔方，这些魔方不仅是常见的正方体，还有圆球形的，锥体的，云悠在去找方牧南样的那只猫玩的时候，有时就会把魔方拿出来拧着玩。
<br/>　　方牧南这样的人，自然是第一时间就发现了云悠的能力，还把她带去自己做的沙盘前，跟她做“游戏”，考验她的空间思维能力，结果却让方牧南感觉深深的震惊和意外。
<br/>　　当时，方牧南就确定，这个孩子如果好好得到培养，她绝对会是一把无形的利刃，令所有敌人都为之胆寒，甚至她的一个名字，就能让敌兵后退千里！
<br/>　　沈琛有些焦躁的在办公室里走动，让云悠离开，这简直就是在沈琛心头插刀啊，而且还不是一刀，而是两刀！
<br/>　　为什么是两刀？因为如果云悠离开红旗学校的话，罗战在很大概率上也会离开，这家伙被老校长带来的时候，就一脸不情不愿，现在他姐姐跑了，他如何还愿意再留下来？
<br/>　　老校长可是费了很大的劲儿才把这两个小家伙给带进了红旗学校，等到老校长回来，发现这两个宝贝全都跑了，然后再了解事情经过，非得把自己活拆了不可。
<br/>　　方牧南思考了一下，对沈琛说道：“小沈，以后就你来教这丫头，你那点干货也该拿出来，别总是藏着掖着，这么好的苗子错过了可就没了，平时让蓝丫头多关照一点，然后提防着诸磊那小子，别让他再给我坏事。”
<br/>　　“方老，这您可以放心，诸磊那小子我保准让他翻不起一点浪花来。”沈琛向方牧南保证道。
<br/>　　方牧南指着沈琛对云悠问道：“小丫头，你以后就跟着他学习好不好？”
<br/>　　云悠看了看沈琛、又看了看自己妈妈最后还是点了点头。看到云悠点头，沈琛总算是松了一口气，只要这丫头不走，罗战也肯定能留下来。
<br/>　　方牧南看了眼罗战，随后对罗昊说道：“看得出来，天宇对这小子很上心，底子打得不错，好钢也要好的铸造师用心去打造，这样才能锻造出利刃。这小子，后面就跟着我学习。”
<br/>　　听到方牧南的话，不管是罗昊还是云熙，心中都是狂喜，得到方牧南的指点绝对是可遇不可求的事情，因为方牧南身体的原因，他很少过问红旗学校的事情，只是自己一个人在他那个小院子里种种花，养养猫，从蛰龙之后，方牧南就再也没有出山过，以致于现在在红旗学校的学员，很少有人知道这个“糟老头子”的存在。
<br/>　　红旗学校里那些学员不知道，办公室里这几个人能不知道吗？他们太清楚“琴魔”这两个字意味着什么了，如果不是方牧南不愿意，每天来他那个小院子看望他的将军，都能把小院子的门槛给踏平了。
<br/>　　虽然方牧南没有正式收徒，但是那几个军区特战大队的狙击总教官看到方牧南，哪个不得恭恭敬敬的叫他一声“老师”？
<br/>　　站在罗昊身边的罗战，眼珠子“咕噜噜”的一转，随后看着方牧南问道：“方爷爷，您是我师公的老师，那就是我爸爸的师公，我现在做你的学生，那辈分是不是就比我家老头子大了？”
<br/>　　罗昊伸手在罗战额头上弹了一下说道，“你小子，这就想着占你老子便宜了？”
<br/>　　罗战捂着额头，看着罗昊嘀咕道：“你这是摧残祖国的花朵！”
<br/>　　“就你还花朵？你脸皮还能不能再厚一点？”
<br/>　　“能啊，谁让我是你儿子呢，亲生的！”
<br/>　　“……”
<br/>　　办公室中，众人一阵哄笑。
<br/>　　……
<br/>　　红旗学校，战术训练场外，罗昊和方牧南沿着小道慢慢走着。
<br/>　　方牧南佝偻着身体站在罗昊身边，比起罗昊矮了不止一点半点，周围偶尔会有红旗学校的教员走过，看到两个人都会停下脚步，站直身体然后敬礼。罗昊清楚，那些教员朝着自己这边敬礼，不是给自己，而是给自己身边这个老人。
<br/>　　罗昊原本以为，蛰龙是方牧南最后一个学生，蛰龙之后方牧南再也不会出山；可是没想到自己的儿子居然会有这样的好运！
<br/>　　“你肯定在想，为什么我会指点你儿子是吧？”方牧南似乎早就猜到了罗昊心中的疑惑，说道：“在带着蛰龙从红旗学校离开的那天，我说过，他是我最后一个学生，蛰龙之后再也不收学生！这句话不是开玩笑，所以你家小子不能算是我的学生，我只是指点他而已。我这一辈子收了三个学生，一个是你师父，一个是飞扬，还有一个就是蛰龙。三个人各有各的优点，但要说谁是我最满意的学生，毫无疑问，是蛰龙。”
<br/>　　“狙击手，被称为‘战场刺客’，他们充满着神秘、行踪隐秘、来无影去无踪，千米之外取敌酋首级。所以，从‘刺客’角度来说，你、你师父都是不合格的，飞扬算是一个‘刺客’，但他不够完美，只有蛰龙才是我想要的‘刺客’，即便是没有了狙击步枪，蛰龙也能顺利完成刺杀。”
<br/>　　“那您想要让我儿子超越蛰龙？”罗昊停下脚步，看着方牧南问道。
<br/>　　“超越？”方牧南轻轻摇头，回答道：“超越不了。”
