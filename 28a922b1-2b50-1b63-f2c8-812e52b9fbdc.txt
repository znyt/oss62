　　“我不要自爆！不要！楚风眠你放过我，我愿意认你为主！”
<br/>　　小耀王寒夜身躯已经是被楚风眠控制住，意识还清醒，看着楚风眠急忙的讨饶道。
<br/>　　他刚刚得到奇遇，实力更进一步，哪怕是失去了霸者武道，他也比起之前还要更加的强横。
<br/>　　他的未来，可能是有着机会去冲击半步天人境的，他可不想死。
<br/>　　“认我为主，就你这个废物还没有这个资格，乖乖自爆，算是你唯一有用的一件事了！”
<br/>　　楚风眠斩钉截铁的说道，丝毫不考虑小耀王寒夜的话。
<br/>　　他本来是想要降伏一些高手，去剑道门镇守的，可是这小耀王寒夜，阴险无比，三番两次的算计楚风眠。
<br/>　　把他留下，哪怕是刻下诸天生死印，都是一个祸害，楚风眠自然不可能将这样一个祸害留下。
<br/>　　这小耀王寒夜现在唯一的一点作用，便是去自爆！
<br/>　　楚风眠冷笑着，一边将这小耀王寒夜之中的全部力量催动而出，燃烧着他的精血，爆发出最为强横的力量来，向着转轮和尚疯狂的冲了过去。
<br/>　　“不！不！楚风眠，你大逆不道，你敢杀我！圣院宗的强者不会放过你的！我是师傅可是呼延武尊，他一定会杀进了你的！”
<br/>　　听到楚风眠无情的回答，小耀王寒夜大骂道，同时也向着那转轮和尚乞求道。
<br/>　　“转轮兄快救我，我师尊是半步天人境的强者，救了我，肯定给你大量的好处，你想要什么我师尊都会给你的！快救我！”
<br/>　　现在这小耀王寒夜，唯一是救命稻草，便是转轮和尚了。
<br/>　　上一次他差一点死在了楚风眠的手中，就是被转轮和尚所救，这一次更是讨饶道。
<br/>　　但是现在的转轮和尚，已经是自身难保，看着逼近的小耀王寒夜，脸色更是难堪无比。
<br/>　　一位九劫古帝巅峰级别的武者自爆，比起许多半步天人境强者的全力一击还要恐怖。
<br/>　　这种力量，几乎是他无法对抗的。
<br/>　　如果不是因为他还想要得到那血邪魔帝骸骨，现在早就施展出秘术逃走了。
<br/>　　“耗费了玉符，难道是要一无所获？”
<br/>　　转轮和尚不甘心的想到，但是他还没有来得及想完，那小耀王寒夜已经是逼近了他的身躯。
<br/>　　轰隆隆！
<br/>　　一股股狂暴的力量，已经是从小耀王寒夜的身躯之上喷涌而出。
<br/>　　在逼近了这转轮和尚的一刻，这些狂暴的力量，同时爆炸开来。
<br/>　　“轰隆！”
<br/>　　这一次爆炸的威力，比起那少帝施展出的天地崩塌，还要恐怖。
<br/>　　一尊九劫古帝巅峰人物，在这一刻将所有的力量，同时爆发出来，爆发出的恐怖威力来，足以席卷周围的一切。
<br/>　　周围许多九劫古帝巅峰的魔族，看到这一幕，他们的心中除了恐惧，更是冰冷无比。
<br/>　　九劫古帝巅峰的实力，不管是在哪一方星域，什么种族之中，都是绝对的强者了，未来一旦是踏入半步天人境，更是会成为一方巨头的存在。
<br/>　　可是这小耀王寒夜，就这样硬生生被楚风眠控制的自爆了，毫无反抗之力。
<br/>　　这让那些九劫古帝的几位魔族，看向楚风眠的眼神之中，已经是尽是恐惧，根本不敢来插手。
<br/>　　巨大的爆炸，震荡这整个大雄宝殿，毁灭狂暴的力量席卷着周围的一切。
<br/>　　那转轮和尚第一个被席卷在其中，他也知道如今已经没有给他考虑的机会了。
<br/>　　一道道耀眼的佛光，在这爆炸中心冲天而起，转轮和尚打出了一道道的秘术，硬生生是从这爆炸中心，撕裂开了一道裂痕来。
<br/>　　只见一道佛光，从中飞出，一下子离开大雄宝殿，消失的无影无踪。
<br/>　　“这自爆，也没有能够困住这转轮和尚，此人身上的秘术太多了。”
<br/>　　看到这一幕，楚风眠感叹一声，他控制住小耀王寒夜，也打算是要让小耀王寒夜以自爆之力，去轰杀了转轮和尚，最好是将转轮和尚轰击到重伤的地步，楚风眠在去出手，擒下转轮和尚来。
<br/>　　不过看来楚风眠还是小看了转轮和尚，哪怕是小耀王寒夜的自爆，也没有能真正威胁到他，只是将他逼退了。
<br/>　　看着那远去的佛光，楚风眠也没有追逐的打算，而是身躯一闪，向着那王座的方向冲了过去。
<br/>　　这血邪魔帝骸骨之上的太初剑锋，还有那一枚人皇罐碎片，才是楚风眠如今最想要得到之物。
<br/>　　“给我收取！”
<br/>　　楚风眠的手心之中，一枚古老瓦罐，突然显现而出，正是人皇罐。
<br/>　　如今的人皇罐，已经是融入了其中六枚碎片，看起来已经不再是那么的破碎了，而只是有着几分残破。
<br/>　　上面刻画的纹路，也恢复了许多，已经是向着曾经人皇手上的第一利器恢复了。
<br/>　　这人皇罐一出，在镶嵌在血邪魔帝胸口之上的人皇罐碎片，突然也感受到了力量，想要飞出。

<br/>　　楚风眠眉头一皱。
<br/>　　刚刚不灭战魂的确死了，但是并不代表这血邪魔帝身躯之上的力量消散。
<br/>　　这人皇罐碎片，已经是镶嵌在血邪魔帝的胸口之上，数十万年的时间了，已经成为了血邪魔帝肉身的一部分。

<br/>　　那转轮和尚一逃，那一枚仙人境玉符唤出的佛陀虚影，力量也在迅速的衰弱，楚风眠可不能在浪费时间。
<br/>　　两道剑锋同时斩落而下，锋利的剑气，撕裂一切，一层层的魔意凝聚出的屏障，都被无情的切碎。
<br/>　　随着楚风眠手心猛然一抓，那一枚人皇罐碎片，也从血邪魔帝胸口处，浮现了出来，飞到了楚风眠的手心之中。
