　　罗昊跟布谷交手的时候，莱昂、天蛇、玄龟也是从丛林中走了出来，站在一旁看着在空地上交手的两个人。
<br/>　　拳影、腿影、刀影交错闪烁，龙牙军刀和FoX战术直刀不断的碰撞，发出“叮叮当当”的声响。
<br/>　　安全屋中，其他战术中队队员也是察觉到外面发生的打斗，全都拿着武器冲了出来；当看到站在一旁的莱昂时，又全都松了一口气，把视线看向还在打斗中罗昊和布谷。
<br/>　　“这小子身手不错，反应、速度都不错。”
<br/>　　孙涛盯着布谷看了一会儿，说道：“可是他毕竟还是年轻了一点，经验上比起狂龙还差了一点，最多还有三招，他就会败在狂龙手中。”
<br/>　　果然，就如孙涛所说的那样，罗昊卖了一个破绽给布谷，布谷抓住“机会”想要追击；却是被罗昊用龙牙军刀顶在了胸口。
<br/>　　“小子，你输了。”
<br/>　　罗昊手中的龙牙军刀已经是顶在了布谷的胸口，但是布谷手中的FoX战术直刀距离罗昊的脖子还有五公分左右的距离。
<br/>　　五公分，足以用来作为决定生与死的距离。
<br/>　　布谷用些泄气的收回了自己手中的刀，收回到刀鞘中，说道：“本来还对北极狼和老师的话有些怀疑，现在看来他们对你的评价还挺中肯的，你果然很厉害。”
<br/>　　“你的老师是？”罗昊有些疑惑的看着布谷，问道。
<br/>　　“夜鹰。”布谷回答道：“蔷薇算是我的师姐，准确的来说，我应该叫你一声‘师姐夫’。”
<br/>　　“那你张口闭口的‘大叔’？”
<br/>　　罗昊瞪了布谷一眼，夜鹰学生的身份，迅速把两个人之间的关系拉近了，这小子不仅是云熙的师弟，跟自己也算是师出同门，都算是方牧南的徒孙。
<br/>　　布谷嘿嘿一笑，说道：“现在90后都算是‘中年’了，更何况是你这个年纪呢？这可是‘官方说法’！”
<br/>　　“刚才在树林里露头的人也是你？”罗昊对布谷问道。
<br/>　　“我堂堂夜鹰的学生，怎么可能犯这么低级的错误？”布谷不满的皱眉，指了指玄龟说道：“是玄龟，我如果犯这么低级的错误，那岂不是给我老实脸上抹黑吗？”
<br/>　　“狂龙好久不见！”
<br/>　　莱昂这个时候也是带着天蛇和玄龟走了过来，跟罗昊拥抱了一下。
<br/>　　“看到你这个老家伙，我心中立刻就踏实多了。”罗昊说道：“本来还想着北极狼这家伙就派四个人来，这也实在太抠门了。”
<br/>　　“这次可不是我带队。”莱昂指了指布谷，说道：“这次是布谷鸟带队，就连我这个老家伙也得听他的。”
<br/>　　罗昊眨巴着眼睛，嘀咕道：“我怎么觉得被北极狼那货给坑了？”
<br/>　　“阿嚏！”
<br/>　　远在法兰西国的云凌风打了一个响亮的喷嚏，被杨晓婷推到了一边，不满的看了他一眼，说道：“也不知道注意点，把感冒传染给你儿子了怎么办？”
<br/>　　“我没感冒。”云凌风揉了揉鼻子，嘀咕道：“肯定是有人在骂我，妈的，一定是狂龙那货，除了他，也不会有人在背后骂我。”
<br/>　　“你不要看不起人。”布谷对罗昊的态度很不满，说道：“我这次过来可是还给你带了一份大礼的，保准你满意。”
<br/>　　安全屋中，布谷把通过无人机拍摄到的视频播放了以后，对罗昊说道：“怎么样，对我的见面礼还满意吗？”
<br/>　　“这东西倒是给我们解决了一个大麻烦。”
<br/>　　梅庆伟沉吟道：“这个巴坦伦是这里的实权派，而且跟布克尔特关系密切，根据我们的情报，这家伙每年从布克尔特手里拿的好处可是个大比例，而且他这次参加大选，背后布克尔特可是给了他很大的支持，如果布克尔特受到打击对他的影响会非常大。为了大选成功，他不可能看着自己的‘金主’倒掉，一旦我们动手，巴坦伦很有可能会派兵给布克尔特支援，到时候，随便给我们头上按一个罪名，我们根本没有任何办法。”
<br/>　　“可是有了这个东西，情况就不一样了。”
<br/>　　罗昊点开一张照片，说道：“这个瓦斯汀是巴坦伦这次大选最大的竞争者，把这个东西交给瓦斯汀，他应该会知道怎么做。”
<br/>　　“这其中有个情况，不知道你们想过没有？”莱昂双手十指交错着，看了眼房间里的所有人，最后把目光停留在瓦斯汀的照片上，说道：“这个瓦斯汀的确是巴坦伦大选最大的竞争者，不出意外，下一届的总统就是在他们两个人中选出；瓦斯汀不可能不知道巴坦伦那些事情，可是他为什么没有曝光出来？”
<br/>　　“他没有掌握到实际证据。”天蛇说道：“你也看到巴坦伦那个秘密别墅了，那里的警戒程度非常高，想要得到实质性的证据可不是一件容易的事情。”
<br/>　　“这是原因之一，可是就算得不到别墅内的证据，从远处拍到这栋别墅也能制约巴坦伦，毕竟以巴坦伦在联合政府军的收入，绝对不可能买下这么豪华的别墅，这只要有点脑子的人都知道，但瓦斯汀为什么没有说？”莱昂问道。
<br/>　　“因为他不敢！”
<br/>　　罗昊接口道：“表面上看，瓦斯汀的确是巴坦伦最大的大选竞争对手，但是两个人完全不是一个层面的竞争实力。巴坦伦不但背后有布克尔特的经济支持，他手中还有军队，这才是巴坦伦最大的底牌，瓦斯汀根本无力跟巴坦伦抗争。”
<br/>　　“那就是说，这个瓦斯汀就没什么用了？”梅庆伟有些失望。
<br/>　　“那倒也未必。”罗昊摇头道：“要想靠这段视频彻底扳倒巴坦伦，基本是不可能的事情，但是用来牵制住他，给我们的营救争取时间，应该是足够了。”
<br/>　　“可是这个瓦斯汀真的敢跟巴坦伦叫板吗？”陈应天问道。
<br/>　　“他没有选择。”罗昊回答道：“他现在已经站在了巴坦伦的对立面，一旦巴坦伦在大选中胜出，你觉得巴坦伦第一个要对付的人是谁？”
<br/>(本章完)
