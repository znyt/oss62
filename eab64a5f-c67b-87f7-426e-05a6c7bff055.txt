
<br/>　　<br/>　　    雾灵山的事很快就解决完了，有午桥这个出马行家在场，自然没啥难度，为陈臣开了香堂以后，从此她就得在山上的庙里做三年堂了，而要不了多久，雾灵山方圆百里左右，就会传开此地有个看事极准的神婆，只是估计没人会想到，这个神婆会是个千娇百媚的姑娘。<br/>　　<br/>　　    午桥开了香堂后的当天就离开了京城返回东北，只剩下了王惊蛰，还有个无处可去的陈三岁，他曾经问过对方有啥打算，陈三岁说以前一直想着要为诗远还魂，为此他努力了多年，这也是迄今为止能唯一支撑着他干下去的一件事，可诗远如今已经被葬了，他忽然发现自己的心里就空落落的了，东南西北上天入地，他都不知道要何去何从了。<br/>　　<br/>　　    “我呢左右也都是一个人，也是东奔西跑的居无定所，正好我过几天还有个事要去处理，你要实在没啥可干的，可以跟我走一趟”<br/>　　<br/>　　    “行！”<br/>　　<br/>　　    王惊蛰诧异的问道：“你都不问我干什么去嘛？”<br/>　　<br/>　　    陈三岁淡淡的说道：“对于我来说，诗远走了寄托也没有了，人还在心已死，去哪里都无所谓，也许有一天我实在厌倦了，就去抚仙湖旁盖一栋房子，常伴在诗远的坟前，了去残生得了，反正我这人也没啥追求，在哪里都一样，我欠你的人情总归得要还了，你说让我去哪我就去哪”<br/>　　<br/>　　    “你是从石头缝里蹦出来的？你爹娘呢……”<br/>　　<br/>　　    “早没了，只有我和诗远相依为命”陈三岁落寞的说道。<br/>　　<br/>　　    “你也是个苦命的人啊”王惊蛰感慨着说道。<br/>　　<br/>　　    接下来两天，王惊蛰紧锣密鼓的在京城安排着见人，先是跟常文书见了一面，确定了下宗教局的审批还有地皮的事，对方的回复是，差不多半个月左右就能落实了，这件事没啥难度，于此同时这时正好是三月份，范二哥要去油菜花开满地的上饶赴任了，正直临走之前知道王惊蛰也来了京城，就特意约他出来坐下聊了聊。<br/>　　<br/>　　    席间，范二哥总是不死心的旁敲侧听，想要跟王惊蛰打听下关于自己的仕途之路，但每次他询问都被王惊蛰两句话给带跑了，他压根就不想指点这个，问了三次以后范二哥也死心了，但最后临散席之前，他拉着王惊蛰的手语重心长，十分不要脸的说，指点一二你不给指点，那我以后有难事找到你头上的话，总归得能帮个忙吧？<br/>　　<br/>　　    王惊蛰被他缠的脑袋“嗡嗡”指疼，就值得点头答应了，范二哥这才欢天喜地喝的醉醺醺的迈着四方步走了。<br/>　　<br/>　　    对于他们这种人来说，有些事不能强求的话，那就得换个循序渐进的策略了，只要他能一直和王惊蛰有联系不断了关系，那以后自然有都是方式接触，怕就怕对方不搭理他。<br/>　　<br/>　　    又过了一天，各种事都安排完了，王惊蛰也要离京的前夕才找上了黄九郎，两人约在了贺三刀的宝瑞福茶楼见面，于情于理王惊蛰都得跟他面见一回，黄九郎这人太会做人了，八方风雨不透的处事方式，让人挑不出一点毛病来。<br/>　　<br/>　　    一壶虎泡老茶，两人对坐，开始寒暄叙旧了几句，都是性格干脆利索的人，也没什么客套话，黄九郎就问到了正题上。<br/>　　<br/>　　    “你要寻那丹方上的两味药？一是龙骨草，二是丹青芽，对吧？”<br/>　　<br/>　　    “是！”王惊蛰这次没隐瞒，直接点头认了。<br/>　　<br/>　　    “你呢也不用奇怪和怀疑，这个消息走漏跟玄门的唐禾祥真一点关系都没有，原因出自你让常文书四处找人翻译那个甲骨文书上了，毕竟在那之前我就一直留意上你了，只是还没有接洽的机会，等你和唐禾祥见完面要离开京城的时候，我这才特意去了车上找你”黄九郎端着茶杯，抿了一口后，接着说道：“我这个部门想查什么自然不难，你让人翻译的丹方，除了你自己手上那份，唐禾祥那里一份外，我这里同样也有，一个字都不差”<br/>　　<br/>　　    王惊蛰叹了口气，揉着脑袋说道：“你让我想起了东厂的那帮厂公们啊，真是哪哪都无孔无入呢”<br/>　　<br/>　　    “哈哈，你这个比喻不算恰当可也挺贴近的了，论查消息上，我们部门确实挺有可圈可点之处，我先问问你打算先去往哪里啊？”<br/>　　<br/>　　    王惊蛰寻思了下，其实他也没太确定是先去有龙骨草的雪域高原还是去往内蒙外蒙那一片草原寻找丹青芽，对于他来讲这两个地方都得去，随便哪都可以。<br/>　　<br/>　　    见他没有回话，黄九郎就说道：“从我个人的建议角度，我觉得你最好先去高原那边，你这时候启程一路寻过去，再有两个月天气就该暖和起来了，而你走这一趟的话时间段了两三月，长了的话四五月也有可能，所以呢时间上来说先去这比较合适，至于草原那边的话，你随便什么时候过去都行”<br/>　　<br/>　　    王惊蛰斜了着眼睛问道：“你这个个人的建议背后，是不是有什么事等着我呢？”<br/>　　<br/>　　    黄九郎顿时哈哈一笑，手指点着他说道：“你的心思太贼了，别把人老往歪了想，我跟你说过真要是有什么事的话我肯定不会藏着掖着直接就阐明了”<br/>　　<br/>　　    “那我也是抱着半信半疑的态度”王惊蛰点了点头，随即又问道：“你让我来京城找你，不会就是随便闲聊这点事吧，你这日理万机的忙人，哪里会有这种闲情逸致啊”<br/>　　<br/>　　    “我确实挺忙的”<br/>　　<br/>　　    “呵呵……”<br/>　　<br/>　　    “还是在你这张方子上”<br/>　　<br/>　　    王惊蛰皱了皱眉。<br/>　　<br/>　　    黄九郎说道：“玄门医字脉历史悠久底蕴深厚，但毕竟还是个民间组织，而我们就不同了，是头顶着官字的，所以从这点上来讲，就注定有些事民间知之不详，可我们却能了解到的挺多的，远了不会说，就说六几年的时候，全国上下扫荡牛鬼蛇神，你知道我们从下面搜上来多少典籍，古书么……”<br/>　　<br/>　　    王惊蛰一听，当即就愣了下。
