        会议结束后，孙莉莉前脚刚进办公室，落以柔后脚就跟了进来。
<br/>
<br/>        傅氏集团总部现任设计部部长，是原来地利置业设计部的副部长孙莉莉，所以，落以柔找上门来说要将童筱颖的名单划掉时，孙莉莉并未同意。
<br/>
<br/>        落以柔刚刚送资料的时候，无意间路过会议室门外，就看到坐在会议室里的童筱颖，心里很是气恼。
<br/>
<br/>        这个女人怎么就阴魂不散？
<br/>
<br/>        童筱颖以前在地利置业工作的时候，也算是孙莉莉的手下，当时也是几番波折，孙莉莉才认可了童筱颖。
<br/>
<br/>        现在童筱颖作为一名澳洲知名大学毕业的建筑设计师，孙莉莉没理由要将童筱颖的名字从这次的设计征选名单中剔除，更何况，童筱颖代表的是夏凌焰设计师的团队。
<br/>
<br/>        落以柔在孙莉莉这里吃了闭门羹，于是搬出了傅夜沉的命令。
<br/>
<br/>        “傅总是对童筱颖下过禁令，不允许她踏进集团半步！”落以柔铿锵有力地说。
<br/>
<br/>        孙莉莉推了推鼻梁上的眼睛，坐在办公桌前，抬眸看着面前“仗势欺人”的落以柔，毫无畏惧道：“落秘书，傅总对童小姐的禁令，一直都是你在以讹传讹，我并未亲耳听傅总说过。其次，我现在是设计部的部长，部长听命于总监，而不是听命于你这么一个特助身边的秘书。虽然，你也是我们傅总面前的大红人，但是，不代表你是一人之下万人之上。”
<br/>
<br/>        落以柔仗着有师父郑忠威撑腰，又得到傅夜沉的重用，在公司里混得很开，除了部长级别以上的管理人员，其他员工几乎是对她都很热络。
<br/>
<br/>        “孙部长，你言重了！我只是跟孙部长提个醒而已，没有别的意思。”落以柔故作爽朗地咧嘴一笑，然后转身离开了孙莉莉的办公室。
<br/>
<br/>        看着落以柔离开后，孙莉莉不禁暗自在心里感慨，这童筱颖也真是的，每次都得罪一些不该得罪的人。
<br/>
<br/>        “呼——还有十五层楼要爬！”童筱颖攀附着楼梯扶手，弯着腰，抹了把额头上的汗水。
<br/>
<br/>        其实，开完会后，她本应该就此离开的。
<br/>
<br/>        但是，她忽然间想去顶楼总裁办看大哥。
<br/>
<br/>        可惜，电梯是刷卡的，除了设计部所在的楼层，其它楼层她去不了，这才不得已选择了爬楼梯。
<br/>
<br/>        嗯，就看一眼，她就走！
<br/>
<br/>        童筱颖心里是这么想的，于是深吸了口气，继续往上爬。
<br/>
<br/>        当她爬到顶楼的时候，她双腿软到站都站不起来。
<br/>
<br/>        童筱颖一路扶着墙好不容易走出楼梯口，刚开防火门，就遇到了一脸凶神恶煞的保安大哥。
<br/>
<br/>        “我、我是来找傅总的！”一个机灵，童筱颖吃力地站直身子，理了理有些褶皱的衣角，一本正经地说。
<br/>
<br/>        保安大哥似乎没那么好骗，凶巴巴地呵斥道：“来找傅总的客人是不需要爬楼梯的！”
<br/>
<br/>        “我锻炼身体啊！”童筱颖干笑。
<br/>
<br/>        “童小姐，请回！”保安大哥一脸严肃道。
<br/>
<br/>        童筱颖忍不住好奇地问：“你、你认识我啊？”
<br/>
<br/>        “童小姐，请回！”保安大哥再次驱赶。
<br/>
<br/>        他之所以认识童筱颖，那还得是落以柔落大秘书拿着照片亲自去他们保安科吩咐的。
<br/>
<br/>        不许童筱颖这个女人踏进傅氏集团大厦！
<br/>
<br/>        听说，是傅总的命令。
<br/>
<br/>        童筱颖抿了抿嘴，眼看着只有几步就能抵达那边的总裁办，却只能悻悻地叹了口气：“好吧！我走就是！”
<br/>
<br/>        果然，她跟大哥的缘分已尽。
<br/>
<br/>        明明近在咫尺，都见不着面。
<br/>
<br/>        “那个……可不可以扶我去坐电梯啊？我脚软……爬楼梯、爬累了……再走下去的话，我的脚都要废了！”童筱颖一脸无辜地看着男保安，顿了顿后，又接着说道，“还有……那个……我有点口渴，想喝水。可不可以‘赏’口水喝？”
<br/>
<br/>        听完童筱颖的这些话，男保安的脸色都黑了：“童小姐是想我叫人来将您扔出去吗？”
<br/>
<br/>        “不用了！我还是自己下去吧！”童筱颖尴尬地笑了笑，不得不转身去下楼梯。
<br/>
<br/>        她突然发现，自己好没骨气啊！
<br/>
<br/>        明明是大哥不信任她，抛弃了她，她居然还想着跑来偷偷去看大哥。
<br/>
<br/>        才下两层楼，童筱颖就真的走不动了，不得不一屁股坐在台阶上，自顾自地捶腿揉脚。
<br/>
<br/>        她这是在自作自受吧！
<br/>
<br/>        而就在这个时候，楼上传来了稀疏的脚步声，还有傅夜沉的谈话声。
<br/>
<br/>        “公司内部人员调动表，明天上班前给我。”傅夜沉的声音有些清冷，不容置喙。
<br/>
<br/>        紧接着是落以柔的回话声：“是！”
