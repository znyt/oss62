　　回到基地后。
<br/>　　林峰去递交任务。
<br/>　　总计三百颗三阶晶石到手。
<br/>　　林峰看了眼排行榜，最高的还是九阶。
<br/>　　而林峰之前的名字都掉到姥姥家去了，一大堆六阶的冒了出来。
<br/>　　七阶的也不像之前那么少了。
<br/>　　不过七阶依然还是前十五的存在。
<br/>　　林峰回到宿舍区报了个平安，一群人在一起吃了顿晚饭后。
<br/>　　翌日。
<br/>　　十个D级空投林峰全部丢入物品栏后，就去看房子了。
<br/>　　房产中心。依然人不少，这实际上和末世之前一样。
<br/>　　末世之前，绝大部分人都说房价太贵了，但总有那么一群人就是有钱，天天去房产销售中心甚至有时候一个新楼盘开售了还会出现排队的现象。
<br/>　　当然了，其中不乏一些商业炒作，但也不乏一些真有钱的人。
<br/>　　眼下也是如此。
<br/>　　虽然绝大部人都拿不出上千颗晶石，但是强大的人不在少数，还是有那么一批人。
<br/>　　林峰来到这里时，就看到几个六阶的高手。
<br/>　　“哎呀亲爱的，这个房子蛮好的，你买给我好不好。”
<br/>　　一个漂亮的女人在一个肥猪男人怀里发嗲，胸部都快要挤爆了。
<br/>　　“小意思的啦。”
<br/>　　肥猪男人哈哈大笑当场就是买了两套房。
<br/>　　林峰瞧了眼苏云，后者倒是一脸的平静。
<br/>　　带苏云过来也是想要让他出谋划策。
<br/>　　最后苏云看中了一套位于南明小区的套房，不大，一百一十平罢了。
<br/>　　这南明小区是个中档小区，但是如今，那就是土豪居住地了。
<br/>　　这个小区内居然有奢侈的人工湖！采光也是极好的。
<br/>　　要知道如今基地被个五十米的墙壁当着，许多地方一天二十四小时都是阴暗的。
<br/>　　在这种情况下还能被阳光照射的房间，那就是奢侈，价格上要更贵一些。
<br/>　　“总价一千五百三阶晶石。”
<br/>　　房产中心的销售人员说道，眼神在林峰和苏云之间扫来扫去。很显然，在她看来，苏云这种大美女肯定是林峰包养的小三了，作为销售人员，不管是末世前还是末世后，他都看惯了。
<br/>　　“一千五百？”
<br/>　　林峰皱起眉头，果然涨价了。
<br/>　　“我有这个玩意可以打折多少。”
<br/>　　林峰问道。
<br/>　　说着拿出了金刀勋章。
<br/>　　“荣誉勋章？”
<br/>　　销售人员楞了一下，说了句稍等就去查这个勋章的价值了。
<br/>　　“这枚荣誉勋章可以给您七折的价格。”
<br/>　　七折？力度很大的折扣了。
<br/>　　可是一千五百乘以七折那也要一千多了。
<br/>　　“好贵啊。”苏云皱了下柳眉。
<br/>　　一千颗普通晶石不算多，但一千颗三阶晶石那可就很多了。
<br/>　　“买不起就滚一边去，穷逼，还想买采光好的房子，穷人一辈子住棚子吧。”
<br/>　　一道嚣张的声音传来。
<br/>　　林峰就看到之前看到的那个大胖子走了过来无比嚣张。
<br/>　　“就是，别打肿脸充胖子。”大胖子的女人也是冷笑道。
<br/>　　林峰呵呵一笑。
<br/>　　“就这套房子了，买了。”
<br/>　　“林峰，我们有那么多晶石吗？”苏云小声道。
<br/>　　“放心吧。”
<br/>　　林峰道。
<br/>　　“装，继续装逼，我就看你怎么收场，是一千多颗三阶晶石，可不是普通晶石，现在的小屁孩，一个比一个会装逼。有些女人啊就是目光狭隘。”大胖子瞥了眼苏云眼中露出一抹惊艳之色，太漂亮了，这么漂亮的女人若是自己能够搞到手玩一玩太爽了。
<br/>　　林峰懒得搭理那个死肥猪只是看着销售人员道。
<br/>　　“你们这可不可以换晶石？我的意思是，高级别的晶石换成三阶的。”
<br/>　　三阶晶石，相当于人名币之中的一百元，普通晶石就相当于十元钱，都有相应的价值。
<br/>　　“自然可以兑换。”销售人员道。
<br/>　　林峰拿出了一颗五阶晶石。
<br/>　　“这东西可以换多少三阶的。”
<br/>　　林峰道。
<br/>　　“一颗五阶可以换两百颗三阶的。”销售人员道。
<br/>　　“呦呵，小子，还有五阶的？哈哈，不会就这么累死累活杀了一个吧？”大胖子戏谑道。
<br/>　　“你屁话怎么这么多啊。”
<br/>　　林峰不耐烦的看了他一眼。
<br/>　　“呦呵，小子，还挺牛逼啊，行，我就在旁边等着，我劝你，买不起这套房子就滚蛋，这套房子，老子看上了！”
<br/>　　大胖子嗤笑道。
<br/>　　“就是，买不起别在这里浪费大家时间，没钱就多去搬砖，这个地方是你们这群穷逼来的地方吗？”
<br/>　　他怀里的女人也是得意洋洋道。
<br/>　　“亲爱的，这套房子不错哦，到时候我们也买了。”
<br/>　　“买了买了，不差钱。正好前段时间一不小心杀了个六阶的。”大胖子一副土豪的样子得意洋洋，故意把那个六阶两个字说的很重，就是为了要显摆自己多么厉害。
<br/>　　“六阶的，很厉害吗？”
<br/>　　林峰无语。
<br/>　　“小屁孩一个，人没多大，口气不小，六阶的怪物，你知道多厉害吗，一口吞三四个你都不带喘口气的。”
<br/>　　胖子嗤笑道，实际上，他能够杀那个六阶的，也算运气好，遇到了两个争地盘的六阶怪物，然后偷袭成功的。
<br/>　　“你就这么确定我杀不了六阶的？”
<br/>　　林峰冷笑道。
<br/>　　胖子一听，还要说些什么，林峰懒得墨迹了，直接拿出了五颗五阶的外加一颗更大的晶石。
<br/>　　“这，这莫非是七阶的？”
<br/>　　顿时那大胖子露出一抹惊愕的看着林峰。作为黑铁战士，他自然知道这么一颗晶石代表的是什么。
<br/>　　“不好意思，拿错了，对了，顺便问一下，这七阶的可以换多少三阶晶石？”
<br/>　　林峰笑呵呵道。
<br/>　　“可以换一千五！”
<br/>　　销售人员羡慕的看了眼苏云，能够有实力这么强的男朋友简直是太幸福了。
<br/>　　七阶晶石她也是第一次看到。要知道现阶段来这里买得起房子的都是些厉害的人物，五阶的六阶的都不少，但是七阶的还真不多见。
<br/>　　大胖子咽了口唾沫，尼玛的，七阶晶石都有，这绝对是大佬。
<br/>　　立即带着他的女伴灰溜溜的溜走了。
<br/>　　林峰嗤笑一声，半瓶子水就跑来装逼。
<br/>　　“那我干脆就要那一套，四百平的。我看打折后好像是两千五。”
<br/>　　林峰干脆要了套更大的。
