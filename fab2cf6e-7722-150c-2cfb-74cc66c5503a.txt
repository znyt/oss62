<br/>    深夜时分，一轮圆月行于黑云之中。
<br/>    
<br/>        步留痕看向身后的三位天仙，“还有多少时辰？”
<br/>    
<br/>        一位天仙道：“执刀人大人，还有半个时辰。”
<br/>    
<br/>        步留痕微微点头，“时候一到，不管情况如何，立刻动手。这是我的第一次行动，我不想出现任何差错。”
<br/>    
<br/>        半个时辰悠悠过去，步留痕挥手，将一个法器面具戴在脸上，这法器面具可以隔绝感知，身后上百地仙，三个天仙，也是学着步留痕的模样，带上面具，而后披上一身黑袍。
<br/>    
<br/>        夜色之下，步留痕道：“现在，可以动手了，我们负责的这个地方，只有一位天仙，我会前去杀掉他，你们要斩除其他人，一个活口都不留！”
<br/>    
<br/>        步留痕踏空飞走，只是一步，已经跨越到一座阁楼之中，这里的地形图，他已经看了数遍，进入阁楼之中，站在阁楼的窗口之上，步留痕负手看向正在打坐的瑶池天仙，瑶池天仙明显能够感受到一股强大的气息席卷而来。
<br/>    
<br/>        他睁眼，看得到一袭黑色披风的步留痕，带着诡异的面具，站在窗口之上，猎猎狂风吹得披风猎猎作响。
<br/>    
<br/>        瑶池天仙并不是瑶池圣地的人，而是本地的天仙投靠了瑶池，此人看向步留痕，怒道：“你是何人，你知道这是何处吗？瑶池，不是你们可以招惹的起的！”
<br/>    
<br/>        步留痕呵呵一笑，飘下窗台，脚尖触底，看向这瑶池的天仙，对于他的一切，他都已经了若指掌。
<br/>    
<br/>        “取你命的人！”
<br/>    
<br/>        瑶池天仙大惊失色，挥手之间便是一柄仙剑悬浮在掌中，双手之中仙力不断灌注在仙剑之中，只听一声爆响，楼阁的地板被剑势震得如同落叶翻滚！
<br/>    
<br/>        步留痕看向这一剑，淡淡道：“斩风剑，凝聚了剑魂，果然有点意思，不过，你在明，我在暗，杀你，只需要一次机会就够了！”
<br/>    
<br/>        瑶池天仙大怒道：“斩风剑下，巍巍山川，滔滔河流，滚滚沧海，浩浩天空，皆在一剑之下，给我死！”
<br/>    
<br/>        步留痕踏步，脚下生青龙，吼！
<br/>    
<br/>        青龙腾飞，步留痕下一步，已经
<br/>    
<br/>        是在三丈之外，而后步留痕继续踏步，每一步逼近瑶池天仙三步，他的身影如同鬼魅般移动，八步登龙，可以横行星空，自然可以横行在大地之上！
<br/>    
<br/>        瑶池天仙的眼中露出一抹惊讶之色，“八步登龙，你是步留痕，糟了！”
<br/>    
<br/>        瑶池天仙不过刚刚反应过来，却是听到背后传来一个声音，“你发现的太晚了！”
<br/>    
<br/>        瑶池天仙回头，迎面便是八步登龙最后一脚朝着他踢来，轰！
<br/>    
<br/>        瑶池天仙咳血，脑中一阵迷糊，就在这时，步留痕又是脚下飞动，连环飞腿。
<br/>    
<br/>        轰！
<br/>    
<br/>        轰！
<br/>    
<br/>        轰！
<br/>    
<br/>        瑶池天仙终于是清醒了过来，口中不由一口鲜血喷出。
<br/>    
<br/>        他想要召唤回来自己的飞剑，却是发现，步留痕这一次的八步登龙已经再次施展而来。
<br/>    
<br/>        轰轰轰轰轰！
<br/>    
<br/>        最后五脚落下，瑶池天仙只觉眼中的世界逐渐变黑，双膝一软，已经是跪倒在步留痕的身前，头颅低下，浑身鲜血滴答，落在地面之上，啪嗒作响。
<br/>    
<br/>        步留痕再次抬脚，一脚正中眼前天仙的脖子，摁在地面之上，直接将他的脖子踩断。
<br/>    
<br/>        “八步登龙，八步可以登龙，同样可以杀人。”
<br/>    
<br/>        他弯腰，而后挥拳，将瑶池仙人的头颅打得粉碎，这才收手。
<br/>    
<br/>        仙人，一般的手段是杀不死的，但是，头颅破碎，必定是死了。
<br/>    
<br/>        手中和脚下都染着鲜血，步留痕来到窗口之上，靠着窗子看向外面，此刻整个营地之中，已经是处处烽火狼烟，四处喊杀声一片！
<br/>    
<br/>        仙人之间的战斗，大地在开裂，若不是此处乃是鬼门，只是普通星辰，恐怕整个大陆都会被崩碎。
<br/>    
<br/>        步留痕看了一眼，淡淡道：“一个不留，杀无赦！”
<br/>    
<br/>        金戈铁马，腥风血雨一片。

<br/>        一个时辰之后，所有的一切终于是安静了下来，步留痕负手行走在众人之中，继续道：“放火，不留下任何痕迹，顺便，带走所
<br/>    
<br/>        有的财物！”
<br/>    
<br/>        众人齐齐拱手道：“我等，遵命！”

<br/>    
<br/>        天仙淡淡开口，“开城门！”

<br/>        地仙们飞身而上，朝着城楼之上飞去，守城的仙人和诸多超脱，不过一盏茶的功夫，就已经全部被杀掉。
<br/>    
<br/>        一个地仙而已，怎么可能是一群地仙的对手？

<br/>        两个天仙在前，朝着城中一路行去，来到一所红袖招酒楼之前，一个天仙道：“瑶池在很多城池你都有着红袖招的酒楼，而且会根据时间改变位置，今天，他们就在此处！”

<br/>        另外一个天仙笑道：“既然如此，那便杀进去，一个不留！”

<br/>    
<br/>        酒楼瞬间被冲开，这座城的城主并没有来，两位天仙驾到，而且这般行事，实在是太过霸道，他并不想死，只能放任步留痕的手下杀戮。

<br/>        所有人都感觉到了，一场灾难，将要来临了。
