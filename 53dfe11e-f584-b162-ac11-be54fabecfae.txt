　　一尺的距离，可以说是呼吸之间，便被苏莫跨越了。

<br/>　　当然，这也因为他本身实力较强，精神控制力极为强大。
<br/>　　不然的话，若是换成另外一个拥有混沌不灭体的人，绝不可能仅靠混沌不灭体就登上剑山之巅。
<br/>　　到了山巅之上，苏莫的压力，立刻全部消失，再也感觉不到任何的剑山锋锐之气。
<br/>　　唰！
<br/>　　霎时之间，滚滚血雾聚拢，瞬间便化为了苏莫的身躯，可谓是毫发无伤。

<br/>　　“怎么可能？”
<br/>　　“这……！”
<br/>　　这一刻，所有人均是心中狂震，面带惊愕之色的望着苏莫。

<br/>　　轰！
<br/>　　整个剑山通体一阵，发出惊天巨响，一股巨大的反弹之力，立刻将拓跋流明和云沐等人，全部排斥了出去，整个剑山之上，只剩下了苏莫一人。

<br/>　　随着剑山消失，整个空间全部崩溃，混乱的虚空之力，裹挟着众人，形成强大的排斥之力。
<br/>　　眨眼之间，空间倒转，所有的人都被强行送出了万道剑冢。
<br/>　　嗡嗡嗡~~~
<br/>　　万道剑冢之外，空间不停的波动，一道道人影飞速的出现。
<br/>　　无论是拓跋流明和韩生等在剑山之上的人，还是停留在剑山脚下的那二十余名天才，亦或是停留在其它几关的人，全部被强行送出了万道剑冢。
<br/>　　眨眼之间，万道剑冢之外，便人头涌动，出现了数千武者。
<br/>　　而万道剑冢所在的深坑，居然在一阵大地蠕动之下，消失不见了。
<br/>　　存在了无数年岁月的万道剑冢，至此消失无踪了。
<br/>　　“怎么回事？”
<br/>　　“万道剑冢怎么了？”
<br/>　　“发生了什么变故？”
<br/>　　无数的人惊呼出声，个个面带惊疑之色，这些人没有到达剑山所在的地方，所以根本不知道发生了什么情况。
<br/>　　“有人登上上剑山之巅，进入了传说中的最后一关！”有人面带震惊的开口，正是之前在剑山脚下的二十余名天才之一。
<br/>　　“什么？”
<br/>　　“这怎么可能？”
<br/>　　“那他要掌控终极剑道的秘密了？”
<br/>　　“此人是谁？”
<br/>　　一石激起千层浪，此言一出，立刻引起了轰动，无数的天才为之震惊。
<br/>　　传闻无人能登上的剑山之巅，传闻无人能进入的最后一关，居然有人达到了。
<br/>　　此人是谁？
<br/>　　怎么会如此厉害？
<br/>　　“此人叫秦陨，是终极神王殿之人！”又有人开口了。
<br/>　　“秦陨？从未听闻啊！”
<br/>　　“此人为何能登上剑山之巅？”
<br/>　　“他有何不凡之处？”
<br/>　　“……”
<br/>　　嘈杂的声音此起必伏，显得喧闹不堪，众人震惊的同时，亦是充满了疑惑。
<br/>　　因为这个登上剑山的秦陨，没有任何的名气，闻所未闻。

<br/>　　他们眼馋不已的终极剑道的秘密，自身没有得到，反而被秦陨得到了。
<br/>　　“可恶啊！”拓跋流明双拳紧握，心中怒吼，但是他却是没有丝毫的办法。
<br/>　　公孙嘉面带感慨之色，同样心中震动，他也没有料到，秦陨会真的登上剑山之巅。
<br/>　　不过，他却是没有丝毫的愤怒，因为，他自己不可能得到的东西，也不会去强求。
<br/>　　而一旁的韩生，出来之后，便立刻传讯回终极神王殿。
<br/>　　他传讯之人，乃是终极神王殿毁灭剑宫的宫主，毁灭剑王。

<br/>　　不过，根本不用韩生传递消息，终极神王殿也已经知晓了大概的情况。
<br/>　　只见终极神王殿所在的神剑山，弥漫的光华更加的璀璨，不仅如此，一股股白色的雾气，从山体之中，不断的喷发而出。
<br/>　　这种迹象，让得神剑山上无数的终极神王殿之中，纷纷心神震动。
<br/>　　“怎么回事？”
<br/>　　“难道有人在万道剑冢之中登上了神剑山？”
<br/>　　“这不太可能吧！”
<br/>　　无数的终极神王殿弟子，纷纷飞了出来，看着神剑山的奇异的场景。
