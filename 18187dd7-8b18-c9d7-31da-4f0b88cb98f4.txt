　　来到天权宫前，抬头望去，偌大的匾额上印着龙飞凤舞的两个金色古篆字体，一股沧桑的气息扑面而来。
<br/>　　天权宫也正是所谓的真武殿，乃是蜀山禁地，同时也是和阳长老的住处。
<br/>　　和阳长老是蜀山五老其一，地仙修士，已经是仙人级别，当真拥有移山填海的本事。
<br/>　　要知道，地仙可是本位面的巅峰强者，由和阳长老在这里坐镇，可见蜀山对真武殿有多么的看重。
<br/>　　收回目光，慕清霄刚要进入真武殿，门口中就走出一名青年。
<br/>　　青年身穿蜀山道袍，只是当他看到迎面而来的慕清霄时，脚步一顿，脸色一僵，慌乱之色自眼中一闪即逝。
<br/>　　很快，僵硬的脸色就恢复正常，慌乱也被他藏于眼底。
<br/>　　慕清霄剑眉一挑，看出了一些端倪，但并未出声。
<br/>　　下一秒，青年脸上就挂起风轻云淡的笑容，缓步迎了上来，道：“这是师弟，不知来真武殿又何事？”
<br/>　　“我找和阳长老。”
<br/>　　“师尊前往天枢宫与掌门商议事宜，暂时不在殿内，有什么事情对师兄说就行，等长老回来，我就为师弟禀报。”
<br/>　　青年脸上依旧挂着沐浴春风的笑意，语气充满和善。
<br/>　　“多谢，只是在下的事情有些急，看来只能去天枢宫了。”话语刚落，慕清霄就转身，准备天枢宫。
<br/>　　然而，就在慕清霄转身的一瞬间，异变突生。
<br/>　　只见青年方才还挂着笑容的脸蛋，瞬间就变得狰狞起来，不知何时，手中已经出现一柄寒光四溢的长剑。
<br/>　　当慕清霄转过身去，青年眼中杀意四射，嘴角噙着一抹狰狞的笑意，长剑直接朝他的心口刺去。
<br/>　　“师弟，你来的可真不是时候，莫要怪师兄啊！”
<br/>　　“噗嗤……”
<br/>　　寒光四溢的长剑，毫无阻挡的刺进心口，慕清霄嘴中吐出一口鲜血，艰难的转过身来，眼中古井无波。
<br/>　　“为什么？”
<br/>　　“为什么！”
<br/>　　冷笑一声，青年原本就狰狞的脸蛋如同恶鬼一般。
<br/>　　“好不容易等到机会，本想拿着仙法趁机溜走，却不想到正巧碰见你，师兄可不敢给你去天枢宫的机会，若是那老不死的东西赶回来，师兄哪里还有时间逃走，莫怪师兄，要怪就怪你命中有此一劫。”
<br/>　　“命该有此一劫？”
<br/>　　被长剑贯穿胸口，慕清霄俊逸的脸上却露出一抹古怪的笑意。
<br/>　　望着他脸上的笑容，青年觉得有些不对劲，心间莫名的升起一股寒意，长剑在慕清霄体内用力一搅！
<br/>　　下一秒，青年就彻底懵逼了。
<br/>　　只见原本被长剑贯穿的身躯一阵扭曲，旋即就变得虚幻起来，耳边响起一阵清朗冷漠的声音。
<br/>　　“确实，你命中有此一劫，否则也不会正巧遇到我。”
<br/>　　“噗嗤……”
<br/>　　话语刚落，青年狰狞的面孔顿时凝固，嘴角溢出一抹血迹。
<br/>　　低头一看，只见一柄虚幻的剑刃已经将他的心口贯穿，察觉到生命的流逝，青年神色瞬间癫狂。
<br/>　　“不……马上，马上就要成功了！”
<br/>　　无力的向前走了几步，竭斯底里的嘶吼而出，身躯栽倒在地，双眼望着蔚蓝的青天，死不瞑目。
<br/>　　望着地上拖出一条长长的血痕，慕清霄面色如常。
<br/>　　刚才看到青年的一瞬间，后者眼中的慌乱就被他捕捉到，同时也看出此人有些端倪。
<br/>　　双眼相对的一瞬间，慕清霄就用写轮眼发动幻术，区区元婴期修士，中了他的幻术，根本分不清真假。
<br/>　　真真假假，假假真真。
<br/>　　在慕清霄的眼中，青年一直举着长剑在对空气说话，画面有些微妙。
<br/>　　恐怕和阳长老做梦也想不到，他的座下弟子居然会在他前往天枢宫的一段时间，携着蜀山仙法准备判出蜀山。
<br/>　　慕清霄也有些意外，区区一个元婴修士，哪里来的这种勇气。
<br/>　　结合青年刚才的话语，在他的背后，肯定有别的门派在推波助澜，甚至允诺了他很高的地位。
<br/>　　在利益面前，任何东西都是虚的，宗门信仰，长老座下弟子的身份，不过只是空谈罢了。
<br/>　　收回目光，慕清霄没有去管青年的尸体，随手卷起一缕鲜血，弹指间，血液已经在灵气的包裹下向天枢宫疾射。
<br/>　　“轰轰……”
<br/>　　顷刻间，天枢宫中一股恐怖的气息冲天而起。
<br/>　　眨眼间，一道身影就出现在了真武殿上空，来者是一名身形微胖，发丝皆白的老者。
<br/>　　慕清霄脸色微微一变，地仙境强者果然名不虚传，整片天空此刻都在地仙的威压下颤抖。
<br/>　　“若是不给老夫一个说法，即便你是掌门钦点的妙丹堂长老，老夫也要让你饮恨当场！”
<br/>　　瞥了眼地上的尸体，和阳长老浑浊的双眼中杀意凌然，地仙境的威压直接向慕清霄碾压了过去。
<br/>　　感受到上空空间的沉重，慕清霄脸色瞬间变得苍白了一些。
<br/>　　当地仙威压全部降临，慕清霄感觉，整片天空仿佛都要崩塌，可就在下一秒，威压全部凭空消散。
<br/>　　不知何时，清微与其余三位长老已经出现在慕清霄的身旁，脸色也颇为沉重。
<br/>　　“师弟，不要乱来。”
<br/>　　闻言，和阳长老收回威压，目光死死的盯着慕清霄，道：“师兄，老夫座下可就这一名弟子……”
<br/>　　微微摇头，清微的目光落在慕清霄身上，道：“究竟怎么回事？”
<br/>　　慕清霄松了口气，地仙境远比他想象中强大的多，刚刚空间瞬间凝固，犹如置身沼泽当中，根本抽不出身。
<br/>　　恐怕想要使出神通《空间门》都有些困难。
<br/>　　若是对上渡劫期修士，慕清霄自然不会如此忌惮，但地仙可是比他高了两个大境界。
<br/>　　说到底他现在只是修士，与仙人之间终究差距太大了。
<br/>　　慕清霄将令牌取出，道：“我只是来阅览一下蜀山的仙术，正巧遇到他，接下来的情景请掌门自己看吧。”
<br/>　　说罢，慕清霄就大摇大摆的朝真武殿中走去。
<br/>　　和阳长老则脸色难看到了极点，清微拂尘一挥，青年一缕鲜血飘起，随后虚空中就呈现出一副画面。
<br/>　　画面如同电影一样，正是慕清霄来到真武殿碰到青年的一幕幕，从两人交谈到青年持剑对着空气胡言乱语。
<br/>　　……
