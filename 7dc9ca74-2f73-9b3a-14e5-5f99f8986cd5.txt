　　洪荒东部一座不知名的洞府之中，洞府没有豪华，没有华丽，没有奢侈，有的只是简单、朴素。但这看起来却没有丝毫的不适，这里的一草一木，一山一石，都充满了大道的韵味。
<br/>　　大道至简！
<br/>　　洞府之中，三个老道正在交流讲道，他们之中每一个人身上都充满大道的气息。
<br/>　　第一个老者一身青色道袍，道袍之上，一个巨大的阴阳太极图，其头顶，阴阳转换，阳极生阴，阴极生阳，阴阳流转，玄之又玄！
<br/>　　第二个老者一身灰袍，头发丝披散，其头顶一个大鼎，缓缓旋转，散发出造化之光，生生不息，乾坤再造，庆云之上，天地万物，具在其中。
<br/>　　第三个老者一身白色道袍，一头白发，眉棱挺翘，给人一种凌厉强势的气息。在其头顶之上，一柄金黄色神剑缓缓旋转，剑形如人之脊骨，共分二十四节，剑柄刻有“昆吾”两字。剑身上散发出凌厉霸道的剑气，剑气纵横，剑意冲天，直上云霄，带有一种煌煌之威！
<br/>　　这三人便是洪荒之中鼎鼎有名的阴阳老祖，乾坤老祖，以及昆吾老祖。
<br/>　　三个老道将头顶庆云相互交织，互相交流各自的道行心得，相互印证各自的大道。这样的状态不知持续了多久。
<br/>　　突然，第三个老者微微皱眉，睁开双眼，眼中凌厉之色毫不掩饰的绽放，剑气凌天。使得其他两个沉浸在大道汪洋之中的老者也惊醒了过来。
<br/>　　“怎么了，昆吾道友？”其中青色道袍老者微微皱眉，面带不喜道。
<br/>　　好不容易几人聚在一起论道交流，他刚刚沉浸在对自己阴阳之道的领悟之中，已有所领悟，再过一段时间，他就有可能突破了，现在却被打断，要是昆吾老祖不给他一个说法，今天说不得要做过一场。
<br/>　　昆吾老祖缓缓站起，手持长剑，并没有回答阴阳老祖的话，而是一只手在快速掐算起来，速度越来越快，最后连手影都看不见。
<br/>　　突然，他脸色大变，好似见了什么不可思议的事物，顿时一口鲜血喷出，倒退几步，这才站好。
<br/>　　“贫道的洞府出事了，阴阳老祖，乾坤老祖，贫道先走一步，来日再聚！还请阴阳道友先行打开洞府禁制。”
<br/>　　说完，昆吾老祖看向阴阳老祖，因他们在一起交流大道，不能让外人打扰，故而阴阳老祖在他洞府之外布满禁制。现如今他想让阴阳老祖打开洞府禁制，自己好出去。
<br/>　　“昆吾老祖，到底何事，让道友如此惊慌？”乾坤老祖好奇的问道。按理来说，像他们这样的人，出来之时自己的洞府本应该布上大阵，隐匿起来，就算被人发现，没有一定的道行，也不能进入。现在看昆吾老祖的样子，明显是洞府出了大事，很可能遭劫了。
<br/>　　“贫道洞府让人给抢了，我倒要看看，谁敢抢我昆吾老祖的洞府。”
<br/>　　“抢了，可算出是何人所为？”另外两人疑惑道，谁敢抢昆吾的洞府，昆吾在洪荒之中，霸道至极，哪个不长眼的，敢抢他的洞府，难不成是鸿钧？亦或者罗睺？
<br/>　　“贫道亦不知是何人所为？”昆吾老祖也十分不解，以自己的道行，都推算不出是何人，洪荒宇宙中，何时出现这等人物了？难不成是贫道的消息落后了？
<br/>　　“怎么可能？！”两人惊诧道，他们的心情与昆吾老祖一样。
<br/>　　“贫道推算之时，冥冥中有某种东西在阻碍这贫道推算，当贫道要强行推演之时，便遭到了强烈反噬。”昆吾老祖郁闷道。
<br/>　　“呼——”两人同时呼出一口气，道：
<br/>　　“还好，还好！这应该是那人拥有某种遮蔽天机的灵宝，这样能够遮蔽天机的宝物在洪荒之中虽然稀少，但还是有的。”两人虚惊一场，还以为是某个大神通者呢。
<br/>　　在说话之际，洞府禁制早已被阴阳老祖打开。不等两人再说什么，昆吾老祖便匆匆离去，向着他自己洞府方向飞去。
<br/>　　在他离去后，阴阳老祖与乾坤老祖面面相觑，不知说些什么。良久，阴阳老祖道：
<br/>　　“真不知是何方神圣，竟敢抢劫昆吾老祖的洞府？”
<br/>　　“是啊！洪荒如此大，有几个隐藏的大能也不足为奇，昆吾道友这一去，却是要花费数年之久，不知道他回去后，还见不见得到他那豪华的洞府。”乾坤老祖也感慨道，只是语气之中，有一股幸灾乐祸的味道，显然，他对于昆吾老祖的关系也不是很好。
<br/>　　阴阳老祖大有深意的看了乾坤老祖一眼，道：
<br/>　　“洪荒最近可不平静啊，龙族、凤凰族、麒麟族，三大种族闹得可是很欢呐，想必不用多久，大劫就要再次来临了。哎！天道难测，真不知道我们何时成道啊！”
<br/>　　“是啊，大道艰难，何时成道啊？”乾坤老祖也感叹道，他们都是在开天之初就转劫而生的人，没有什么人比他们还知道成道之难了。
<br/>　　大道无涯，何时能成？前路遥遥无期，他们都不免有些感伤。
<br/>　　两人顿时沉默，不再说话，谁也不知道对方在想些什么。
<br/>　　就在此时，天地灵气动荡，洪荒之内，四海翻滚，巨浪滔天而起，一条金色神龙冲天而起，此龙凶猛无比，带有无边神威，气息磅礴滔天，一股股强烈的龙威，刹那间，向着四面扩散，弥漫八方。
<br/>　　这龙的气息，极为恐怖，似有镇压天地苍穹的威力一般，他张口大吼，声若雷霆，翻滚间便传遍洪荒大地：
<br/>　　“昂！昂！昂！吾为龙祖。”
<br/>　　浩大的龙吟之声响彻洪荒，强大的龙威，威慑天地，只见那横亘高空的金色神龙，咆哮着腾于天际。
<br/>　　骆头、蛇身、鹿角、龟眼、鱼鳞、虎掌、鹰爪、牛耳……
<br/>　　这是神龙，带有莫大神威，他身具皇者之气，霸道绝伦。
<br/>　　“天道在上，从今往后，鳞甲之辈以龙祖为尊，吾为龙祖！”
<br/>　　顿时，四海之内，龙祖之下，万千龙族齐齐现身，高声嘶吼，气势威动四海八荒，先天白族里，所有鳞甲类，还有水族，尽皆拜服。
<br/>　　天降功德气运，祖龙气势更进一步，修为本已为大罗金仙圆满，现在在量劫主角气运的帮助之下，使得肉身突破至准圣初期。
