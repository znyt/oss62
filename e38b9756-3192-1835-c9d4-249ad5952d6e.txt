　　白天竞技场出了这么大的事情，崔钰自然有些耳闻，对于李承乾找他的目的也是一清二楚，只是他实在想不到某人会如此直接：“崔钰，朕要见长乐！”
<br/>　　“陛下，您现在的身份非同以往，若是再……”
<br/>　　“朕要见长乐，崔钰莫非你想抗旨？”
<br/>　　因为长乐的事情，李承乾自然不会有什么好心听崔钰来解释原因，他现在的目的很简单，那就是见到长乐问清楚一切，然后再带她回来。
<br/>　　“臣不敢！”崔钰纵然有地府判官的另一个身份，但也不敢在人间帝王面前放肆，李承乾的威压下只能俯首。
<br/>　　“那就快点！朕耐心有限。”李承乾坐在暗影中，声音有些嘶哑，就算他不这样说，崔钰也能听出他没有多少耐心。
<br/>　　……
<br/>　　还是那片与阳间衙门毫无二致的空间，四周的物品还是摆放在原来的位置，唯一的区别就是这一片空间里只有李承乾与长乐两人。
<br/>　　长乐诧异的看着李承乾，脸上满是恐惧：“哥？！你，你也死了？！”
<br/>　　“啪”的一声，回答长乐的是一记响亮的耳光，以及李承乾低沉的怒吼：“李丽质，你到底清不清楚你自己在干什么？！是不是这些年我把你给宠坏了！你有没有想过你死以后的事情！”
<br/>　　“你……”
<br/>　　“说啊，为什么？这一切到底是为什么？！是谁蛊惑你的！说！”李承乾根本不理长乐眼中的委屈，大声的喝斥道。
<br/>　　“我……”
<br/>　　“你什么你，你觉得你这样很有道理么？有什么事情不可以摆出来说，非要用死来解决问题！你这样子让我怎么跟父皇和母后解释！你说啊！”
<br/>　　李承乾这一次可是真的急了，也不听长乐的解释，只是在那里发泄似的吼着，这个妹妹也是不让他省心，现在搞了这么一出儿，让他这个当哥哥的完全就是束手无策。
<br/>　　长乐其实也很后悔，不过眼下这个情况她也不知道应该怎么办，正傍徨的时候，李承乾却出现在她的面前，这让长乐既担心，又兴奋。
<br/>　　可是李承乾一记响亮的耳光打过来，却是把长乐给打傻了，以前连句重一点的话都没有被说过，现在竟然被打了，让她有些难以接受。
<br/>　　想说话，李承乾又一个劲的在那里唠叨，使得这位大唐长公主再也无法忍下去，歇斯底里的喊道：“够了！李承乾，你知道什么，你知不知道我这些年是怎么过来的，你只知道你的天下，你的子民，你的大唐，可是你想过我没有？”
<br/>　　“这么多年了，我每天都在你背后看着你，帮你打理着一切，可是你正眼看过我一次没有？既然你不在乎我，当初你又为什么帮我退了冲表哥的亲事！李承乾，你就是一头猪，一头最蠢是的猪，我恨你，恨你！”
<br/>　　面对疯了一样的长乐，李承乾的神情一黯，他此前并不是不知道长乐的想法，可这事是世俗所不容许的，出于保护长乐的态度，他选择了漠视，希望时间可以让她忘掉这一切。
<br/>　　只是他没有想到，时间并没有淡化一切，反而使得长乐心中的那份期待像野草一样疯长，以至于最后酿成如此悲剧。
<br/>　　长乐冲李承乾喊完之后，似乎已经将心中所有的怨气都吐干净了，凄笑一声说道：“不过现在说什么都没有用了，我们都已经死了，一了百了！”
<br/>　　“谁告诉你一了百了的？又是谁告诉我已经死了的？”李承乾见长乐的情绪好一些了，也强压下心头的火气，没好气的说道：“这里本是判官的地盘，可是却只有你我两人，你就不觉得奇怪？”
<br/>　　“有什么好奇怪的，你什么事情干不出来！”长乐慢慢稳住情绪之后，轻轻拭了拭眼角，但却没有一滴眼泪。
<br/>　　李承乾看着长乐柔弱的样子也是有些心疼，也不再责骂她，叹了口气说道：“我能来这里还是崔钰那老小子帮的忙，那小子一身双职，日审阳，夜断阴，这里便是他的地盘。”
<br/>　　“什么？就是那个……瘦瘦小小的崔钰？他是这里的判官？”长乐的神经也是足够大条，自己的事情还没整明白呢，又开始八卦起崔钰。
<br/>　　“对，崔钰就是这里的判官，这件事情以后再说，以后你可以好好问他。”李承乾有些不耐烦摆了摆手，接着道：“现在你抓紧时间把你的事情告诉我，到底你是什么回事儿，为什么突然之间会想到自尽。还有，独孤玉凤他们是怎么回事，你怎么可能有力量控制住他们几个。”
<br/>　　地府的时间与阳间不同，李承乾因为自身气场的关系崔钰已经没有办法拉他入梦，只能带他魂入地府，所以在地府每一分钟的时间都很关键，扯的时间长了，阳间那边非得出问题不可。
<br/>　　长乐抿了抿嘴，似乎在犹豫要不要说，但最后还是深吸一口气说道：“哥，你还记得玄寻雪么？”
<br/>　　“玄寻雪？那个小道姑？这件事情和她有关系？”李承乾面色微微一变。
<br/>　　那小道姑的确是很久没有出现过了，久到李承乾几乎已经忘了她的存在，没想到那小道姑竟然与长乐联系上了。
<br/>　　“是的，独孤玉凤还有舒天和程华安就是玄寻雪带人掳走的，枪也是他们带人弄走的，至于再多……，我不能说。”长乐简单的为李承乾指了一个大致的方向，然后便紧闭双唇再也不肯开口。
<br/>　　不过这对李承乾来说也已经足够，只要知道了一点线索，他相信情报科那边一定可以查清楚这段时间到底发生过什么。眼下重要的还是长乐，这丫头稀里糊涂就给自己来了一枪，这事儿的确有些不好解决。
<br/>　　想到这里，李承乾拍了拍长乐的肩膀：“丫头，这段时间你就先在这里待着，有崔钰在这里不会有人敢为难你，等哥回去之后想想办法，看看能不能把你救出去，记住，哪里都不准去，就在崔钰这里等着，否则我就再也不管你了。”
<br/>　　长乐听到李承乾的话之后，瞬间抬起头：“哥，你，你还能回去？！你不是死了？”
<br/>　　“我什么时候说过我死了，我就是来看看你这个不让人省心的丫头，回头还有一大堆的事情要处理呢，你在这里老实待着吧。”李承乾瞪了长乐一眼，而后对着大堂外面沉声说道：“崔钰，进来吧！”
