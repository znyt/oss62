　　随着这一波团战的结束，叶帝他们这边的发育时间大大提前，最终的结果自然是没有丝毫的悬念。
<br/>　　拥有着这样一套完美的3保2的真容，无论是猪妹这个肉，还是萝卜的adc，对方都没有丝毫办法将他俩杀死，而杀不死他们的结果自然就是，蓝队要被反灭。
<br/>　　最终这一局比赛在时间到达35分钟的时候，便是早早结束，虽然萝卜最后的数据是6-0-6，不过最终的mvp依旧是落到了夜雪狂歌的头上。
<br/>　　对此大家倒是都没有什么异议，反而是非常赞成这个mvp。
<br/>　　“这个mvp给的没有什么问题，虽然夜雪的战绩并没有萝卜这样豪华，不过白队这边之所以能够取胜，夜雪这一局最少占了6层的功劳，前期的意识流反蹲，大龙团那一波极限秀，都是夜雪狂歌的功劳。”米粒开口大声的喊道。
<br/>　　“你知道我现在是一种什么感觉么？我感觉白队越来越可怕了，如果夜雪一直以这样的高水准发挥下去的话，我实在是不知道还有哪个战队能够取胜他们，这一场比赛我们已经能够看到，一个出色的打野是可以盘活整个局面的。”林雅楠跟着开口说道。
<br/>　　“是啊，估计看了夜雪今天的这场表演，确实是一场表演，他的这个打野给我们的是一种视觉上的享受，想必接下来其他的打野选手也都是有了目标了吧。”米粒笑着说道。
<br/>　　而事实上也正如米粒说的一样，无数的打野、无数的职业选手在看到夜雪狂歌的这场表演后，心中都是生出了一种我也要像他看齐，我也要努力的想法。
<br/>　　这一局夜雪狂歌再一次见识到了用智商来打野的这个套路，飞刀的打野被他玩弄于鼓掌之间，一个前中期这么强势的蜘蛛，居然生生的被一个猪妹弄的没有地方可以gank。
<br/>　　这只是想想就让他们感觉有些热血沸腾，试想一下如果白队这一局选的不是这么怂的阵容，而是非常刚的阵容，如果夜雪的表现在这么好的话，恐怕这一局就要更加早的结束了吧。
<br/>　　蓝队比赛室，
<br/>　　“这一局夜雪真的是超神发挥了，如果下一局在这样的话，那咱们就真的没法玩了。”蒋念皱着眉头开口说道。
<br/>　　“下一局选个稳点的阵容吧，前期找不到机会，那咱们也来一个后期的阵容。”林萧开口对着李景禹建议道。
<br/>　　经历过上一局之后，林萧真的是没有勇气在和对方选一个前期的阵容对刚了。
<br/>　　“那就选一个偏后期一点的阵容吧，这一局视野做好，别给他们gank的机会，夜雪这一波超神发挥，真的是有些让人绝望了。”李景禹也是有些无奈。
<br/>　　十分钟的休息时间一过，伴随着场下传来的巨大的欢呼声，双方的第四局比赛终于是正式开始。
<br/>　　而在这一局比赛开始的时候，现场双方的粉丝都是爆发出了无与伦比的热情，那欢呼声要远远超过之前三场都能。
<br/>　　白队的粉丝自然是希望叶帝他们乘胜追击，争取3：1灭掉对手，这样他们也就更有了吹嘘的资本，要知道春季赛的时候，我们可是打满了5场呢，现在4场就结束了，是不是我们更牛逼一些。
<br/>　　当然了更多的粉丝则是不想在继续煎熬下去了，如果打到第五场，他们真的是不确定白队就一定能够继续赢下去。
<br/>　　而反观蓝队的那些粉丝呢，他们的想法就很简单了，这一刻我么已经落后了，我们这群粉丝自然是要大声的欢呼，让场内的那些人听到他们的热情，从而更加的自信。
<br/>　　“这一局蓝队这边终于是决定针对一手香蕉了，把他的奶妈给ban掉了，感觉这一手做的很明智，上一局我们已经看到了香蕉的表现了，他用奶妈其实是可以抗住压的，只要抗住压不死，那么到了后期奶妈这个英雄爆发出来的威力可就真的太大了。”看着对方的ban人，米粒连忙开口说道。
<br/>　　“白队这边也是一如既往的继续针对着蓝队的下路，之前老林表现的非常好的德莱文被ban了，估计接下来两个又是锤石和机器人了。”林雅楠跟着开口分析。
<br/>　　骨肉按接下来的双方的ban人正如两人预料的那样，白队ban掉了锤石和机器人，而蓝队那边在针对了一手香蕉后，将更多的注意力则是放到了夜雪的身上，直接ban掉了他的盲僧，还有就是蜘蛛这个强力的英雄。
<br/>　　以夜雪现在的状态，他们还真怕夜雪狂歌选出这两个强势的打野，如果那样的话，他们这边承受的压力可就真的是太大太大了。
<br/>　　白队比赛室，
<br/>　　“香蕉这一局拿那个英雄行么？”叶帝开口问道。
<br/>　　听着叶帝的话，香蕉眼睛一亮，自信的开口说道：“行，怎么不行，流浪这个英雄我已经练了好久了。”
<br/>　　叶帝所说的正是流浪上单的这个套路，要知道前世的时候流浪不仅仅可以走上路，在上单同样是非常牛逼的，在中路他可能会被许多英雄压制，毕竟其他的中单英雄在前期无论是爆发还是伤害，都要领先流浪一头的。
<br/>　　不过真要是让流浪去上路的话，那这个局面可就变的不一样了，一个长手的流浪，不仅不会被对方的上单压制，还会占据着主动反压对方，这也是为什么那个时期那么多人愿意用上单流浪的原因。
<br/>　　而且最重要的是，流浪这个英雄真的很肉，玩上单流浪他完全可以不出大天使这件装备，在做出时光之后，就转冰心这些加蓝的防御装。
<br/>　　这样的话，流浪不仅有输出，而且也会肉的让人没有打下去的欲望。
<br/>　　这一局蓝队这边在蓝色方，白队在紫色方，所以在双方ban人结束后，蓝队拥有着先选人的权利。
<br/>　　“先抢鳄鱼吧，这个英雄不能放给香蕉。”李景禹开口说道。
<br/>　　当然了他之所以敢这么选，完全是因为楚南也是非常擅长鳄鱼的，要知道如今世界上最著名的三大鳄鱼，可就包括着香蕉、楚南呢。
<br/>　　既然楚南在这其中，那对于这个英雄自然是有着他的见解了。
