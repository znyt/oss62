　　叶少阳定睛一看那人头，自己也懵了，果然是曹雨兴！最重要的是，脸皮也是完好无缺的……
<br/>　　这是怎么回事？
<br/>　　“你把曹雨兴杀了！”陈晓宇叫喊起来。
<br/>　　“我怎么知道是他！”叶少阳反驳。
<br/>　　“不知道，你不会多看一眼，我当时都看到了，喊你停手，你还是动手了！”
<br/>　　叶少阳冷冷看了他一眼，“你眼瞎了刚才没看见，我要是停手，现在躺在这的就是我了。”
<br/>　　妙心走上来，让他们不要吵，然后前去检查曹雨兴的人头和躯体，的确……是他的身体，看上去完好的，脸皮也没有被剥掉。
<br/>　　“怎么会这样？”众人十分不解。
<br/>　　梧桐道：“那我们之前看到的那个脸皮被揭掉的又是谁？”
<br/>　　没有人作声，因为都不知道怎么解释这件事。
<br/>　　叶少阳走到尸体前面，检查了被自己枭首时脖子上的伤口，伤口整齐，但是检查不到一点尸气的存在。
<br/>　　陈晓宇还在那里借题发挥，说叶少阳杀人，叶少阳很烦，但也没有解释，刚才那一幕，大家都是看到的，如果之前扑向自己的曹雨兴真的是人，一只枣木剑，绝对不可能就那么轻松将他的脑袋削下来，但是被斩杀之后的曹雨兴的尸体，却并没有尸气，这就给了陈晓宇发挥的借口，一副义愤填膺的样子，威胁等出去之后，要告诉法术公会这件事……
<br/>　　“这是误杀，最多也就是误杀。”卢晓清倒是给出比较清晰的判断。
<br/>　　“对，误杀，我就跟雨兴山门的人说，他被人误杀了。”陈晓宇歪着脑袋看着叶少阳，嘴角带着一抹笑意。“公会就不说了，雨兴的山门，人家上百年才出这么一个人才，就这么被误杀了……”
<br/>　　叶少阳懒得看他拙劣的表现，好像真是为了曹雨兴抱不平似的，之前得知曹雨兴有危险，但是不愿意进石门来找他的，也是陈晓宇……对他的人品，叶少算是彻底放弃了最后一丝希望，一步迈到他面前，冷冷道：“你再啰嗦一句？”
<br/>　　陈晓宇猛然抬头看他，目光也很犀利，他手下的猴王也一步跳到他的肩上，恶狠狠地瞪着叶少阳。
<br/>　　“怎么，你杀了雨兴还不够，还想杀我灭口不成，你一个小小天师，哦，天师都还不是，谁给你的勇气？”
<br/>　　一直肩膀，搭在了叶少阳的肩上，叶少阳转头一看，是毛小方，毛小方只是看了他一眼，什么也没说，叶少阳已经明白了他的意思：无论怎么样，他是站在自己这一边的。
<br/>　　“你们两个想一起上？”
<br/>　　毛小方怎么也是天师，陈晓宇还是有点担心的，往后退了两步，单手捏决，放在胸前，这是法术界斗法的一个起手式，更多是摆出一个想要一战的姿态。
<br/>　　叶少阳心中也在犹豫，眼看对方都逼到这地步，甚至下了战斗的邀请，自己要是不应战的话，给人的感觉实在有点怂，他倒是不怕别人说怂，但是……这实在不是自己的作风，何况对于陈晓宇，他也实在是忍了太久，不想再忍了。
<br/>　　“来吧，让我给你长长见识！”陈晓宇再度发出挑衅。
<br/>　　“听你的。”话音未落，叶少阳冲了过去。
<br/>　　陈晓宇也扑上来。
<br/>　　毕竟是法力上差距太大，叶少阳没有跟他硬碰硬，而是施展茅山凌空步，在他身边游走，寻找着机会。
<br/>　　因为是单挑，而且叶少阳暂时也没有落败的迹象，毛小方也不方便立刻就上，在一旁密切注视着局势。
<br/>　　“你小子还挺灵活！”陈晓宇冷哼一声，“不敢跟我硬碰吗？”
<br/>　　叶少阳不理他，继续在他身边游离，寻找机会。陈晓宇越着急，他的机会就越多。
<br/>　　叶少阳自己内心其实也挺悲催的，陈晓宇的实力是不错，不过……也就是那么回事，跟当初被自己踩过的凌宇轩差不多，而自己，早就不是那个时候的自己了，灵仙牌位，与地仙的差距，有一道不可能越过的鸿沟，更何况陈晓宇只是个准地仙。
<br/>　　如果是全盛时期的自己……叶少阳内心无奈地叹了口气，不敢多想，全力应付眼前的局面。
<br/>　　“你是怕了吗，只会躲吗？”陈晓宇已经有点气急败坏。
<br/>　　“你抓不到我吗？”叶少阳故意反唇相讥。果然陈晓宇被这句话更加激怒，出手也是更加狠辣，一心想要擒住他。
<br/>　　来了！
<br/>　　叶少阳总算等到了一个绝佳的机会，人猛然一个反转，险而又险地避过陈晓宇的攻击，绕到他身后去，一抬腿踢中了他的膝关节。
<br/>　　陈晓宇向前一个趔趄，叶少阳抓住机会，顺势向前，左手成刀，要去砍他的后脑勺，这一下是顺势而为，是必中的，突然一道人影冲过来，挡在了两人中间，叶少阳急忙收势，定睛一看是妙心。
<br/>　　“你们闹够了没有，我们是干什么来了？”妙心对叶少阳斥道，当然这话也是对陈晓宇说的。
<br/>　　梧桐也上来劝说，那边卢晓清看他们都来劝架了，于是上去把怒气未消的陈晓宇拉住。
<br/>　　叶少阳自然没什么可说的。陈晓宇倒是对他说了几句狠话，叶少阳只是对两个姑娘耸了耸肩，意思不是自己惹事。
<br/>　　“我不管你们有什么恩怨，你们不是定下约定了吗，到龙华会上再去解决吧，古墓里可不是你们决斗的地方，我们还要去地宫找炼尸缸。”妙心斜眼看着陈晓宇说道。
<br/>　　一场意外的战斗暂时平息。
<br/>　　大伙商量了一下，决定先把曹雨兴的尸首留在这里，等探完地宫，出来的时候再收拾带回去，不然扛着一具尸体在古墓里行走也很麻烦。
<br/>　　一行人越过了花坛，继续往前走，讨论起曹雨兴的尸体为什么会在花坛里，谁也说不上是为什么，至于之前看到的那个被撕去脸皮、引他们进来的曹雨兴，更是一个谜，也只能解释成是通灵佛子花造成的幻象了。现在通灵佛子花已经被烧了，倒是也不用纠结这些细节了。
<br/>　　下桥之后，再往前走不远，道路收缩，脚下再度变成了砖石铺的路。
