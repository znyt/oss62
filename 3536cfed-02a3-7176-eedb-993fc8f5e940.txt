　　这无色火焰，沾染到这虚魔的一刻，顿时熊熊燃烧起来。
<br/>　　寻常武者对付虚魔，棘手无比，也正是因为这虚魔可以从魔意之中一次次的重新凝聚，难以斩杀。
<br/>　　可是一旦是燃烧上这纯阳天火，这虚魔周围的魔意，都被燃烧殆尽。
<br/>　　这虚魔的气息，在这纯阳天火的燃烧之下，越来越弱，一点的气息都不在存在，当即毁灭。
<br/>　　在楚风眠杀了这虚魔之中，楚风眠感觉到了在这魔意之中本来是蠢蠢欲动的虚影魔影，纷纷退去了。
<br/>　　楚风眠掌控纯阳天火，对虚魔这等依附魔意而生的魔族，有着极大的克制。
<br/>　　纯阳天火，足以是将他们身边的魔意都给全部燃烧殆尽，让他们都没有可以再度借助魔意凝聚身形的机会。
<br/>　　“这些魔族，正好是可以当做心魔之王的补品！心魔之王！”
<br/>　　楚风眠看到周围无数逃窜的魔族，嘴角露出了几分冷笑，一声命令之下，这心魔之王便是钻入到了魔意之中。
<br/>　　啊啊啊！
<br/>　　一时间无数魔族惨叫的声音响起。
<br/>　　在楚风眠的身边，无数窥探这，打算来出手偷袭对付楚风眠的魔族，纷纷是被心魔之王抓住。
<br/>　　被心魔之王所无情的吞噬，每吞噬一尊魔族，心魔之王的力量便是会强横许多。
<br/>　　平日里在大罗天星域之中，楚风眠也很难是寻找到大量的魔族，来给心魔之王吞噬。
<br/>　　现在这正是最好的机会。
<br/>　　楚风眠一路向前飞行，心魔之王便是守护在楚风眠的身边，大肆猎杀这，打算来出手偷袭楚风眠的魔族。
<br/>　　其中几尊实力强悍，心魔之王无法对付的魔族，也是由楚风眠亲自动手，将他们轰杀之后，再由心魔之王的吞噬。
<br/>　　吞噬了越来越多的魔族，心魔之王的力量也越来越强横，后来心魔之王守护在楚风眠的身边，已经是没有什么魔族，赶来靠近楚风眠了。
<br/>　　“这魔意，已经是快到尽头了。”
<br/>　　楚风眠一路行走，他已经是察觉到，周围的魔意已经是越来越弱了。
<br/>　　看来楚风眠已经是即将要穿越这一片魔意了。
<br/>　　魔意变的稀少无比，楚风眠目光所见的方向，也看的更加清楚了几分，在他面前的不远处，浮现出了一座山谷。
<br/>　　在这山谷之内，突然之间一条金龙，冲天而起。
<br/>　　这一条金龙，并非是真正的金龙，而是以灵力所化，耀眼无比。
<br/>　　“这是圣龙秘境的召集令？”
<br/>　　楚风眠看到这金龙的一刻，脸色微微一变。
<br/>　　这一条金龙，正是圣龙秘境内部的召集令，足以召集周围圣龙秘境的弟子。
<br/>　　“过去看看。”
<br/>　　楚风眠观察了一眼，便是身躯一闪，向着那金龙的方向飞了过去。
<br/>　　如今在这佛寺之中，楚风眠对佛寺还是一无所知，仅仅凭借他一个人，想要弄清楚这佛寺之中的情况，无比困难。
<br/>　　现在正好是碰到了其他圣龙秘境的弟子，楚风眠也可以去从他们的口中，打听一番情报。
<br/>　　楚风眠身形连连闪烁，很快便是逼近到了山谷之中，他的遁光便是缓缓的慢了下来。
<br/>　　楚风眠也没有贸然进入其中，而是一点点的靠近着山谷的入口，就当楚风眠靠近的一刻，两道身影飞了过来。
<br/>　　“你是谁？是圣龙秘境弟子？那一宗那一堂的弟子，怎么独身一人在这佛寺之中？”
<br/>　　这两道身影，都是八劫古帝的弟子，应该都是圣龙秘境的圣子。
<br/>　　不过看起来的样貌，却是无比陌生，显然是跟楚风眠没有见过，应该是之前并不在圣地之中，在外历练的圣子。
<br/>　　他们二人见到楚风眠的到来，眼神之中都露出了几分警惕之色。
<br/>　　“我是……”
<br/>　　楚风眠还没有开口，在山谷之内，突然走出来了一道熟悉的身影，看到楚风眠，急忙走了过来。
<br/>　　“两位师弟，这位是圣龙宗的楚师兄。”
<br/>　　这一道身影到来，对着楚风眠客气的说道。
<br/>　　“没想到又见到了楚兄。”
<br/>　　这一道身影，正是炎撼世。
<br/>　　之前在那一方小世界分别之后，没想到在这佛寺之中，楚风眠又跟炎撼世相见了。
<br/>　　“圣龙宗？楚风眠？”
<br/>　　那两名八劫古帝，听到炎撼世的话，大吃一惊道，看向楚风眠的眼神之中，猛然露出了几分警惕之色，甚至是有着几分敌意。
<br/>　　虽然只是一扫而过，这两名八劫古帝的脸上，又是露出了几分笑容。
<br/>　　但是这却瞒不过楚风眠的双眼。
<br/>　　“原来是楚师兄，请进，这一次我们五灵宗，正巧是在佛寺之中寻找到一处宝藏的踪迹，但是缺少人手，有楚师兄相助，便是有如神助，楚师兄还请进入山谷之内休息。”
<br/>　　说着，这两名八劫古帝，便是让开了一条通路。
<br/>　　楚风眠察觉到其中的古怪，脸色却是不动声色，一步上千，踏入到了这山谷之中。
<br/>　　炎撼世，则是仅仅跟随在楚风眠的身边，一同步入到了山谷之中。
<br/>　　进入到了山谷之中，楚风眠便是察觉到了不少的气息，足有上百位古帝，位于这山谷之中。
<br/>　　看起来，皆是五灵宗的圣子，不过许多人的模样，都是无比陌生，楚风眠都曾经没有见过。
<br/>　　“炎师弟，发生了什么事？这些人是谁？
<br/>　　楚风眠表情不动声色，却是悄然之间的对着楚风眠传音说道。
<br/>　　这些五灵宗圣子之中，不少人看向楚风眠的目光之中，都带有着几分敌意。
<br/>　　楚风眠之前在赤红大陆之中，曾经救过不少五灵宗圣子一命，哪怕是跟其他人素未相识，也不应该对楚风眠有着如此敌意才对。
<br/>　　而且楚风眠还发现了，除了炎撼世之外，像是木麟，还有其他的一些五灵宗圣子，如今都已经不在这山谷之中了。
<br/>　　“这些也都是我们五灵宗的圣子，不过他们都是属于另一派系的人，这些人，都是之前先前一步，来到九域的，我跟木麟师兄等人，是随后而来的。”
<br/>　　“木麟师兄，之前被魔族偷袭，身受重伤，现在已经是回到圣地之中去了，还有许多的五灵宗弟子，也都身受重伤，我也是迫不得已，才跟他们联合在了一起。”
