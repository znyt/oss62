　　七彩霞光，依然笼罩这整个天元商会。
<br/>　　突然间七彩霞光缓缓打开，一道身影从其中走了出来。
<br/>　　楚风眠手持那七彩晶石，从天元商会之中离开，如今楚风眠的脸上带有着几分淡淡笑容。
<br/>　　这一次在这天元商会的遗迹之中，楚风眠收获无数。
<br/>　　首先是得到了九龙神苍鼎，这件位列天下百鼎的药鼎，又是得到了大量天元商会的宝藏。
<br/>　　当然对楚风眠收获最大的，还是现在楚风眠已经是踏入了帝尊境，更是一举踏入到了八劫古帝的层次。
<br/>　　现在楚风眠踏入到了八劫古帝的境界，一身实力更是跟九劫古帝没有了任何的区别。
<br/>　　他身上的天地法则数量，甚至是比起九劫古帝还要多，在整个大罗天星域，楚风眠都绝对称得上是强者行列了。
<br/>　　“恩？这一方小世界里，居然没有人？五灵宗的圣子，都已经是离开了？”
<br/>　　楚风眠灵识扫过，突然发现，如今在这一方小世界内，已经是没有了其他人的气息。
<br/>　　木麟，炎撼世一行人，好像都是离开了这一方小世界。
<br/>　　这让楚风眠有些奇怪。
<br/>　　楚风眠踏入到了那天元商会之中，在加上突破境界，一共才过了一天左右的时间。
<br/>　　以那些五灵宗圣子的伤势，一日的时间，他们的伤势都未必可以恢复的完，应该是在这一方小世界之中好好的恢复伤势的才对。
<br/>　　可现在这些五灵宗圣子，却都是离开了，不知去向。
<br/>　　“难道是赤红大陆之中，发生了什么事？”
<br/>　　如果没事，这些五灵宗圣子不应该是这么匆匆离去的才对。
<br/>　　楚风眠想了一下，便是向着小世界的出口处飞去。
<br/>　　这一方小世界之中的宝藏，就是那天元商会的遗迹，现在其中的宝藏都已经是被得到了。
<br/>　　在留在这一方小世界之中，也没有任何的意义。
<br/>　　楚风眠在突破之后，遁光的速度也要比起之前快上了三四倍的地步。
<br/>　　本来楚风眠拥有始祖天龙的血脉，他的遁光都快到了一种不可思议的地步，在准帝时，就接近九劫古帝级别的遁光了。
<br/>　　现在楚风眠突破了之后，速度更是快到惊人，几个呼吸之间，楚风眠在这一方小世界之中连连穿梭，就已经是前往了小世界的边缘。
<br/>　　随即身形一闪，楚风眠刹那之间，离开了这一方小世界，重新回到了赤红大陆之中。
<br/>　　刚刚踏足赤红大陆，楚风眠就闻到了一股浓烈的血腥味。
<br/>　　“这附近，刚刚发生过战斗？”
<br/>　　楚风眠闻了一下，这血腥味还无比清晰，最多就在一两个时辰，这里还发生过战斗。
<br/>　　“武者的血，还有魔族的血？”
<br/>　　在这血腥味之中，楚风眠足以感觉到了几分魔意。
<br/>　　居然是魔族，莫非是魔族也在这赤红大陆之中动手了？
<br/>　　大量的魔族已经是潜入到了九域之中，楚风眠在刚刚回到九域的时候就发现了这一点。
<br/>　　不过比起大罗天星域，阎罗星域的武者，魔族却是显得低调得多，一直是在潜伏这，没有出手。
<br/>　　可现在既然是出现了魔族血迹的话，多半是魔族也出手了。
<br/>　　这九域本来就无数势力的强者在其中，错综复杂，一旦是魔族再度出手，只怕是会更加的麻烦。
<br/>　　就在楚风眠思考的时候，一道刀锋突然间的飞了出来。
<br/>　　“楚风眠，没想到你居然躲在了这里！给我死！五行神刀！”
<br/>　　这一刀刀锋之上，蕴含五行之力，凝聚在一起，相辅相成，向着楚风眠直接斩杀了过来。
<br/>　　偷袭？
<br/>　　楚风眠刹那之间感觉到了几分危险，身形一闪，猛然退去。
<br/>　　在他刚刚所在的地方，一道刀锋斩落而下，将地上都打出了一道鸿沟，这一刀是绝对的杀招。
<br/>　　一旦是硬抗，八劫古帝也是必死无疑。
<br/>　　这至少是一名八劫古帝实力的武者出手。
<br/>　　“敢偷袭？给我滚出来！”
<br/>　　楚风眠目光冰冷无比，扫视了周围一眼，猛然伸出手来对着天空之上一抓。
<br/>　　只见就在这天空之上，一道身影被直接的抓了出来，这是一名年轻男子的样子，虽然年轻，但是一身实力已经是达到了八劫古帝，比得上是圣龙秘境的圣子。
<br/>　　“五行天的人？”
<br/>　　楚风眠看到这年轻男子的衣着，便是看的出来，此人是属于五行天的人，这种实力，应该就是五行天的圣子。
<br/>　　大罗天星域的四大势力，不止是圣龙秘境拥有圣地，这五行天，齐林山，沧海都三方，也都有着像是圣地一样的存在。
<br/>　　他们也有着秘密培养的天才圣子，眼前这年轻男子，就是五行天的一位圣子。
<br/>　　五行之术，修炼到了无比精湛的地步，那五行神刀，都不是实体，而是他以五行灵力凝聚而出的。
<br/>　　但是真正的锋芒，比起真正的刀锋，还要锋利的多。
<br/>　　“能看出本少的身形，看来你还有些本事，怪不得小天主特意嘱咐要来杀了你。”
<br/>　　那年轻男子，看向楚风眠嘴角露出几分冷笑之色道。
<br/>　　“小子，你是准备束手就擒，还是想要反抗？束手就擒的话，可以少受一些皮肉之苦。”
<br/>　　“束手就擒？就凭你们几个废物？”
<br/>　　楚风眠看向这年轻男子，眼神之中闪过几分不屑道。
<br/>　　“不用在躲躲藏藏了，都出来吧！”
<br/>　　随即，楚风眠的目光看向了天空之上。
<br/>　　不久，天空之上，突然出现了一道裂痕。
<br/>　　又是两道身影，从中走了出来。
<br/>　　“好眼力，看来敢毁了执法堂跟万龙堂，还是有着几分本事。”
<br/>　　这出来的，又是一名男子，不过这男子的手臂，却是不像血肉，而是一种如同树木一般的质感。
<br/>　　肉身木化，这是修炼木系功法，接近巅峰才会出现的情况。
<br/>　　在大罗天星域之内，只有一方势力的强者，才将木系功法，修炼到了如此地步，那就是齐林山。
<br/>　　而最后的一名年轻男子，则是一身澎湃的水系灵力，这男子不是武者，而是一位妖神，一种海妖成就的妖神，在他的额头之上，还有着几枚鳞片，证明他的身份。
