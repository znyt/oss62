　　不周印！
<br/>　　“呼——！”深呼一口气，元始看着手中大印说道，
<br/>　　“你是我在不周山上领悟的。不周印，从此之后，你就叫不周印了。你代表我不屈的信念，同时也代表盘古不屈之意，就像这不周神山，顶天立地，战天斗地，永不屈服。”
<br/>　　“虽然你现在还不完善，但我相信早晚你会如同现如今的不周山一样，顶天立地，俯瞰天地，威压众生！”元始心中默默道。
<br/>　　元始抬头，看向上天，循着不周山山体向上望去，想要追寻那山顶所在，但是，他失望了。目之所及，透过层层云彩，并没有看见山顶，看到的只是一段永远望不见顶的山体罢了。
<br/>　　“看来，我是还要继续攀爬啊。”元始道。
<br/>　　此刻，元始心中，尽是一股不屈战意，他想看看，这不周神山，到底有多高！
<br/>　　头顶不周印，元始再次踏步，继续向着山顶方向攀爬而去。
<br/>　　盘古意志，威压天地，但元始同样心志坚定，都能够与盘古大战之人，意志能不坚定吗？感悟着不周山的磅礴，心中升起一股战天斗地的豪情，驱使着他，继续向前。
<br/>　　一步，两步，三步，……
<br/>　　一天，两天，三天，……
<br/>　　时间在无声无息之间不断流逝，转眼间，就是数百年的岁月过去了。
<br/>　　登天之难，难以想象，花了数百年时间，元始也不过是攀爬了十二万九千六百里之高而已，因为，越往上，天地威压就越是强大。强大如他，有时也要停下来仔细参悟很久，等适应了此处威压，将此处的意志转化在不周印中，他才继续前进。
<br/>　　而，他的不周印，却也越来越强大了，仅仅释放的威压，就能与天地威压抗衡了。神山模样的不周印，也越来越清晰了。
<br/>　　天，依旧是望不到尽头，但，此时此刻的他，却已然处在九霄云层之中，身边随处可见，流云飞舞，缥缈如烟。
<br/>　　蓦然，元始停住了脚步！
<br/>　　不是因为他已经攀爬到了不周山的顶峰，只因为，从此处再往上，已经不再如先前一般一片坦途了。
<br/>　　在往上，便是一片无限宽广，覆盖了整个洪荒天地的庞大天幕，一眼望去，尽是晶莹深青，丝毫不见杂色，震撼人心！
<br/>　　九天罡风层！
<br/>　　九天罡风层，正是划分洪荒天地的分界线，亦是天地之间的保护膜，元始还从未探索过，亦未曾听说过，这九天罡风层，到底有多厚？
<br/>　　向上望去，这不周山体虽然缩小了许多，但，依旧没有到顶，剩下的山峰，好似一柄利剑，直插入九天罡风层之中！
<br/>　　“还没有到顶吗？”元始感叹道，接着再次看向上空，道：“不过，九天罡风层又如何，我元始倒想看看，你有多厉害？”
<br/>　　不过，就算如此，但，不周山的高度，还是超乎元始之想象，要知道，此处距离洪荒大地足足有十二万九千六百里之高啊！
<br/>　　洪荒大地，计量单位与后世不同，这里所说的一里，可不是一千米，而是一万丈！
<br/>　　万丈一里，千吨一钧！
<br/>　　眼下此山，他已经攀到了十二万九千六百里，折算为丈，那就是十二亿九千六百万丈之高啊！
<br/>　　两者相比较，就知道这天柱不周山到底高到何种程度了！
<br/>　　不过，也正是如此，不周山才能成为天柱，顶天立地，镇压乾坤！
<br/>　　“继续向上吧！都已经到了此处，更不能放弃，况且，要是我能够顶住压力，克服重重困难，攀上不周神山峰顶，我定会得到一场蜕变，对于我来说，这是求道路上，不可多得的磨砺！”
<br/>　　抬眼看着上方一层看似平静，实则酝酿着无尽凶险的青色天幕，再次坚定心中意念，元始深吸一口气，蓄足力气，头顶不周印，再次踏出这关键的一步，闯入了隔绝天地的九天罡风层！
<br/>　　伴随着元始一步踏入九天罡风层之中，霎时间，宛若一个蝴蝶煽动翅膀，在远处掀起了滔天巨浪，瞬间引动这个天幕，发出剧烈无比的反应。
<br/>　　罡风怒吼，席卷过来，顿时，原本平静流动的九天罡风层，随着元始的踏入，变得狂暴起来。
<br/>　　瞬间，一股毁天灭地的能量洪流，直奔元始而来。
<br/>　　痛！
<br/>　　剧痛！
<br/>　　这一刻，一股难以形容的剧痛席卷元始的整个身体，纵然是元始大罗金仙圆满境界也难以抵挡。此时此刻，在九天罡风层中的无数毁灭之力的侵袭之下，元始的身体难以承受，瞬息之间，就已经遍体鳞伤！
<br/>　　元始催动《玄黄不灭》，放出无上玉身，霎时间，一座高达数十万丈的洁白高贵的无上玉身，出现在九天罡风层之中。
<br/>　　罡风化作毁天灭地的能量洪流，充斥在元始全身，从周身十万八千个毛孔中钻进去，再吹出来，这一进一出，体内血肉之中的无尽生机，全部被罡风带走，侵蚀，血肉被吹散，骨骼被侵蚀，五脏六腑化为粉碎，要是抵挡不住，就会死。
<br/>　　元始被这无尽罡风侵蚀，但他并没有放弃，而是在运转《玄黄不灭》的功法，因为这罡风对于他来说，既是灾难，也是机缘，因为罡风吹走的不仅仅只是血肉生机，还有那藏在血肉之中的无尽杂质。
<br/>　　这罡风，他要是度过了，他的《玄黄不灭》就会更上一层，度不过，那就烟消云散。
<br/>　　伴随着元始运转玄功，他的身体各处伤口在快速愈合，无尽新嫩更多血肉在伤口之处产生，弥补他破碎的身躯，这些新长出来的血肉，经过罡风的淬炼，在强度上，变得比以前强上不少！
<br/>　　只是，很快，这些新长出来的血肉，又再次被罡风攻击，再度皮开肉绽、遍体鳞伤！
<br/>　　不过，之后它又再次长出来，再度被吹散，……就这样，无限循环下去！
<br/>　　“真没想到，在此处竟然能淬炼增强我的无上玉身？！”
<br/>　　感受到肉身的增强，元始心中微喜，当下，全力运转《玄黄不灭》，虽然这样的方法很是磨人，但，只要肉身能够进步，元始也是认了。
<br/>　　一步，两步，三步……
