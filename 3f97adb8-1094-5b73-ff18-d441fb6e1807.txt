        所以，傅夜沉用脚趾头想想都知道，这床上的人，十有八九是落以柔。
<br/>
<br/>        她今天来给他送文件，但是他没料到她到现在还未离开。
<br/>
<br/>        “起来，把衣服穿上，给我滚！”傅夜沉冷冷地呵斥，随即拿起床头柜上的座机给管家刘叔打了通电话，“刘叔，麻烦你过来一趟，给我换床被褥和床单。”
<br/>
<br/>        临海城的依山别苑，他留给了十爷冷逸峰和季雅珊居住，而他这段日子要住在美国，需要管家打理日常起居，于是就把刘叔也带来了。
<br/>
<br/>        为了不引起安防局的人注意与怀疑，刘叔特意去给傅老太爷递了辞呈，告老还乡，但是却换了个身份陪同傅夜沉一起出国来到了这里。
<br/>
<br/>        落以柔听这话立马抱着被褥从床上下来，红着脸，眼中含泪，咬牙切齿道：“我是个处，很干净！你没必要嫌我脏！”
<br/>
<br/>        傅夜沉下意识地瞥了落以柔一眼，拿起雪茄盒，朝落地窗边的躺椅上坐去。
<br/>
<br/>        他抽出了一根香烟，拿起火机点然后，吸了一口，慢条斯理地说道：“你不脏，是我有洁癖，受不了床上有其他女人的气味。”
<br/>
<br/>        “筱颖都已经死了半年了！你到底要多久才能忘记她？我答应过她，要好好照顾你。可你却在折磨你自己！”落以柔潸然泪下。
<br/>
<br/>        童筱颖出事后，她向他撒了谎。
<br/>
<br/>        明明是她出卖了童筱颖，她却把童筱颖夸赞得十分伟大。
<br/>
<br/>        说是童筱颖自爆了身份，承认了她童筱颖才是他傅夜沉的女人，这才保全了她落以柔，以至于她童筱颖独自赴死。
<br/>
<br/>        落以柔心知肚明，自己的谎言，他傅夜沉一定是信了。
<br/>
<br/>        只不过，他傅夜沉是到底信了她落以柔的话，还是信了童筱颖会是那么一个为了救她落以柔而牺牲自己的傻女人。
<br/>
<br/>        到底是哪一种，落以柔猜不透，也揣摩不出傅夜沉的心思。
<br/>
<br/>        以前，她从来没有见过他傅夜沉抽烟，但是自从童筱颖死了后，他就开始抽烟了，这半年的时间里，雪茄就从未离过手。
<br/>
<br/>        傅夜沉噙着雪茄，狠狠地吸了一口，又夹在指尖，吐出了嘴里的烟圈：“你知道我现在每次看到你，我心里在想什么吗？”
<br/>
<br/>        落以柔顿时受宠若惊了般抬眸看向傅夜沉那边，她以为，他会说出什么动听的话来。
<br/>
<br/>        结果……
<br/>
<br/>        “我在想，死的那个人，是你落以柔该多好！”
<br/>
<br/>        傅夜沉的话语清冷，斩钉截铁，甚至冷酷无情。
<br/>
<br/>        他的筱颖，怎么就那么傻？
<br/>
<br/>        不惜自己肚子里的孩子，也要去救季雅珊；如今连自己的命都可以不要，还把她落以柔送进安全线里，自己去赴死。
<br/>
<br/>        明明，他的筱颖也可以自私，可为什么，她就不自私点呢？
<br/>
<br/>        而这件事情的真相，也就只有她落以柔知道。
<br/>
<br/>        是她把童筱颖逼入绝境的。
<br/>
<br/>        但是她没想到，童筱颖会在那个时候还想着要救她脱险。
<br/>
<br/>        其实，那个时候，是她落以柔特意暗示，要她童筱颖去赴死。
<br/>
<br/>        谁都知道，那个集团的人抓她们，就是给他傅夜沉设了一个陷阱。
<br/>
<br/>        傅夜沉若是去救她们，就等于是去赴死。
<br/>
<br/>        而童筱颖做了最对的选择。
<br/>
<br/>        倘若换做是她落以柔，她绝对没这胆量。
<br/>
<br/>        因为，傅夜沉还活着，她也想活着。
<br/>
<br/>        只有活着，她才有希望成为他傅夜沉的女人。
<br/>
<br/>        在连华生的口中，得知童筱颖死了的消息后，落以柔发现自己竟然在没心没肺地笑。
<br/>
<br/>        她以为她赢了，但是她发现，她还是输给了童筱颖。
<br/>
<br/>        在男人心里，得不到的，永远都是最好的。
<br/>
<br/>        童筱颖一死，她便成了他傅夜沉心里的那抹白月光。
<br/>
<br/>        “我想过要替筱颖去死，但是她一直在撇清我和你的关系，一直在说她才是你的女人。更重要的是，那个戴着假面眼镜的男人信了她的话。”落以柔声泪俱下，抬起手来，捂着脸，痛哭了起来，“我也想死的那个人是我，不是筱颖。这样的话，你就不会伤心难过。大哥，我可以为你殉情，当然也愿意成全你和筱颖。但是筱颖说，希望我能好好照顾你的余生。大哥，我的心也好痛，真的好痛……”
<br/>
<br/>        但是她的哭声，还有她的哭诉，却令傅夜沉感到心烦意燥。
<br/>
<br/>        “你给我闭嘴，别哭了！”傅夜沉一声冷斥，成功地止住了落以柔的哭泣。
<br/>
<br/>        此时，房门被人叩响。
<br/>
<br/>        落以柔心想着可能是刘叔过来了，于是拿起床头柜上的衣服，躲被子里穿上。
<br/>
<br/>        下床后，她去给刘叔开了门。
<br/>
<br/>        刘叔抱着新被单被套，看到落以柔时一脸诧异。
