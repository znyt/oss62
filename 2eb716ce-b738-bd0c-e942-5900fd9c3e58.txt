<br/>“紧张？为什么要紧张？我现在可是兴奋到了极点！”楚林飞举着手中那对雌雄双斧兴奋的叫道。<br/>　　<br/>　　这对雌雄双斧是双属性的魔法武器，也是李振邦拿给他用的，这雌雄双斧顾名思义一雌一雄，每一把都是双面开刃的大斧。<br/>　　<br/>　　雄斧是火属性，每一次攻击不但会让对手感觉到灼热，还附带着火属性的爆裂攻击，如果攻击到了人，还会有持续的灼烧伤害，让人苦不堪言。<br/>　　<br/>　　雌斧不是水属性，而是冰属性。每次攻击都会附带着冰霜攻击，不但会让对手感觉到寒冷，还会影响对手的速度。如果攻击到对方的身体，还会将攻击的地方冻僵，影响行动效率。<br/>　　<br/>　　这一对雌雄双斧可比以前精钢打造的车轮板斧要好的多，楚林飞对这对斧子那可是爱不释手，天天斧不离手，就是睡觉都抱着。<br/>　　<br/>　　其他人也没有紧张的感觉，人人爱抚着自己的武器，一股跃跃欲试的样子。这段时间他们不但训练着彼此的默契度，还训练着和武器的亲切度。一个合格的战士对自己的武器也会格外爱惜，武器堪比自己的第二生命。<br/>　　<br/>　　已经很久没有人挑战团队了，李振邦一行人这一次的选择让他们成为了舆论的焦点。甚至一些学生开出了赌局，赌他们能撑住几回合，几分钟。<br/>　　<br/>　　赌他们赢的比例已经达到了1:1000，但是却没有一个人选择。赌他们能进入皇家佣兵团的赔率也达到了1:100，赌他们失败的赔率已经达到了1:0.01。即便如此，很多人依然选择仅仅只有0.01盈利的赌局，可见众人对他们并不看好。<br/>　　<br/>　　众人对这赌局有些了解，甚至他们还参加了赌局，不过他们都是蒙面参加，并没有透露自己的身份，而且赌的并不大，都是压的是进入皇家佣兵团。<br/>　　<br/>　　张大友和李振邦商量了很久，最后压了一百金币在进入佣兵团赌注上。这几天李振邦去了平民委员会，将自己的工资拿了出来，留下了两百多金币，其他的都送给了张大友。<br/>　　<br/>　　李振邦用这两百金币赌自己队伍会赢，二百金币对他来说和两个铜币没什么区别，就算输了也无所谓。万一如果侥幸赢了的话，那可就是二十万，他不缺这二十万，但是短时间内可以解决很多燃眉之急。而且这二十万给张大友的话，他心里也容易接受一些。<br/>　　<br/>　　出乎所有人意料的是，今年加入皇家佣兵团的比赛竟然是院长金万年亲自主持的，可见他对今年比赛的重视，要知道往年都是一名副校长来主持，金万年院长连面都不露。<br/>　　<br/>　　金万年院长显然不是一个喜欢废话的人，开幕式很简单，他并没有发表什么长篇大论，只是简单的鼓励了一下大家，希望大家好好发挥，取得好成绩。<br/>　　<br/>　　金万年院长说完以后，皇家佣兵团的一名副团长也发表了讲话，主旨就是希望大家努力去的好成绩，就算没有成功也不用气馁，明年还会有机会云云。<br/>　　<br/>　　两个人演讲完毕以后，选手们开始抽签进行比赛。抽签很简单，同一年级的人随机分成两批，每人从透明的抽号箱中抽取一张卡片，卡片上会写着一个号码，两批人中号码相同的互为对手。<br/>　　<br/>　　如果双方人数相差一个的时候，会有一个人幸运的轮空，直接进入下一局。确定对手以后，会有专门的负责人安排选手们进入某一个场地比赛。比赛不分职业，都是1V1，一共十个场地，每个场地都是50*50米的方形区域。<br/>　　<br/>　　正常情况下，召唤兽法师在1V1的时候并不占便宜，所以皇家佣兵团里面的召唤兽法师并不多。而这为数不多的召唤兽法师中，大部分还是走后门进去的。<br/>　　<br/>　　召唤兽法师在皇家佣兵团里面很是受排挤，不过召唤兽法师们也清楚自己的实力，所以他们也很低调。哪怕佣兵团内已经上五年级的召唤兽法师，碰到三年级的战职者，也会低声下气的。学院内尚且如此，更何况外界呢！<br/>　　<br/>　　召唤兽法师即便是面对魔法师学徒也不得不低声下气的，谁知道对方会不会成为一名强大的魔法师呢！但是召唤兽法师，永远都是召唤兽法师。<br/>　　<br/>　　李振邦看到了一些上台比赛的召唤兽法师会携带着等级不低的契约兽上场。选择在场上召唤魔兽的，基本都是那些平民子弟，他们没有实力和金钱去抓取高等阶的魔兽做契约兽，只好在赛场上搏一把。<br/>　　<br/>　　召唤兽法师的未来不在于召唤出来的召唤兽有多强大，而在于契约兽的强大，这不得不说是这个职业的悲哀。<br/>　　<br/>　　李振邦虽然也是凭借着契约兽，但是毕竟这些契约兽是他通过其他手段召唤出来的，所以他和那些召唤兽法师还有些不同。不过外人看来，他也是一名通过契约兽来提升自己的召唤兽法师罢了。<br/>　　<br/>　　赛场上的战斗进行的很快，期间也有几名召唤兽法师运气不错，召唤出来了强大的魔兽，奠定了比赛的胜局。<br/>　　<br/>　　但是凡事总有例外，其中一名三年级的召唤兽法师竟然召唤出来了一条一个脑袋的地狱犬。尽管仅仅是一个脑袋的地狱犬，但那也是强大的七阶魔兽。<br/>　　<br/>　　不过可惜的是他并没有赢得胜利，因为他召唤出来的瞬间，就因为精神力耗尽昏迷了，而召唤兽不得不遗憾的消失了。<br/>　　<br/>　　三年级的比赛相对来说是比较激烈的，一些人刻意没有升到四年级就是因为三年级的级别相对要低一些，赢的机会更大一些。<br/>　　<br/>　　四年级和五年级比赛人数要少了很多，所以仅仅大半天的时间，比赛就结束了。<br/>　　<br/>　　团队赛只有李振邦一个队伍，也是大家十分期待的队伍，只不过多数人期待的是李振邦他们什么时候会输。<br/>　　<br/>　　对面一共七支队伍，李振邦两个姐姐所在的队伍有五个人，她们朋友的队伍一个队伍是六个人，一个队伍是四个人，她们对手的队伍和那个打酱油的队伍都是五个人，有一个队伍竟然只有三个人，最后一个队伍是七个人。<br/><br/><br/>
