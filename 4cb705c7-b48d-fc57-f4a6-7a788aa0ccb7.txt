        “那是怎样？”恭如兰反问，起初在集训的时候，她有看到那傅大教官抓住蓝筱蝶的肩膀一脸焦急的样子。但因为隔得太远，她也不知道傅大教官跟蓝筱蝶说了什么。
<br/>
<br/>        后来，那声枪响，和周遭女生们的尖叫声，恭如兰自己也吓得半死，倒是看到蓝筱蝶直往筱颖的大哥冷教官怀里扑。
<br/>
<br/>        “哎呀！他只不过是问我，有没有看到筱颖。不过了，在他的怀里真的很有安全感噢！长得帅，又有绅士风度！筱颖，你家大哥真的是太招人喜欢了！蓝筱蝶随之看向童筱颖，抬手指了指童筱颖的鼻子，忽然露出了狐疑地表情，“嗯？你和你家大表哥的关系肯定不简单！”
<br/>
<br/>        “当、当然不简单，有血缘关系啊！”童筱颖心虚地支支吾吾。
<br/>
<br/>        蓝筱蝶竖起食指摇了摇：“NO、NO、NO！你家大表哥紧张你的样子，可不是只有血缘关系这么简单啊！我怀疑！”
<br/>
<br/>        “嗯？”童筱颖和严恩恩异口同声。
<br/>
<br/>        “你和你家大表哥有JQ！”蓝筱蝶怕是福尔摩斯附体了。
<br/>
<br/>        童筱颖干笑：“什么‘JQ’，你想多了，我大哥他有未婚妻。”
<br/>
<br/>        “筱颖，我能懂你们心里的苦，近亲不能结婚，爱上了，就是地狱。”蓝筱蝶感慨着。
<br/>
<br/>        童筱颖汗颜了：“真不是你想的那样……”
<br/>
<br/>        “你大表哥未婚妻漂亮吗？”恭如兰才不关心蓝筱蝶所说的，重点是傅大教官的未婚妻啊！
<br/>
<br/>        童筱颖下意识地便想到了苏漫雪，那张跟自己有着一模一样容颜的曾经的闺蜜。
<br/>
<br/>        漂亮？不漂亮？
<br/>
<br/>        为了金钱，苏漫雪竟然可以整容成她的样子，去欺骗傅夜沉。
<br/>
<br/>        可她却没料到，傅夜沉似乎很好骗，他对苏漫雪确实不错，苏漫雪要什么，他就给什么。
<br/>
<br/>        或许，大概，大哥傅夜沉也误认为那晚救他的人是苏漫雪吧！
<br/>
<br/>        就算真误会了，童筱颖也不打算去解释。
<br/>
<br/>        “到底漂不漂亮呀？”蓝筱蝶追着问。
<br/>
<br/>        良久后，童筱颖回过神来，苦笑着说：“漂亮。”
<br/>
<br/>        她这么说的话，算不算是在自己夸自己？
<br/>
<br/>        蓝筱蝶和恭如兰又接着开始跟她八卦起傅夜沉的未婚妻的事情来，堪比派出所的民警上门来查户口。
<br/>
<br/>        只有严恩恩一个人沉默着，对傅夜沉的事情完全不感兴趣。
<br/>
<br/>        打完靶后，她们临海大学的大一新生，终于可以收拾行李返校了。
<br/>
<br/>        上了返校的校车，童筱颖和严恩恩坐在了一起，聒噪的蓝筱蝶和恭如兰坐在了一块。
<br/>
<br/>        严恩恩真的是一个很安静的女孩子，她的话虽然不多，但也不是那种完全不理人，她还是会关心身边的人。
<br/>
<br/>        童筱颖很喜欢和这样的女生待一块，因为她会给周遭的人带来一种岁月静好的感觉。
<br/>
<br/>        回到宿舍后，童筱颖和严恩恩拿了衣服和洗浴用品提着桶子一起去了澡堂，蓝筱蝶和恭如兰就打算在宿舍里冲个凉。
<br/>
<br/>        宿舍里明明只有四个女生，却因为性格上的原因，还是分成了两个傅昼团体。
<br/>
<br/>        童筱颖自然而然地和严恩恩走到了一块，蓝筱蝶和恭如兰一块。
<br/>
<br/>        就在童筱颖和严恩恩洗完澡从澡堂里出来后，回宿舍的途中，一辆白色的豪华跑车停在了校道边，很招人瞩目，加上，倚在车身前的男人，英俊倜傥，修长指间夹着一根烟，尽显纨绔贵气。
<br/>
<br/>        童筱颖注意到那男人时，一眼就认出了他来。
<br/>
<br/>        是滨江城里那个爱泡“夜惑”，与无数女人暧昧不清，走到哪都耀眼张扬的季少。
<br/>
<br/>        他的全名叫什么，童筱颖一时间没想起来，给忘了。
<br/>
<br/>        严恩恩无意间也朝季运恒看了过去，恰巧，季运恒的目光往这边看了过来。
<br/>
<br/>        “筱颖。”严恩恩忽然弱弱地唤了声。
<br/>
<br/>        童筱颖应道：“嗯？”
<br/>
<br/>        “那个男人……”严恩恩的话还未说完。
<br/>
<br/>        季运恒看到童筱颖时，心里确认是她后，于是掐灭了手里的烟，扔进了一旁的垃圾桶中，朝童筱颖和严恩恩走了过去。

<br/>        这个男人在“夜惑”里是什么得性，到了外边，自然也是个什么得性。

<br/>        童筱颖的直觉告诉自己，这个男人少惹为妙。
