　　第590章
<br/>　　程大雷和小白狼相伴上路，二人同乘一骑，倒是悠哉悠哉。
<br/>　　没用几天时间，便从青州到了幽州。那次大战过后，幽州尚未恢复生息，入眼只见荒草枯槁，千里之地没有半点人烟。
<br/>　　戎族的铁骑杀过来时，人命如猪狗，大半人都成了戎族刀下亡魂；那运气好的，则逃到了南方，勉强能过得去日子。剩下一些人，没钱没势没关系，则是躲入了荒山野林中，成了天不收地不管的孤魂野鬼。
<br/>　　枯草黄土之下，说不得能踢出一具白骨。野林子里，说不得藏着那披头散发的野人。
<br/>　　程大雷和小白狼走在这里，想起那年的事，二人唏嘘感慨颇多。
<br/>　　“杨龙停死后，正义教又闹了一次，幽州千里成了荒地。这里，就没什么人烟了。”小白狼道。
<br/>　　“朝廷没再派人过了么？”程大雷牵着牛走在路上。
<br/>　　“幽州这里要人没人，要粮没粮，谁愿意过来。”小白狼呵呵笑了一声：“你问我这个，上面人想什么，我怎么知道。”
<br/>　　戎族并未占领幽州，但帝国事实上已放弃了对幽州的统治。程大雷叹了口气：“上面人想什么，的确没人知道。可不知道他们有没有想过，幽州是北地的最后一道险关，突破青牛山，整个帝国将无险可守。”
<br/>　　当初戎族大举入侵，亏得程大雷力挽狂澜，守住了青牛山。时至今日，幽州已成了荒地，帝国不在青牛山驻兵，说来说去，还是吃得苦头不够大。
<br/>　　小白狼皱起眉头：“你说，戎族会打过来么？”
<br/>　　程大雷摇了摇头：“这件事我也说不太好。以往戎族打过来，无非就是抢点人，抢点财物，倒是对帝国的城池没有多大兴趣。可野心这种东西，本就是一点点养起来的，帝国在军事上不争气，免不得戎族就起了吞天野心。”
<br/>　　“那么朝廷就不想想这件事么？”小白狼问。
<br/>　　程大雷道：“朝廷是有聪明人的，可想到未必有力气去做。如今帝国是二帝十八王，各路诸侯割据一方，都趁这个时间抓紧扩张势力。想着，是以后群雄逐鹿，争一争天下。但谁能想到，关外还有狼子野心的戎族，在盯着帝国的花花世界。有朝一日，戎族的大兵南下……呵呵，这帝国到底是谁家天下，一切还说不好喔。”
<br/>　　刚开始，小白狼还能跟得上程大雷的思路，但渐渐她便有些听不太懂了。她骑在牛背上，皱眉盯着程大雷的背影，半晌后悠悠道了一声。
<br/>　　“程大雷，你现在想的，比以前多了。”
<br/>　　程大雷咧嘴笑了笑，没有说什么，他心里道：我以前想得也很多，只是你不知道而已。
<br/>　　天色慢慢暗下来，到了冬天，天气一日比一日冷，夜也一日比一日长了。程大雷开始收拾晚上睡觉的地方，二人找了个避风的土坡，割草引火，小白狼去打了两只兔子裹腹。
<br/>　　她仍旧在琢磨着程大雷的话，过了半晌问：“你这次去戎族，就是为了这件事？”
<br/>　　“不是。”程大雷摇摇头，随之道：“也不全是，主要是找个人报仇。”
<br/>　　“为了个女人……”小白狼顿了顿：“你这处处留情的脾气，什么时候能改一改？”
<br/>　　程大雷正抱着烤兔子啃，听到这话，忽然怔住了。咧开大嘴道：“我哪里处处留情了，我很守身如玉的好不好。”
<br/>　　好杀往往和好色相伴相随，程大雷在江湖上杀名赫赫，不说外人，就连小白狼都如此想他。但在程大雷想来，他自己还是蛮检点的。
<br/>　　正这时候，远处的荒草无风自动，唿律律马鸣。小白狼顿时一激灵，没了给程大雷看玩笑的心情，立刻拔出刀，警惕的盯着四周。
<br/>　　就见一头灰马从荒草里冲出来，快活的打了个响鼻，差点一头将程大雷顶在地上。
<br/>　　程大雷愣了愣，随之手拍在马脑袋上，咧嘴笑道：“老伙计，你还活着呢。”
<br/>　　这匹灰马，正是程大雷在青牛山时的座骑。后来程大雷被杨龙停追杀，这瘦马也抛下了。却是没料到，兵荒马乱的，这么多年过去了，这瘦马还活着。
<br/>　　如今它可不瘦，这样糟糕的世道，苦了人，却是活了畜生。浑身毛发油光锃亮，和从前相比也精神许多。
<br/>　　小白狼呆呆看着这一幕，咂舌叹道：“幽州千里地，你们这样都能碰上，也不容易。”
<br/>　　程大雷拍拍灰马的脑袋：“我是个念旧的人，一向如此。”
<br/>　　其实进入幽州后，程大雷的情绪便不怎么好。戎族打过来一次，林少羽闹过一次，才让幽州成了今日的幽州。这两件事都和程大雷有些关系，而且，都不是很令人开心。也就逢着这匹瘦马，程大雷的心情才稍微好些。
<br/>　　第二天早上，二人继续赶路，遇到这匹瘦马后，二人都有了脚力，向前赶路的速度便也快了。从此地出发，穿过青牛山。程大雷在青牛山并未停留，虽然说是故地，但青牛山当年血流成河，尸横遍野，这故地……实在没办法令人怀念。
<br/>　　而马不停蹄赶往戎族的二人并不知道，此时此刻，戎族的情况其实并不比帝国好多少。
<br/>　　戎族占了关外草原，由数百个部落组成。这些部落也有大有小，有强有弱，一般情况是小部落依附大部落，大部落依附王族。互相之间也是你抢我，我抢你，弱肉强食，有过之而无不及。
<br/>　　燕不归属于戎族白狼部，而白狼部由依附于呼延一族。呼延一族在戎族算是颇有势力的，最起码，人家有姓氏啊。
<br/>　　当初呼延跋还在世时，呼延一族是实实在在的草原王族。草原上数百个部落，都以呼延跋为王。不过，后来呼延跋被程大雷杀了，呼延族本部也陷入争权夺势当中。这么多年过去，倒也不是戎族不想打帝国的主意，实在是他们自己还打得不可开交。
<br/>　　这次燕不归一行人，便是被呼延族的大长老派出去，专程到徐州盗书。
<br/>　　柳褐岩一行人跋山涉水，终于赶回了部落。
<br/>(本章完)
