　　在剑堂长老的带领下。
<br/>　　萧逸来到剑堂后的一处阁楼。
<br/>　　阁楼高而雄伟。
<br/>　　一层禁制，包裹着。
<br/>　　观其强度，起码要破玄九重以上才能强行破开。
<br/>　　阁楼前，有两人，乃是剑派里老一辈的两个执事。
<br/>　　皆是破玄三重武者。
<br/>　　两人常年在此，一边清修，一边镇守阁楼。
<br/>　　“长老。”两人见了剑堂长老。
<br/>　　没有行礼，只是微微拱了拱手。
<br/>　　剑堂长老点点头，道，“这是十一的弟子，来参悟极界碑。”
<br/>　　两人点点头，不语，但让出了过路。
<br/>　　剑堂长老推开阁楼之门。
<br/>　　萧逸对两个执事行了一礼，而后紧随着进入。
<br/>　　阁楼之内，空荡荡的，只有一座石碑，正是极界碑。
<br/>　　只是，观这极界碑，无比之高，若是放到外面，起码高耸入云。
<br/>　　这小小的阁楼，根本不可能放下它。
<br/>　　这时，剑堂长老拿出令牌，轻喝一声，“开。”
<br/>　　石碑前十米之处，陡然间出现一道空气扭曲。
<br/>　　一个‘缺口’出现。
<br/>　　“进去吧。”剑堂长老语气冷漠，“能参悟多少，全凭你自己本事。”
<br/>　　“你也可随时结束参悟，支会一声外头的守楼执事即可。”
<br/>　　说罢，剑堂长老转身离去，只留萧逸一人在阁楼中。
<br/>　　当初，顾长空来此时，他多番叮嘱，详细介绍。
<br/>　　轮到萧逸，却只说了寥寥两句便自顾离开。
<br/>　　萧逸也懒得理他，走向那‘缺口’，而后迈步走了进去。
<br/>　　恰在他完全迈入后，周围的环境瞬间变了。
<br/>　　他已不在阁楼，仿佛身处另外一个‘世界’。
<br/>　　“原来如此。”萧逸恍然，“是灵气屏障。”
<br/>　　“还运用了空间之道，难怪这阁楼能放得下如此巨大的极界碑。”
<br/>　　这个灵气屏障，显然比之前狂血玄君的洞府屏障要强大得多。
<br/>　　萧逸摸了摸，一股坚毅的感觉传来。
<br/>　　“好强大的屏障，怕是我全力一击，都撼动不了分毫。”萧逸吃了一惊。
<br/>　　这是裂天剑派先辈留下来的灵气屏障。
<br/>　　至于是谁布下的，就不得而知了。
<br/>　　只知，屏障受无数剑派前辈加持过，威力惊人。
<br/>　　哪怕是现在，剑派十大长老，也会定时前来注入真气。
<br/>　　久而久之，这个不知存在了多少年的灵气屏障，早已强到了非人的地步。
<br/>　　若无开启的办法，就是地元境武者，都休想强行破开。
<br/>　　萧逸向前看了一眼，眼前，一座巨大得看不到顶的石碑赫然出现。
<br/>　　那便是极界碑。
<br/>　　极界碑，不知存在多少年了。
<br/>　　单单之前易老所说，已经数百年没有弟子能完全参悟这块极界碑。
<br/>　　便知，这块极界碑，存在的时间最少有几百年。
<br/>　　甚至于，不止这个时间。
<br/>　　裂天剑派史籍上，从未记录过有关极界碑，以及这位地极境先辈的一丝一毫资料。
<br/>　　萧逸自然也对这极界碑毫不了解。
<br/>　　萧逸没有多想，只要自己彻底参悟了这块极界碑，便可知道它的来历。
<br/>　　再者，如易老所说，一年后，自己会明白的。
<br/>　　极界碑周围，无数道氤氲之气自动环绕，看起来，甚是美丽。
<br/>　　这些，是无数年来各个先辈注入真气下，让得屏障内头灵气量极为浓郁，而自动凝聚成的力量。
<br/>　　每参悟极界碑深入一分，这些氤氲之气就会注入参悟者体内一些。
<br/>　　顾长空，就是在此参悟了半成极界碑。
<br/>　　故而得到了数量极多得的力量，修为大增。
<br/>　　“极界碑，让我看看你有多难参悟。”萧逸低喝一声。
<br/>　　身影一闪，瞬间来到极界碑面前。
<br/>　　盘膝坐下，彻底开始领悟。
<br/>　　几乎是一瞬间，一股惊人的剑意，立刻缠绕了他。
<br/>　　这些剑意，便是极界碑上的参悟来源。
<br/>　　若承受不住，便连参悟的资格都没有。
<br/>　　“有些意思，才刚开始，就有如此惊人的剑意。”萧逸微微一笑。
<br/>　　这些剑意，轻微剐着他的身体，让他如蒙针刺。
<br/>　　同时，剧烈地侵蚀着他的心神，让他难以静下心来参悟。
<br/>　　顾长空参悟半年，方参悟了半成。
<br/>　　极界碑，太难参悟了。
<br/>　　时间，一分一秒的过去。
<br/>　　渐渐的，一个时辰过去。
<br/>　　萧逸挺了过来。
<br/>　　一日过去，萧逸仍旧挺了过来。
<br/>　　半个月过去，萧逸脸色无比苍白，汗流浃背，脸部抽搐。
<br/>　　显然就承受着难以抵抗的折磨和痛苦。
<br/>　　“半月时间，我连百分之一都参悟不出，甚至连门槛都达不到。”
<br/>　　萧逸咬了咬牙，却始终没有放弃参悟。
<br/>　　极界碑上的剑意和力量，绝非寻常武者能抵挡。
<br/>　　这数百年来的记录保持者，乃是一位破玄九重武者，领悟了5成。
<br/>　　但，也是凭着破玄九重的强大修为，才堪堪能承受住剑意，最终也不过领悟了5成。
<br/>　　再后面，到了6成、7成以上，怕是地元境武者，都承受不住。
<br/>　　地极境武者一生的武道知识，岂是轻与的。
<br/>　　萧逸再次咬了咬牙，仍旧挺着痛苦的折磨。
<br/>　　据昨日易老所说，顾长空在这里待了半年。
<br/>　　但并非每时每刻都在参悟，他每次都是承受不住便放弃了。
<br/>　　半年内，参悟了无数次。
<br/>　　最终才参悟出半成。
<br/>　　参悟极界碑，越是参悟到后面，剑意便越强越恐怖。
<br/>　　他参悟了半成以后，再无法承受剑意，才彻底离开。
<br/>　　言归正传。
<br/>　　萧逸边忍受剑意的折磨，边思索着。
<br/>　　“参悟，需一气呵成，方不会浪费时间，我必须挺住，不能放弃。”
<br/>　　“但这些剑意，若我强行去忍受，强行去参悟，根本不可能承受太久时间。”
<br/>　　“必须找到办法。”
<br/>　　萧逸还是头一次陷入这样的困难境地。
<br/>　　他的领悟力过人。
<br/>　　但，他现在连石碑上的文字都看不到一个。
<br/>　　必须参悟了剑意，才能透过剑意，直接得到极界碑上的内容和文字。
<br/>　　他很清楚，若他现在放弃抵抗剑意，大可远离极界碑。
<br/>　　而后在不承受痛苦折磨的情况下，再细细回味剑意，通过记忆，慢慢去参悟。
<br/>　　但这种方法，太过耗时了；而且不能一气呵成，对后面的参悟，影响极大。
<br/>　　最多不过个来月，就能参悟半成以内的剑意，而后看到石碑上的文字。
<br/>　　但其下场，必然是如顾长空一样，最终只能领悟半成。
<br/>　　以后再无增长。
<br/>　　“我就不信难得了我。”萧逸痛苦的脸上，泛起了无比的坚毅。
<br/>　　“灵鹤。”萧逸低喝一声。
<br/>　　原本盘膝的他，忽然摆出了一个奇怪的造型。
<br/>　　正是形意五绝之一的灵鹤。
<br/>　　这一绝，能调动体内真气，快速治愈伤势。
<br/>　　同时，也能稳固心神。
<br/>　　当然，他现在没有受伤。
<br/>　　他只是用灵鹤之形来减缓痛苦。
<br/>　　在这个状态下，他心清如明镜，平淡如止水。
<br/>　　脑海中，一头灵鹤，从东方而来；在一梧桐枝上略微停留。
<br/>　　灵鹤，仰望远方，那里，一片红云彩霞。
<br/>　　那里，是西方。
<br/>　　半晌，灵鹤踏彩而去，并非鹤驾西行，而是紫气东来。
<br/>　　脑海中的灵鹤，每一个动作，玄幻，奥妙。
<br/>　　他熟悉无比，因为，那是他的真意。
<br/>　　前世，自己悟透而出的五绝真意。
<br/>　　“嗯？”忽然，萧逸脸色一变。
<br/>　　伴随灵鹤真意的出现，渐渐的，那些剑意，竟然变了。
<br/>　　剑意，化成了再简单不过的知识，一眼便可悟透。
<br/>　　它不再有攻击性，不再折磨萧逸。
<br/>　　它，似乎认可了萧逸有资格参悟极界碑。
<br/>　　……
<br/>　　一个月后，萧逸蓦然睁开了眼睛。
<br/>　　“极界碑，已参悟三成。”
<br/>　　“原来，这位前辈，走的竟是霸剑一道，难怪如此难以参悟。”
<br/>　　极界碑，已参悟三成。
<br/>　　萧逸大概知晓这位前辈生前的剑道了。
<br/>　　他的剑道，霸道，却又纯粹。
<br/>　　一生所修，皆是霸道之剑，再无其它。
<br/>　　因此纯粹，也因此霸道无比。
