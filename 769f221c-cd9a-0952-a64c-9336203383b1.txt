
<br/>　　<br/>　　    这个寨子，看起来比以前小草和惊蛰走过的所有苗寨都要古老得多，从里到外都透着一股质朴和沧桑的气息，看起来就很有历史感。<br/>　　<br/>　　    这里的苗人很淳朴也很好客，想象中的戒备和警惕并没有出现在这些苗人的脸上，反倒是王惊蛰背着小草风尘仆仆进来的时候，几个上了年纪的大妈就迎了过来，操着一口两人都听不懂的苗语问候着。<br/>　　<br/>　　    王惊蛰皱眉听着他们七嘴八舌的苗族土话，完全不知道该从哪开口的好，无奈的僵持了半天之后，他忽然反应过来了，将背上的小草放了下来，然后伸出一根手指指了指她的眼睛。<br/>　　<br/>　　    茅小草瞳孔中两个眼仁中的那两道黑线异常清晰，几个苗族人看见后顿时就全都愣住了，足足过了半晌，其中一人才指了指寨子里一处山坡上的一栋老旧的草屋点了点头。<br/>　　<br/>　　    对方这一点头，王惊蛰提着的心瞬间就放下了，他轻声跟小草说道：“我有一种感觉，这里就是你获得新生的地方，你信么？”<br/>　　<br/>　　    疲惫的茅小草歪了歪脑袋，闭着眼睛声音有些发颤的说道：“我也感觉到了……”<br/>　　<br/>　　    有些事情就是这么奇妙，王惊蛰和茅小草奔波四十几天，差不多找遍了所有能去的苗寨，都没能找到一个可以为小草解蛊的寨子，但就是当两人在第四十八天，已经要彻底放弃的时候，曙光又再次出现了。<br/>　　<br/>　　    这就像是老天爷在和他们两个开了一个不大不小的玩笑，人就宛如坐了过山车一样，从云端到低谷来回折腾了好几次，最后才抵达了终点，看见了那个等待良久的出口。<br/>　　<br/>　　    王惊蛰和小草被热情的苗族同胞领着上了山坡，来到了那栋草屋的门前，一到外面就闻见了里面蔓延而出的一股刺鼻的药味，而且屋内还隐约散发着一股阴阴的气息，在这夜晚都有将近三十度的天气，让人冷的忍不住都要打个冷颤了。<br/>　　<br/>　　    房子里面的气息明显让人挺不舒服的，就跟往前迈一脚就进了阴曹地府差不多。<br/>　　<br/>　　    领路的苗人朝着屋内指了指，示意两人进去，王惊蛰牵着小草的手，皱了皱眉后还是迈过了门坎。<br/>　　<br/>　　    “当啷”王惊蛰刚进屋，一脚就不知道踢到了地上的什么东西。<br/>　　<br/>　　    一个黑漆漆的罐子被他给踢倒了，罐子滚了出去后，从口子里就爬出一堆黑色的甲虫，然后顿时就到处乱窜着，密密麻麻的看的人头皮一阵酸爽，从脚底到脑袋上瞬间就起了一层鸡皮疙瘩。<br/>　　<br/>　　    那些浑身上下都漆黑无比的甲虫从罐子里出来后到处乱爬，有几只速度极快的爬上了王惊蛰脚面上，两人连忙停住脚，倒吸了一口冷气后就想退到外面去，但这时屋内忽然传出一道清亮的啸声，那些乱窜的甲虫顿时就好像听见了号角的士兵一样，非常有序的又重新钻回到了罐子里，不到片刻工夫，外面一只虫子都没有了。<br/>　　<br/>　　    王惊蛰眯着眼睛往里看，屋子只有一间，也不是很大，东西杂乱无章的放着，多数都是那种用来养蛊的瓦罐，只有对面靠墙的地方，放着一张藤床，上面躺着倒人影。<br/>　　<br/>　　    只是藤床上躺着的人影比较怪异，看体型差不多跟个十三四岁的孩子似的，而且四肢明显还挺短小的。<br/>　　<br/>　　    当王惊蛰走过去的时候，看清床上的人影，瞬间就呆愣住了，不是对方体型太小了，而是此人的四肢都是断的，无手无脚。<br/>　　<br/>　　    这是个大概六七十岁的老人，正仰着脑袋看着王惊蛰，他身上穿着简单的苗族服饰，静静的躺在床上，手腕和脚腕上光秃秃的，没有手脚，看其断口的话全是齐腕而断的，十分平整，很明显应该是被刀砍出来的。<br/>　　<br/>　　    两人四目相对，半晌无声。<br/>　　<br/>　　    “你们能找到这个寨子，不容易啊，多少年都没有外人来过了……”对方先开的口，没想到还是一口虽然有点别扭但还挺标准的普通话。<br/>　　<br/>　　    对方说跟王惊蛰说了一句话后，眼神就落在了他后面小草的身上，当他看见小草眼仁中的两道黑烟，瞳孔顿时就放大了，四肢顿时挣扎了几下，似乎想要把身子给撑起来。<br/>　　<br/>　　    “你是在哪里中的虱蛊？”<br/>　　<br/>　　    就这一句，王惊蛰和茅小草全都同时心神巨震了，一个多月的时间里，他俩走访了二三十处苗寨，唯独这个寨子里，有人叫出了小草身上的蛊。<br/>　　<br/>　　    这在他俩的耳朵里，这就跟听见了上帝，如来，玉皇大帝和三清等满天神佛的动静一样，顿时就嗨了。<br/>　　<br/>　　    “在川中！”王惊蛰连忙说道：“我们当时碰上了一具尸体，蛊虫是从尸体里爬出来的……”<br/>　　<br/>　　    王惊蛰简要的把那天晚上的事说了一遍，床上的老人从他说第一个字开始，就一直在凝神静听，直到王惊蛰把前后都说了一遍，他才开口说道：“那是虱虫，专门寄生在人尸中的，尸蟞你们听过吧？跟这差不多，要说是区别的话，那就是虱虫还能操控人尸，就跟活死人差不多”<br/>　　<br/>　　    “呼……”王惊蛰长吐了口气，舔了舔因为紧张发干的嘴唇，小心翼翼的问道：“您既然知道这种蛊，那应该是能解的开了吧？”<br/>　　<br/>　　    那老人没有回他的话，而是看着小草，半天后说道：“如果我没看错的话，你身上的蛊应该中了能有四十几天，马上就要到四十九天了吧？”<br/>　　<br/>　　    小草嗯了一声，说道：“明天就到了”<br/>　　<br/>　　    “解蛊，有些麻烦，但是也能解”<br/>　　<br/>　　    王惊蛰和小草听见这句话，顿时彭拜了，他忍不住的哆嗦着嘴唇说道：“那麻烦您了，我们找了一个多月的苗寨，马上都要放弃了，没想到来到了这个寨子里，再晚两天就要来不及了”<br/>　　<br/>　　    “你知道我的手脚是怎么断的么？”对方又没直接回王惊蛰的话，而是抬起一只断手晃了晃。<br/>　　<br/>　　    王惊蛰皱眉说道：“应该是被人砍断的……”<br/>　　<br/>　　    “应该说，是被给你们下蛊的人砍断的”老人阴着脸，淡淡的说道：“这个虱蛊也是他从我这里学过去的”
