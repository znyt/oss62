<br/>        “而且，我看殿下浑然不知的模样，这无字碑的位置，必定是没有任何讲究的，必定风水极差！”

<br/>        月宫太子也不懂这些，不过转念一想，身为月宫的大祭司，确实有资格决定无字碑的位置。而且，则燃罪上师浑身没有半点法力，即便是让他接触无字碑，那又能怎么样呢？

<br/>        主角光环真的是牛逼。

<br/>        陈明回到自己在月宫的住所，体内，已经是又有三十六颗道种凝聚而出！

<br/>        很好，接下来，只需要进入其他的三大仙国，便可以凝聚完成自己的三千道种！
<br/>    
<br/>        不得不说，这无量道体，真的不是人修的东西啊！

<br/>    
<br/>        可真对不起这难度！

<br/>        不过刚刚过了一天，月宫太子再次找到了陈明，问道：“燃罪上师，这改变了无字碑的位置，真的可以定鼎天命吗？”

<br/>        月宫太子道：“父.......小子，确实有些不信。”

<br/>
