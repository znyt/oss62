怀里的香软一旦撤离，陆寒霆滚了滚喉结，然后抬手解开了衬衫的第一颗纽扣。
<br/>
<br/>        现在什么都不想做，只想跟昨日一样，将她抱回房间，扯落床幔，将她抱坐自己大腿上，凶狠的吻她。
<br/>
<br/>        这时就听到夏夕绾清丽的嗓音，“奶奶，我就不在家吃晚饭了，我有一个好朋友今天回来了，我现在要去机场接机。”
<br/>
<br/>        陆老夫人迅速道，“好哇，绾绾，我让司机送你去，如果你朋友愿意的话让她过来这里住两日，陪一陪你。”
<br/>
<br/>        “好的奶奶，你的心意我会传达到的。”
<br/>
<br/>        这时陆寒霆走过去，拿了车钥匙，“我送你去机场。”
<br/>
<br/>        “不用了…”夏夕绾想拒绝。
<br/>
<br/>        陆寒霆看了她一眼。
<br/>
<br/>        夏夕绾并不希望在奶奶面前表露出什么，或者是吵架，他性格强势，说送就一定要送的，她没有坚持的必要。
<br/>
<br/>        “哦，好，谢谢。”
<br/>
<br/>        两个人出去了，管家福伯嘀咕道，“老夫人，我看今天少爷和少夫人之间怪怪的。”
<br/>
<br/>        陆老夫人拿起筷子，瞄了福伯一眼，“你家少爷这是作的，你看吧，今晚绾绾不会回来了，等着你家少爷慢慢追妻去吧。”
<br/>
<br/>        福伯看着幸灾乐祸的陆老夫人：您是亲奶奶么？
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾上了劳斯莱斯幻影，坐在了副驾驶座上。
<br/>
<br/>        豪车平稳的驶进了繁华的车流，夏夕绾扭头看着蹭亮的玻璃窗外，晚上的海城真的很美。
<br/>
<br/>        这时陆寒霆出声道，“你是不是为了那十二个亿的事情跟我生气？”
<br/>
<br/>        “不会，钱是你赚的，我没有帮过忙出过力，无权干涉你的使用权。”
<br/>
<br/>        “如果我说我跟夏妍妍之间什么都没有发生，我跟她也不熟，只见过几次面，你会信么？”
<br/>
<br/>        夏夕绾扭头看着正在开车的陆寒霆，“陆先生，你是在变相的告诉我夏妍妍魅力大，还是我该笑话你人傻钱多，十二个亿都给了，连床都没有上到。”
<br/>
<br/>        陆寒霆深邃的狭眸薄冷了几分，低沉的嗓音带着训斥，“陆太太，你可以生气，跟我耍小性子，但是不要乱说话，我想上谁的床你不是不知道。”
<br/>
<br/>        夏夕绾将头扭了回去，不再看他。
<br/>
<br/>        寂静奢华的车厢陷入了沉默。
<br/>
<br/>        陆寒霆继续道，“夏妍妍曾经救过我，我答应她三件事，第一件是送她进F国圣黎院，第二件是往夏氏医疗注资十二个亿。”
<br/>
<br/>        夏夕绾抬手将腮边的一缕秀发掖到了耳后，然后轻嘲薄讽的勾了一下红唇，“这么说来，只要夏妍妍提的要求你都不会拒绝，救命之恩嘛，可以理解，不过如果不出所料的话，夏妍妍第三个要求就是要你娶她，你娶么？”
<br/>
<br/>        陆寒霆侧眸看着她清丽动人的侧脸线条，“我已经有陆太太了，怎么娶，还是说，你又想将我推给别的女人，不打算要我了，恩？”
