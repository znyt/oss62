<br/>				披上外衣后准备动手时，却见一道熟悉的声音传了进来。<br/>　　<br/>　　“陌尘公子，我是影一，我家主子请你马上过去。”影一从屋顶轻点脚尖跃落他的房门外，压低着声音唤着。<br/>　　<br/>　　听到这话，陌尘眉头微皱，他手指一弹点亮房中灯火，将衣服穿好，将墨发简单束起迈步走了出来打开房门，见门外之人果然是影一。<br/>　　<br/>　　“出什么事了？”他询问着，一边提气而起，跃上屋顶往凤府而去。<br/>　　<br/>　　影一紧跟在他的身后，道：“好像是鬼医出事了，具体如何，我并不清楚。”<br/>　　<br/>　　凤府中，主院里，厢房内，轩辕墨泽解开凤九身上的一处伤口查看了下，当看到那已经上了药的伤口非但没有愈合，反而微肿散发着黑色气息时，一颗心骤然往下沉下。<br/>　　<br/>　　他冷着一张俊脸，嘴唇紧抿着，看着身体发热已经不醒人事的凤九，他起身来到外间打了盆清水，用小手巾沾湿后拭去她额头渗出的汗，再泡了会冷水，放在她的额头上降着温。<br/>　　<br/>　　外面脚步声传来，紧接着，便是关习凛担忧焦虑的声音。<br/>　　<br/>　　“轩辕，小九怎么样了？要不要紧？吞云那边情况不是太好，小九现在怎么样了？”<br/>　　<br/>　　“进来吧！”轩辕墨泽的声音传出，外面的众人紧跟着关习凛走了进去。<br/>　　<br/>　　一下子，房中挤满了人。八名凤卫首领，以及冷华杜凡他们，还有关习凛和叶箐，满满的十几人围上前去。<br/>　　<br/>　　床上，凤九身上盖着被子，额头处敷着冷水巾，脸色因热气而泛红，嘴唇却是苍白的。他们这么多人进来，她也没有睁开眼睛，只是眉头微拧着，神色带着几分的痛苦。<br/>　　<br/>　　“我来看看。”<br/>　　<br/>　　懂医药的范霖上前为凤九把脉，又检查了她的伤口，半响，脸色凝重的对轩辕墨泽道：“跟吞云一样，没有中毒的迹象，但都是伤口微肿，有黑色的气息弥漫在伤口上，我见识浅薄，不知这黑色气息到底是什么来的，因此也无法对症下药。”<br/>　　<br/>　　“看来定是这伤口上面的黑气在作怪。”关习凛说着，道：“以往小九就算是受伤了，也能很快恢复过来，像这样的情况还是第一次碰见。”<br/>　　<br/>　　“那怎么办？她的伤口在恶作，若是不治疗岂不是……”叶箐有些担心，若是凤九清醒着还好，可如今她明显已经没了意识，若是再耗下去会不会更严重？<br/>　　<br/>　　“陌尘跟在天机老人身边，又精通医术，我已经让影一去请他过来了。”轩辕墨泽说着，目光落在床上的凤九脸上，见她眉头紧皱睡得不安宁，他便取下那冷水巾再浸了下凉水，再敷在她的额头上。<br/>　　<br/>　　冷霜见状上前将水端走再换了一盆水过来，看着主子这样，她也不知如何帮忙，只能静静的守在一旁。<br/>　　<br/>　　“主子，陌尘公子来了。”<br/>　　<br/>　　影一的声音自门口传来，房内众人朝门口处看去，就见陌尘迈着脚步走了进来。<br/>　　<br/>　　陌尘见房中众人都在，便看了他们一眼后，询问着：“怎么回事？”<br/>
