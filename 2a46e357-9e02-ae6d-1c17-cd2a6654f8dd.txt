　　“你……”
<br/>　　几千人，此时此刻，无人开口说话，因为他们没有资格没有权利再去说话了。
<br/>　　林峰对陆勇点了点头，陆勇也没任何墨迹，特等区虽然因为几个A级怪物被林峰斩首了，暂时安全了，但是他们还要商榷反攻计划，肃清平民区然后把破洞补上去。
<br/>　　回到作战临时指挥室。
<br/>　　林峰问道：“这个外围之强到底被什么怪物给击破的，那么厚的钢门，金库的钢门也不过如此了吧。九阶怪物都根本炸不开吧，难道出现十阶怪物了？？”
<br/>　　“我也不知道，不只是一头A级怪物，是很多头A级怪物，一个个忽然发疯了一样，甚至不惜牺牲自己也要撞开大门，难以想象。”
<br/>　　陆勇摇了摇头，眼中还有后怕和心惊。
<br/>　　“发疯了？”
<br/>　　林峰一脸诧异。
<br/>　　“难道他们吃了坏东西闹肚子了？”
<br/>　　陆勇摇了摇头，他也不知道到底发生什么了。
<br/>　　“首长，林峰，现在不是去管那些怪物是不是吃坏肚子了，而是我们当务之急就是要夺回曙光基地。”
<br/>　　周华在旁边催促道。
<br/>　　“也是。”
<br/>　　陆勇道。
<br/>　　林峰点了点头。
<br/>　　众人整顿了一下，然后准备夺回基地的作战计划。
<br/>　　“林峰小兄弟，这一次还是要靠你了。”
<br/>　　陆勇认真道。
<br/>　　“陆将军，都这个时候了，这些话就不需要多说了，唇亡齿寒的道理我初中就知道了。”
<br/>　　林峰道。
<br/>　　“嗯。”
<br/>　　陆将军点了点头，也不墨迹，留下三百军人踞守特等区，然后他们一群人大概有两千人不到，朝被丧尸和怪物霸占的平民区冲去。
<br/>　　这是一场夺回基地的战斗。
<br/>　　放下吊桥，林峰和周华一马当先，他们两个人都是九阶。
<br/>　　其余的士兵全副武装的跟在后面，仅有的几辆坦克也都没有缺席。全部加入了作战计划。
<br/>　　此时平民区内，还有许多人在据守，他们都是些默默无名之辈，但哪怕是在这种情况下，本应该履行自己职责的宪兵都丢下枪杆子跑了，他们这样人却站了出来。
<br/>　　这群人是形形色色的人，他们来自不同的地方，甚至说着不同的方言。
<br/>　　但此时此刻，就好像有一种无形的力量，把他们汇聚在了一起，孤木不成林，单丝不成线，此时此刻，他们汇聚在了一起。
<br/>　　只是因为，在他们看来，无论睡大街也好，还是睡板屋也罢，这里就是他们赖以生存的最后地方，跑，又能跑的到那里去，跑到特等区区就不用死了吗？
<br/>　　仗义每多屠狗辈，负心总是读书人！
<br/>　　“杀！！”
<br/>　　一个三十多岁的汉子拿着手中的西瓜刀怒斩而去，他生之前就是个西瓜商贩。
<br/>　　噗嗤。
<br/>　　一头三阶丧尸的脑袋，直接像是劈砍西瓜一样，在他的西瓜刀之下碎裂。
<br/>　　“死！”
<br/>　　在他身边不远处，一个穿着饿了么外卖衣服的年轻人拿着手中的头盔就是猛砸下去。
<br/>　　“给我滚！这是老子的地盘！！”
<br/>　　一个穿着海尔售后员工制服的男人怒吼着拿着钢管就是砸了下去，顿时一只正在啃人类尸体的四阶变异怪物全部被打爆了。
<br/>　　如此一幕，在这偌大的平民区，屡见不鲜，他们有最基础的员工，有大城市内最忙碌的身影，也有为了客户一个要求不断改稿的设计师，也有包揽工程的小老板等等。
<br/>　　许多人都依然还没有放弃抵抗，当人类没有放弃抵抗时，那么人类就没有输！
<br/>　　然而。
<br/>　　他们虽然勇气可嘉，但他们势单力薄，在一大群变异怪物和丧尸面前，他们显得岌岌可危。
<br/>　　一个中年大叔拿着砍刀刚砍翻了四头丧尸，忽然一个变异狗从背后的屋檐下一跃而下，把他扑倒在地，中年大叔实力足足有五阶，他立即一刀刺穿了这个想要咬断他咽喉的变异狗头颅，然而还没有站起来，几十头丧尸一下子把他扑倒在地，中年大叔砍翻了十几个丧尸，最后，只剩下了惨叫。
<br/>　　“该死！太多了！！”
<br/>　　七八个大学生背靠背的不断后退。
<br/>　　他们都是好基友，其中好几个人都是一个宿舍的。
<br/>　　“小三，杀了多少个了。”
<br/>　　一个男生问道。
<br/>　　“不多不少，刚刚五十个，其中七个四阶的。”
<br/>　　穿着黑色短袖的男生道。
<br/>　　“你呢？”
<br/>　　“我，我比你多一些，也就百来个了，妈的，这群畜生，还真是源源不断啊。”
<br/>　　几个男生喘着气，他们实力平均都有四阶了，最强的足足六阶，算是个素质很高的小团队。
<br/>　　不过没用，他们七个人，在七百个在七千个丧尸怪物面前，那就是沧海一粟罢了。随时都要被吞没的无影无踪。
<br/>　　刷刷刷。
<br/>　　也就在这个时候。
<br/>　　一个变异丧尸出现在一个房屋顶上，他就这样俯瞰着这群人，似乎在嘲笑他们的不自量力一样。
<br/>　　这个变异丧尸的两把手居然是两把弯刀。
<br/>　　此时此刻，这两把手刀上面都占满了鲜血。
<br/>　　“不好，是七阶的变异丧尸！”
<br/>　　几个大学生都是面色一变，足足七阶还是变异的，他们必死无疑。
<br/>　　“难道我们就要死了吗？”
<br/>　　每个人内心之中都充满了绝望，他们虽然也逢凶化夷了好几次，但是任何一次都没有眼下的这么惊险。
<br/>　　刷。
<br/>　　也就在这一刻。
<br/>　　他们每个人都看到了这一辈子最为炫酷的一刻。
<br/>　　在他们视线之中，那个仿佛在冷笑他们的七阶变异丧尸，不知道被什么玩意一下子斩为了两半，上半身和下半身直接分开，鲜血狂飚。
<br/>　　“怎么回事？”
<br/>　　“这，这丧尸怎么忽然死了。”
<br/>　　每个大学生都愣住了。
<br/>　　然后下一刻他们就明白发生什么了。
<br/>　　一个比他们还要年轻的少年，拿着一把沾血的宝刀，出现在了屋檐之上。
<br/>　　“你们可以去军部综合办事处，那个地方，已经被清理出来，作为夺回平民区的据点了，现在正缺大量人手，需要你们这样的黑铁战士。”
<br/>　　林峰说完就是一跃而出，消失不见。
<br/>　　“太帅气了吧。”
<br/>　　‘我去，来无影去无踪说的就是这种吧。“
<br/>　　“实力好强，居然一刀秒杀了那个七阶的变异丧尸！”
<br/>　　“别废话了，我们还是先去综合办事处吧！看来，军部的人还活着！”
<br/>　　“没错！”
<br/>　　一群人立即收拾好自己激动的心情朝综合办事处奔去。
