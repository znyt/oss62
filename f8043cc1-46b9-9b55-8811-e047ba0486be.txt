　　“又是我开？”高少辉愣在原地，紧接着又猛然惊醒：“我靠，那这和我一个人开着到处乱晃最后被白忧发现有什么区别？和你一块儿我还会落下个勾结的嫌疑！”
<br/>　　楚涵似笑非笑的看着他：“咱连勾结貌似不是嫌疑吧！还是说你怕了？怕白忧？”
<br/>　　“怎么可能！”高少辉顿时大声反驳：“我怕个屁！白忧算个啥！”
<br/>　　楚涵收回目光，点着驾驶座：“那就进来，起飞。”
<br/>　　高少辉咽了咽口水，只能一脸颓丧的照办。
<br/>　　只是当两人进入机舱内后，高少辉又猛然反应过来：“不过我们咋出去？从这里飞到阴谷出口，在进入阳谷，然后路过……额，好长的路，天照3000不一定钻的过那么多地方啊！”
<br/>　　楚涵指了指沙滩上的鱼柔女：“她送我们。”
<br/>　　“啊？”高少辉愣住。
<br/>　　还不等他有所反应，忽的一阵海浪就从后方涌上沙滩，以一种不可抵挡的强势，强行将天照3000整个卷入水中。
<br/>　　“啊！”高少辉吓了一大跳，就想要启动天照3000控制：“要淹死了！”
<br/>　　“淡定，出去再控制飞行。”楚涵却一把将高少辉的手拉住，阻止了他的动作。
<br/>　　高少辉惊慌失措，想质问楚涵什么，但这时候他的声音却忽然卡在喉咙出不来，因为这会儿他已经发行天照3000整个都被水浪托着，以一种惊人的速度朝着远处海洋飞掠而去！
<br/>　　“什么情况？！”高少辉大惊失色，他忽然想到刚才楚涵指着那个女人说‘她送我们’，顿时心脏狂跳。
<br/>　　女人？
<br/>　　高少辉连忙将内部的屏幕调整，很快屏幕上就看到了后方沙滩的场景，只是因为海浪的速度太快，此时屏幕上已经完全见不到沙滩的踪迹，放眼望去一片蔚蓝的海洋。
<br/>　　哪还有什么女人？
<br/>　　“她，是谁？”高少辉扭头看向楚涵，声音结巴，冷汗狂冒。
<br/>　　楚涵面无表情的回复：“哦，鱼柔女，水族的圣女。”
<br/>　　轰！
<br/>　　高少辉的内心掀起惊涛骇浪，震惊到一个字都吐不出，这是他活了二十年来的整个生命中，最震撼他的一句话。
<br/>　　水族，圣女！
<br/>　　水族回来了？
<br/>　　他们在阴阳谷！
<br/>　　水族的圣女觉醒了，她就在阴阳谷入口！
<br/>　　“啊！！”高少辉忽然一声大叫，以表达他此时此刻难以宣泄的情绪。
<br/>　　楚涵捂住耳朵，皱眉道：“叫什么，神经病啊？”
<br/>　　“水族啊！圣女啊！握草啊！”高少辉继续大叫着，整个人都有些癫狂。
<br/>　　高少辉不是笨蛋，虽然当下的事件足以让他震惊到不能自已，甚至比见到制裁者更情绪起伏不定，但制裁者一直存在，他只是害怕而已，但只要不犯事制裁者并不会对他做什么。
<br/>　　但是水族可就不一样了，圣女更完全属于另一个意义的存在。
<br/>　　那是回炉四大正门之一，凌驾于四偏门的神秘家族之上，哪怕他是嫡系见了水族也完全抬不起头，更别说堂堂圣女。
<br/>　　那可是要行礼的啊！
<br/>　　可是现在，水族的那位高贵的圣女，竟然送他们出谷！
<br/>　　此时的海浪还在继续，天照3000飞速的在这片海域穿梭，有着水族圣女的控制，这片水域没有任何阻拦的就让天照3000冲出了阴谷入口的古怪磁场，而后平稳的停在了这片海域之外的另一方海洋之上。
<br/>　　感受着完全不一样的磁场，高少辉连忙调转镜头往后看，果然阴阳谷的踪迹已经完全消失，被隐藏在了身后的那片神秘海域。
<br/>　　“真的出来了？”高少辉还是没回过神，愣愣的发着呆。
<br/>　　楚涵敲了敲前方屏幕：“出发，去南署基地。”
<br/>　　高少辉收敛了表情，快速的启动飞行，他的目光看着前方，但却在对楚涵严肃发问：“告诉我这到底是怎么回事？”
<br/>　　水族早已消失千年，所有四偏门的神秘家族都以为他们已经灭族，不可能再会出现，同样在他们看来火族也是。
<br/>　　这同时也关系到前面提到的阴谷禁制，罗家之所以能够对肖家下手的另一原因，那就是四偏门家族已经默认水火这两家族，已经灭族……
<br/>　　既然灭族阴谷的禁制当然不可能解决，四个神秘家族内部的自相残杀，又算的了什么？
<br/>　　但让高少辉完全没想到的是，事实情况与他们任何一个家族的推论，都完全不同。
<br/>　　灭族个鬼啊！
<br/>　　人家水族就在阴阳谷呢，还觉醒了圣女！
<br/>　　楚涵当然能猜到高少辉的想法，嘴角勾了勾道：“巧合之下，和水族交了朋友，现在我能自由进入阴谷，也能进入他们的水族旧址，水族人很热情，族长跟我哥们，圣女人很好。”
<br/>　　高少辉这下子完全无法绷住情绪，带着震惊和难以抑制的羡慕盯着楚涵：“你到底开了什么挂？竟然能跟水族这么亲密！我靠！族长都跟你是哥们，圣女还对你这么好！送我们出去，她竟然肯送我们才这个门出去，这简直违背常理！”
<br/>　　楚涵面无表情的继续道：“我还见过火族。”
<br/>　　“啊靠！”高少辉继续大叫，他发现今天这一天过的很刺激，一连串的惊天秘密都呈现在了他眼前。
<br/>　　“但我跟火族的关系就不怎么样了。”楚涵继续说着，略带遗憾：“我不仅破坏了他们在阴阳谷之外的居住地，还抢了他们从水族那里拿来的东西。”
<br/>　　这回高少辉已经彻底无语，一句话都说不出，看向楚涵的眼神就像在看一个怪物。
<br/>　　他心想这神秘家族找了一千年都没找到的两大家族，关系到他们四偏门最大秘密的水火两族，竟然全部被楚涵遇到了，还发生过这么多骇人听闻的事件。
<br/>　　跟水族族长拜把子，水族圣女帮他忙，毁了火族居住地又抢了他们东西。
<br/>　　甚至还活着？
<br/>　　不可思议！
<br/>　　当高少辉和楚涵已经离开阴谷，消失在这片海域的时候，此时还站在阴谷入口内那片沙滩上的鱼柔女，却深深皱起了眉。
<br/>　　她回忆着之前看到的那一幕，万分确定楚涵揪住了高少辉的衣领，并且还在机舱内轻而易举的拍了下高少辉的肩膀……
<br/>　　一个普通人，能够触碰到神秘家族正统的嫡系血脉者？
