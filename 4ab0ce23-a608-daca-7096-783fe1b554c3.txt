　　可以说现在涂小安这条蛇，是名正言顺的蛇王，周身散发皇者之气，只要微微往哪里一杵。
<br/>　　任何蛇种都要顶礼膜拜，除非你比他这条蛇更强更可怕。
<br/>　　正常来说，生物界，每个种族的动物已亿万为单位，但是能成为三阶生灵，登基坐殿的，就只有一位。
<br/>　　三阶凌驾普通生物之上，就算是人类，也是如此。
<br/>　　而且具备了化形的能力。
<br/>　　已然不是普通的生物，超凡脱俗的存在。
<br/>　　涂小安能深刻的感受到自身的变化，这第二次阶段性的进化，融合变异吸血蛇的血脉血统，让他还没有成为三阶，就已然登基坐殿了。
<br/>　　就冲眼前大规模的蛇类顶礼膜拜，就说明了一切。
<br/>　　与此同时，涂小安的统治能力也成了鸡肋，因为他现在隐隐就散发类似统治能力的王者之气。
<br/>　　激动了很久，涂小安才冷静下来，三十米巍峨的蛇躯开始迅速的原地缩小，变回了他习惯自在的一米长度。
<br/>　　倏然，他成了紫金色的小蛇，只是这条蛇高贵而漂亮，用涂小安自己的话来说，就是他貌似又变帅了。
<br/>　　变帅的同时，浑身上下，还多了一股气质。
<br/>　　没有任何人，能看到他这条蛇而不痴迷。
<br/>　　没错，是痴迷，一般人类都是有惧蛇症，都会怕蛇，尤其是女人。
<br/>　　但涂小安这条蛇一出，别说怕了，瞬间都能让你变成花痴。
<br/>　　哎...帅的不要不要的了。
<br/>　　自恋之后，涂小安看到了自己蜕下来的三十米蛇皮。
<br/>　　此蛇皮已然是天材地宝的级别，罕见蛇种吃了他的蛇皮，血脉瞬间提升，可以进化成珍稀蛇种，而珍惜蛇种的毒蛇，吃了蛇皮，可以成为觉醒类的蛇种。
<br/>　　就是那么的吊炸天。
<br/>　　涂小安第一次脱的蛇皮就蕴含纯粹的能量，能别说这次是第二阶段性进化而脱下来的蛇皮。
<br/>　　这拿出去当药用，恐怕绝症都不要话下了。
<br/>　　看了几眼自己的三十米蛇皮，涂小安眼睛浑然一眯，三十米蛇皮骤然消失不见，被收入到系统之中。
<br/>　　做完这一切，涂小安眸光朝着某个地方小山丘看去。
<br/>　　小山丘异香弥漫，遍地蛇莓红彤彤的闪烁光泽，这在黑夜里，就是一颗颗小星星，闪烁着淡淡的光晕。
<br/>　　他之前跟吸血蛇打的天翻地覆，可这小山丘还屹立在哪里，归根原因，就是无论是他还是吸血蛇，都是刻意的在保护这座小山丘，不让它受到伤害。
<br/>　　不然，就这样一座小山丘，一条蛇尾，就能轰塌。
<br/>　　“我到要试一试这蛇莓到底有何神奇的功效”
<br/>　　一条黑斑蟒，吃了蛇莓可以长成十五米，这东西有点逆天。
<br/>　　另外，它能将吸血蛇吸引过来，能长在蛇谷最深处，能让万千毒蛇趋之若鹜，很显然，说它是蛇类的圣果，都不过分。
<br/>　　现在涂小安登基坐殿，这里的一切都属于他，包括这小山丘上的蛇莓，包括这里的每一条毒蛇。
<br/>　　蛇眸朝着小山丘看了一眼，然后张开了蛇口，霎时间，只见一颗红彤彤的蛇莓神奇的悬空飞了过来，直接投怀送抱的飞到了涂小安的口中。
<br/>　　一口咬下去，涂小安蛇眸明亮起来，直接入口即化，随后一股非常强劲的能量就钻入他的身体之中，很霸道，很勇猛，这是一股炙热滚烫的能量，在身体里面疯狂窜动，无声无息间。
<br/>　　那滚烫的能量像是要撑爆他的身体。
<br/>　　涂小安感觉自己的体型要暴涨了，唯有暴涨，才能消耗这股能量。
<br/>　　只是涂小安的身体并没有得到暴涨，甚至一点都没有涨。
<br/>　　原因很简单，他的身体刚暴涨过，进化过，达到三十米，区区一颗蛇莓，还不足以让他的体型发生变化。
<br/>　　不过如果是别的蛇，肯定体型就暴涨了。
<br/>　　归根结底，这种蛇莓应该是促发身体的二次发育，让你的身体跟力量都达到提升。
<br/>　　很好的天材地宝，可惜现在对涂小安自身，并没有太大的作用。
<br/>　　当然，他要是连续吃个十几颗，还是有帮助的。
<br/>　　忽然，涂小安觉得蛇莓跟在系统商城中买的强化果有点相似，但好像更加的霸道点。
<br/>　　宝贝是好宝贝，那么他就不客气了，这长满的蛇莓，先收了在说，总不会有坏处。
<br/>　　这时，涂小安强大的意识力顷刻间将整座小山丘笼罩其中，意念转动，骤然间，所有生长的蛇莓都不由自主的悬空起来。
<br/>　　“给我收”
<br/>　　蛇眸浑然一眯。
<br/>　　一个呼吸，小山丘变成了光秃秃，而涂小安的系统之中出现了整整一千三百颗蛇莓。
<br/>　　他并不知道蛇莓生长周期要多久，但他相信这座小山丘迟早还会长出新的蛇莓。
<br/>　　将这座小山丘的蛇莓收了之后，涂小安心中冒出一个恶趣味，这蛇莓应该不是单单只有蛇才可以吃。
<br/>　　别的生物应该也能吃，但要是被一个人吃了，也能暴涨身躯？
<br/>　　那岂不是要变成巨人了。
<br/>　　如此的话，对人可不算什么好处，试想，如果涂小安将蛇莓给老姐吃了，老姐直接长到三米高。
<br/>　　那么她将来还嫁的出去吗。
<br/>　　稍稍想想，好像还挺可怕的。
<br/>　　所有的事情都告一段落，涂小安也应该离开了，事情也得到了完美的结局。
<br/>　　蛇沟之王被吸血蛇给吃了，涂小安虽然登基坐殿，但他也不可能生活在蛇沟之中。
<br/>　　另外蛇莓这种宝贝，总要有蛇守护。
<br/>　　虽然小山丘现在是光秃秃，才迟早就长出新的。
<br/>　　“双头蛇”
<br/>　　噌...一身双头的双头蛇摇曳着两颗凶煞的蛇头出现。
<br/>　　“你就留在这里发育吧”
<br/>　　涂小安觉得将双头蛇留在这里挺合适，一是能帮他守护蛇莓，不让长出来的蛇莓被其他毒蛇吃了。
<br/>　　二是双头蛇需要发育，需要成长，它要有自己的空间。
<br/>　　留在自己的身边没什么用。
<br/>　　这蛇沟之大，难以衡量，里面有很多厉害的觉醒蛇，就让双头蛇在这里自己发育，自己成长。
<br/>　　到时候它也成为二阶蛇精，那就厉害了。
<br/>　　况且涂小安要想召唤双头蛇出现，一个意念就可以了。
<br/>　　好，就这样办，打道回府。
