　　这五名半步天人境武者，都是来自于圣域九华天，有着一种高高在上的心里，对现在这已经没落的九域，不屑一顾。
<br/>　　至于剑道门，在他们看来，更是丝毫不放在眼中。
<br/>　　现在看到剑源界出现裂痕，根本没有任何的探察，便是直接化为遁光，进入到了剑源界之中。
<br/>　　看到这一幕的楚风眠，心中都是松了一口气。
<br/>　　如果这五人谨慎的话，对现在的楚风眠来说倒是无比棘手。
<br/>　　这五个人来历惊天，如果不是将他们引入到剑源界之中，楚风眠还真没有将他们一网打尽的自信。
<br/>　　不过现在，这五个人已经是进入到了剑源界之中，楚风眠心中便是无边的自信了，哪怕是真正的天人境，来到这剑源界之中，楚风眠都不会让他们好过。
<br/>　　“剑道门，不过如此，这点实力，在我们九华天之中，就连三流势力都不如。”
<br/>　　那五个人进入到了剑道门之中，便是扫视着剑道门之中的弟子，眼神之中露出几分不屑之色道。
<br/>　　“这样的势力，如果不是在九域之中，就连臣服于我们的机会都没有。”
<br/>　　“算了，这剑道门毕竟在九域之中也算是一个有些名气的势力了，我们先将这剑道门降伏再说。”
<br/>　　一阵阵的议论之下，那名为火天的半步天人境武者，一下子落了下去，一步便是踏入到了云羽天宫之中去。
<br/>　　目光扫视向了其中的寒月漓，洛汐，燕皇等人，眼神高高在上，如同是在看向一群蝼蚁一般。
<br/>　　不过当看到寒月漓跟洛汐的一刻，火天的目光之中，却是露出了几分异样的神情，一抹玩味笑容出现在他的脸上。
<br/>　　“先天神体？混沌气息？没想到这小小的九域之中，还有这样的天才，你们两个，有资格做我的小妾。”
<br/>　　火天开口的一刻，目光也是落在了寒月漓跟洛汐的身上，仔细的打量着。
<br/>　　仿佛现在寒月漓跟洛汐，已经是成为了他的囊中之物一般。
<br/>　　“我，火天来自圣域九华天，你们两个，现在速速跪下臣服，做我的小妾，将这剑道门奉上，日后不会少了你们的好处。”
<br/>　　这火天一开口，语气便是高高在上，让寒月漓，洛汐，燕皇等人都是脸色一寒。
<br/>　　不过他们还没有来得及开口，便是只见一道身影从他们的背后走出来。
<br/>　　“好大的口气，九华天？我剑道门跟九华天，向来没有恩怨，井水不犯河水，你们为何来攻打剑道门？”
<br/>　　这走出的身影，便是楚风眠。
<br/>　　楚风眠的目光，也在打量着眼前这火天，想要看看这来自于圣域的天才，到底是有何不同。
<br/>　　楚风眠感觉到，这火天应该只是刚刚踏入到了半步天人境的武者，但是此人的一身灵力，却是雄厚到了一种不像话的地步。
<br/>　　几乎是要比起五行刀圣那样的老牌半步天人境强者的力量还要强横几分。
<br/>　　尤其是在加上他们手中的半仙器，这火天，几乎是面对寻常半步天人境武者，可以以一敌二。
<br/>　　这种人物，纵然是放在半步天人境的武者之中，也足以位列中上了。
<br/>　　而且楚风眠也感觉到，这火天等人，定然也是有着没有暴露出的底牌来。
<br/>　　不过如今这五个人，已经是落入到了剑源界之中，楚风眠有着自信，哪怕是他们有着再强的底牌，今日楚风眠也可以将他们，诛杀与剑源界之中。
<br/>　　随着楚风眠的现身，楚风眠也是心神一动，催动了剑源界的力量。
<br/>　　顿时整个剑源界之中出现的无数裂痕，都是被瞬间修复，刚刚出现的那些裂痕，不过是楚风眠故意弄出来的，为了将这五个人引到剑源界之中罢了。
<br/>　　现在目的一到，楚风眠当然不用在伪装下去。
<br/>　　“哦？”
<br/>　　这剑源界的变化，那火天也是注意到了，他的目光闪烁，刹那间便是了解了这一切的种种，不过他的脸上依然是无比的狂傲，没有丝毫的担心。
<br/>　　就算是眼前楚风眠故意将他们引入到了这剑源界之中来，这火天也丝毫不惧。
<br/>　　火天的目光，打量了楚风眠一眼，眼神之中露出了几分不屑之色道。
<br/>　　“还当是一个什么人物，还敢算计我们，居然只是一个小小的九劫古帝，看来你就是这剑道门的门主，楚风眠了？”
<br/>　　火天说道。
<br/>　　“实力不错，看来有些机缘，不过在我的眼中，依然只是蝼蚁罢了，据说你得到了剑道之主的传承？小子，现在跪下，将剑道之主的传承交出来，臣服为奴，我可以饶你一条命，让你为我们做事。”
<br/>　　“是吗？死到临头，还敢如此嚣张？看来九华天的人，不过只是一群没有脑子的蠢材罢了。”
<br/>　　楚风眠听到这火天狂傲无比的话，也是冷笑一声。
<br/>　　什么圣域九华天，这名号下的了其他人，可吓不到楚风眠。
<br/>　　现在楚风眠融入了太初之气，已经是化为了天地异种，先天神兽都只能跟楚风眠平起平坐，楚风眠还会怕这什么所谓的圣地九华天？
<br/>　　更何况现在这些九华天的人，已经是攻打剑源界，攻打到楚风眠的老巢来了。
<br/>　　到了这个地步当然不会再给这些九华天的人，任何好的脸面看。

<br/>　　楚风眠看向那火天，如同是看向一个死人一样的开口道。
<br/>　　“蠢材？小杂种，给我死来！”
<br/>　　轰！
<br/>　　听到楚风眠的话，火天彻底暴怒了，他来自圣域，地位显赫，在他眼中，现在剑道门的一切，不过都是一群蝼蚁。
<br/>　　让剑道门现在臣服，对他来说都已经是天大的恩赐了，现在楚风眠居然还敢反抗，让他彻底暴怒。
<br/>　　火天怒喝一声，在这开口出声的瞬间，从他的手心之中，一杆长枪凝聚而出，正是火天的灵器，半仙器。
<br/>　　他的身形在这一刻，便是跟手中的长枪，融为了一体，突然间向着楚风眠猛然穿刺而来。
<br/>　　这一枪之威，锋芒强横到了极点，简直是要撕裂苍穹。
