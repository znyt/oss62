　　砰！
<br/>　　一圈烟尘在黑暗中扩散开来，藤蔓一样的根须纤维被吹得倒伏。
<br/>　　“哦，到底了！”比杨德不带一丝烟火气的落地，入目一片黑暗，只在角落中有着些微荧光，估计是什么发光苔藓活着昆虫之类。
<br/>　　“视线有些暗。”说着，比杨德深邃黑色的眼睛逐渐变化。从漆黑变得放射出的暗淡红光，最后竟然变得和红宝石一般。
<br/>　　如果有窟卢塔族的人在这里，一定会发现，这里似乎多了个同族！
<br/>　　“不管使用多少次，依旧会为火红眼感到神奇。”瞪着一双火红眼，比杨德满脸的感慨。那些有着收藏怪癖的人，看到的是火红眼的凄美颜色。而比杨德看中的是火红眼的巨大潜力。
<br/>　　它能够承载很多种变化，比普通人的眼睛强大百倍！比如现在，即便周围漆黑一片，比杨德眼前却亮如白昼。
<br/>　　他是窟卢塔族人吗？不是！这是比杨德的独特能力，为了探险而开发出来的强大念能力——适者生存！
<br/>　　有着恐怖的模仿复制能力。通过主观研究，模仿其他人或物的能力，然后在适当的时候施展出来，并根据当前恶劣环境迅速进化，最终——适者生存！
<br/>　　他就是用这种能力，模拟改造了自己的眼睛，让他拥有了火红眼的全部潜力！
<br/>　　正是因为这种强大能力，他才能多次进入危险境地，却全身而退。
<br/>　　这也是他敢于前往黑暗大陆的真正底牌！
<br/>　　“表面光滑如琉璃，质地坚硬堪比钢铁。只有高温和强压同时作用，才能形成这种结果。真是强大的能力。”比杨德站在苏重开辟出的通道入口，右手摸着圆形通道的墙壁。仔细感受着墙壁的强度，以此推断苏重的能力。
<br/>　　“顺着世界树根须进入地下，果真是在找熔岩。结合通道墙壁的结果，可以初步推断。这种能力需要吸收外界热量！”比杨德双眼放着红光，在黑暗中显得妖异无比。
<br/>　　沉吟片刻，比杨德身影一闪，消失在坑洞深处。
<br/>　　顺着苏重留下的通道，比杨德根本不需要分心寻找方向。直接向着地下快速前进。
<br/>　　“温度越来越高了？”感受着四周温度，比杨德发现，还没到目的地，头发胡须竟然已经开始变得卷曲。
<br/>　　周围温度急剧上升，身边不远处的世界树根须，更是灼热非常。就像一根烧热的钢铁柱子，除了不发红，该有的温度一点儿不差。
<br/>　　“看来需要做些改变！”比杨德喃喃自语。
<br/>　　伸手按在身旁世界树根须上，嗤的一股带着焦臭的白眼冒起。手掌已被灼伤。
<br/>　　比杨德一脸平静，丝毫不理会自身伤势。脚下不停，不断前进。一边走，不时的还要再世界树根须上拍一下。
<br/>　　手掌不断冒出白烟。
<br/>　　随着下潜深度变深，温度越来越高。比杨德手掌上的伤势却反而越来越轻。到了最后，不仅不怕被更高的温度灼烧，就连最初的伤疤也消失不见。
<br/>　　而且，手上皮肤也发生改变。似乎有了些类似植物般的纹理。不仅是手，随着他潜行。树皮状纹路沿着手臂不断向上蔓延，最后蔓延到脖颈，旋即笼罩整个脑袋。最后他全身皮肤都带上了这种纹路。
<br/>　　模仿世界树组织，快速改变自身皮肤组织。在这一刻，他不仅不再惧怕高温。反而还能从把高温吸入体内，化作自身能源！就像世界树一样！
<br/>　　这就是他的能力——适者生存的能力！
<br/>　　一手扶着漆黑岩壁，比杨德站在蛋形地下空间的入口处，旁边就是一根粗壮的世界树根须。
<br/>　　“怪不得世界树停止生长，地下岩浆湖已经快要被抽干。没了能量，根本无法生长。”比杨德看着脚下，变成了漆黑色的地面。
<br/>　　抬头看天，蛋形地下空间顶部，无数根须破壳而入，垂落在下方漆黑地面中。
<br/>　　他能够感受到漆黑地面的高温。可惜，这种温度已经无法给世界树提供足够热量。
<br/>　　比杨德并不知道，就在数天前，这里还是一望无际的艳红岩浆湖。可现在，这里岩浆却全部凝固。
<br/>　　砰！
<br/>　　一圈白烟散开，比杨德身影消失在原地。再次出现，已经站在了另一根世界树的根须之上。
<br/>　　“远处一片红亮，应该是地下熔岩。揍敌客家的小子既然要找岩浆，肯定在那里。”好似一道黑色闪电，比杨德身影在世界根须之间不断折射，速度飞快。
<br/>　　他实力强大，又有适者生存模仿世界树根须。地下空间内的灼热温度，根本无法对他造成伤害。甚至连他的头发都无法烤焦。
<br/>　　不到一刻钟，他便已经接近了地下岩浆河流。黑色地面上，已经有了一些红色斑块。他知道，那是未完全凝固的熔岩。
<br/>　　到了这里，他反而放缓了速度。比杨德并未发动绝来隐藏气息，可他的气息却不断变化。到了最后，竟然和身旁世界树根须变得一致！
<br/>　　对于恶劣环境，他的选择不是对抗，而是融入。这种能力不仅让他生存力大增，也意外获得了强悍的隐匿能力。如果不是看到他本人，任谁用圆感知，都会把它当做世界树的一部分。
<br/>　　小心潜伏在一根根须自后，比杨德双眼冒出红光，死死盯着不远处的熔岩。
<br/>　　在哪里，一个黑红色半透明的蛋正在岩浆中不断沉浮。
<br/>　　透过光膜，隐隐可以看到一个人影。而且相比于其它地方，光蛋周围的岩浆明显在变黑凝固。“果然实在吸收熔岩热量！”
<br/>　　眼睛眯起，比杨德调整了一下姿态，身体微蹲，双脚好似钉子一般，牢牢钉在世界树根须之上。
<br/>　　魔像！
<br/>　　轰！
<br/>　　毫无征兆，一个漆黑色的巨人陡然出现在他身后。
<br/>　　巨人足有十层楼高，身穿威严甲胄，腰间挂着一把漆黑长刀，身体周围黑烟缭绕，好似火焰一般不断摇曳。
<br/>　　可魔像不仅没有威严庄重之感，反而透着强烈的邪恶之意。
<br/>　　最诡异的是，整个巨人全身上下，竟密密麻麻布满了各种脸孔。脸孔分不清男女老少，表情诡异夸张。有的笑有的哭，有的怒吼，有的疯叫！
<br/>　　猛然看去，整个巨人就是无数面孔堆积而成！不断纠结缠绕之间，好似一大堆虫子在蠕动纠缠。恶心而诡异，凶残之气陡然爆发。
<br/>　　比杨德站在原地，衣服被气势吹的猎猎作响。他却脸色沉静至极，对身后巨人面孔的哀嚎怒吼毫不在意。
<br/>　　不知何时，他的手上出现一把长剑，拔剑出鞘，双手紧握剑柄，一脸肃然。
<br/>　　魔像·抚斩！
<br/>　　轰！
<br/>　　长剑猛然斩击。身后巨人像几乎同时拔出长剑，瞬间斩向前方！
<br/>　　黑烟缭绕，魔像巨大长剑，狠狠击中熔岩中的光蛋。
<br/>　　就好似一个皮球，噗的一声，笼罩苏重的光蛋一下被砸进了岩浆内。
