　　过了一会儿，那个声音再度响起：“你怎么会大周天吐纳心法？”
<br/>　　这次叶少阳听清楚了，是一个男人的声音，有点苍老，声音是夹在那些鬼魂的残念之中的。
<br/>　　“什么人？不对，你是什么？”叶少阳反问。很惊奇炼尸缸里为什么还会有“人”在。
<br/>　　“一缕残念。”那人停顿了一下，道，“你要死了。”
<br/>　　“废话，不用你提醒。”要是自己没事的情况下，倒是很想跟他说说话，现在自己被尸气烧得痛苦不堪，连基本的好奇心都快没有了。
<br/>　　“你从哪里学的大周天心法？”
<br/>　　叶少阳没工夫理他，道：“等我死了再告诉你吧。”
<br/>　　“你既然会大周天心法，为什么只有天师的修为？”
<br/>　　叶少阳这次根本理都不想理了，而且已经难受得动用神识都有点费事了。
<br/>　　“告诉我，或许可以救你。”
<br/>　　救我？
<br/>　　叶少阳心中一动，求生的欲望燃烧起来，在神识中说道：“我丹田受损，法力尽失，得亏会大周天心法，才把丹田恢复，但还没有恢复好，无法贮存更多罡气……前辈啊，你如果能救我，赶紧动手啊，我快不行了。”
<br/>　　“我只是一缕残念，怎能救你？”
<br/>　　叶少阳嘴角抽搐，“敢情你在这跟我闹着玩呢？”
<br/>　　“能救你的，是你自己，你为什么不用大周天心法重铸丹田，恢复到最初的强度？”
<br/>　　叶少阳一怔，道：“大周天心法还有这功能？不会吧。这不是养气法门么？”
<br/>　　“一重养气，可提升修为，二重可塑体，复原肉身，你既然会大周天心法，难道不知晓这一点？”
<br/>　　“复原肉身……这是说，就算什么地方坏了，立刻就可以复原吗？这不科学！”
<br/>　　“什么？”
<br/>　　“额，我意思是——”
<br/>　　没等叶少阳开口，男子又说道：“塑体，并非是肉身，肉身若是快速复原，人已经不是人了。人体内又五脏，自成五行，一气贯通，你丹田既已复原，但还无法容纳更多罡气，便是气虚使然，塑体养气，可提升五脏六腑强度……脏器、经脉的强度，变书庶人与法师之别，大周天之法，可以养气强体，庶人修行此法，不多日脏器和经脉便可加强，更何况你丹田只是受损，若运行大周天吐纳心法，顷刻之间，便可恢复……”
<br/>　　这人一口气说了这么多，叶少阳起初听的云里雾里，但是最后一番话，却是让他蓦然心惊，急忙说道：“等等，你说的这些，听上去不错，但我平时用大周天心法吐纳，怎么没这好处？”
<br/>　　“你空守宝山，却不得而入……你而今抱元守一，吐纳之时，在丹田内自成周天，按照我说的做……”
<br/>　　随即念出了几句口诀，并给与讲解。
<br/>　　叶少阳心中泛起了惊涛骇浪：这明明就是大周天吐纳心法的口诀，但是被他这么一讲解，又多出了一重自己之前完全没有想到的奥义。
<br/>　　原来，自己并没有完全理解大周天吐纳心法的秘密……
<br/>　　叶少阳强压住内心的激动，按照他的指点，运用大周天心法吐纳起来……
<br/>　　立刻就感受到了丹田的损伤，果然，从生理的角度来说，自己的丹田的确已经恢复了，但是当有气息通过穴位时，立刻能感受到丹田上的伤痕，这就好像是骨头断了，虽然长好，但必须多休养一段时间，否则刚长好的骨头很容易再次碎裂，因为在其内部，还是有些地方并没有复原。
<br/>　　叶少阳的丹田也是这样，现在，他通过那个神秘人的提醒，感受到了丹田“内部”的伤，然后用大周天心法修复起来……
<br/>　　现在唯一的问题是，尸气还在燃烧他体内的罡气，叶少阳不知道自己能不能在罡气消耗干净之前，将丹田修复完整，只能拼一把，压制住急躁的情绪，将自己的精神完全置身在大周天心法之中……
<br/>　　炼尸缸外，碧清睁开了眼睛，神色之中，流露出了一丝困惑。
<br/>　　之前那一抹残念跟叶少阳的交流，是在神识中进行的，碧清完全不知情，她只是通过操控尸气，感受到了叶少阳身体内部的一些微妙的变化。
<br/>　　她也不知道发生了什么，只能继续操控尸气，燃烧叶少阳体内的罡气。
<br/>　　蛤蟆峡谷，古墓的洞穴外面，梧桐一行人坐在地上，各自吐纳恢复着，之前对抗五行杀阵和碧清根据五行杀阵改变的阵法中，他们都消耗了大量的法力，此刻亟需恢复。
<br/>　　调息了一个周天之后，梧桐站起来，朝洞口望去，面色中带着几分凝重。
<br/>　　终于，洞穴里头传来了一些动静，有人在往外走。
<br/>　　梧桐急忙前去迎接，结果……出来的却是陈晓宇。
<br/>　　“叶少阳呢？”梧桐看他后面没人，急忙问道。
<br/>　　“他……被那个邪物抓住了，可能已经死了。”
<br/>　　什么！
<br/>　　梧桐当场石化，一把抓住他的肩膀，失声道：“他为什么会被抓住？是不是跟你有关！”
<br/>　　“当然没关系……嗯，你们没看我的猴王也牺牲了，我是好不容易逃出来的！”
<br/>　　梧桐愣了几秒钟，松开他，转头望着大伙，说道：“我们去救叶少阳！”
<br/>　　大伙面面相觑。
<br/>　　“我们刚逃出来，现在下去不是送死么！你也不是不知道那邪物的厉害，在古墓里，我们不是她的对手！”陈晓宇第一个表示反对，他刚从下面逃出来，想起碧清和那个可怕的阵法，仍然心有余悸，说什么也不想下去。
<br/>　　“既然要下去，我们当时不要出来就对了，现在要下去的话……”卢晓清也在犹豫。
<br/>　　“当时我们以为叶少阳一定会出来，怎么知道会是这样，不要忘了，是他把我们救出来的！没有他破阵，我们现在可能都已经死了！”
<br/>　　梧桐一席话说得大伙都不言语。
<br/>　　陈晓宇道：“他救我们，也是让我们活下去啊，不给为了又让我们去送死，而且按照时间来说，他……可能已经命丧那邪物之手了，我们现在下去送死，很没意义。”
