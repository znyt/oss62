　　第730章
<br/>　　魏牧冲深吸一口气，可以看出来，他是极勉强才压抑住内心翻腾的情绪。
<br/>　　之所以肯压抑自己的怒火，是因为他已经认出了程大雷。
<br/>　　得到手下传来的消息后，他便马不停蹄的赶来。消息中说对方的坐骑是头黑牛，再加上对方粗犷的相貌，不计后果的做事方式。这已令魏牧冲确认对方的身份。
<br/>　　程大雷表情已经流露出不满：“怎么还不跪下？”
<br/>　　满堂俱静，刚才大家还怀疑是不是耳朵出现幻听。这次是千真万确，然后大家觉得，这莽汉疯了不成。在汝南城伤人动手，该想的是自己能不能活着离开，竟然还想一城之主与他下跪。
<br/>　　接下来，大家就看到不可思议的一幕，魏牧冲双手合在一起，微微弓腰，向程大雷行了一礼。
<br/>　　程大雷眯了眯眼睛，却也没再说什么。
<br/>　　“敢问程当家，入城之后为何无故伤人，打伤我城内官兵？”魏牧冲。
<br/>　　众人皆是一怔，随之已有人推测出程大雷的身份。怪不得敢要魏牧冲于他下跪，原来是那一位。据说陛下已经传旨到凉州，要他入京接受封赏，十拿九稳是下一任凉州王。
<br/>　　如此，魏牧冲与他跪一跪也不算什么。
<br/>　　只是大家不知道，他竟然真的敢来？他难道就不怕死么。
<br/>　　在知晓程大雷身份后，满场哗然。朝九山身体噗地跌倒，从嘴里咕咚咚涌出血来。
<br/>　　刚才的事情对他来讲是奇耻大辱，他一直憋着一口气。但如果伤他的是程大雷，那就不是耻辱，反而是荣耀。毕竟，自己活着从对方手下走脱了。
<br/>　　面对魏牧冲的询问，程大雷目光放在那年轻军官身上，道：“这是你儿子？”
<br/>　　“正是犬子魏明。”
<br/>　　程大雷微微点头，道：“你认得这柄剑么？”
<br/>　　魏牧冲当然认得这柄剑，就算没亲眼见过，也听说过。
<br/>　　“你大概不知道这柄剑是从何来的，是明帝老人家亲手交给我的。”程大雷道：“当初明帝将剑交给我时，他对我说，帝国混蛋玩意太多，日后你看到有那个不顺眼，就持这柄剑斩了他，上斩昏君，下斩王侯。”
<br/>　　程大雷忽地一抬眼：“魏牧冲，你可知罪？”
<br/>　　魏牧冲眼皮跳动了两下，攥紧的拳头青筋暴露，到最后他怒极反笑：“本官做人做事，上对苍天，下对厚土，中间对得起苍生百姓，何罪之有？”
<br/>　　“你瞧瞧，说你胖你还喘了起来。”程大雷道：“我这刚入城，可就听说你纵子行凶，让其在城中勒索卡要，谁若不给便视为崔府同犯。”
<br/>　　魏牧冲一楞，回头看仍跪在地上的魏明：“此事当真？”
<br/>　　“话说得好听，你对得起天对得起地我不知道，可这汝南城的老百姓，你怕是真真对不起。”程大雷。
<br/>　　魏明脸色惶恐，忙道：“父亲，你莫听他的，没有这样的事。”
<br/>　　满头血污的李槐冷哼一声：“此事不假，魏公子自从入城以来，强掠财物女人，有一人家的女子有些姿色，便被他强掳走，对方的丈夫去告状，直接被打死尸体丢在大街上。”
<br/>　　魏牧冲手腕颤抖，魏明则是脸色发白，忙不迭道：“父亲，你听我解释……”
<br/>　　“为国为民，为天为地，终究是没有自己儿子重要喔。”
<br/>　　魏明双目中的怒火渐渐消失，漠然看着跪在地上的儿子，道：“明儿，你怕不怕死？”
<br/>　　“父亲，我……”
<br/>　　“不要怕，为父亲手送你上路。”
<br/>　　他拔出随身佩戴的腰刀，闭上眼睛，砍在魏明脖子上。
<br/>　　程大雷腾地一下起身：好狠地人。
<br/>　　想要将一人的脖子砍断，并不是一件容易的事，普通人根本做不到。魏牧冲这一刀没能让魏明立刻毙命，他倒在血污中，临死前还唤了几声『父亲』方才气息断绝。
<br/>　　程大雷倒真是误会了这个魏牧冲，他接手汝南城城主，里里外外有许多事情要处理，忙得不开开交。而自己儿子，本是个老实人，无论如何也不会生出这种事来。
<br/>　　可他那里知道，魏明跟着他过了二十年苦日子，别人欺负他也不敢还嘴，只因父亲的官职太小。这次好不容易抓住这汝南城城主的实权官职，自然要把前二十年没享受过的统统还回来。
<br/>　　程大雷倒抽一口凉气，缓缓坐下，只感觉后背一阵阵发寒。
<br/>　　魏牧冲眼里落下两滴老泪，一挥手道：“抬下去，昭告全城，再有徇私舞弊，侵扰百姓者，斩立决。”
<br/>　　谁能不怕这位斩子的城主，众人感觉比刚才看到程大雷还感恐惧。七手八脚将魏明的尸体抬了下去，过了片刻，二楼渐渐变得清净。
<br/>　　“犬子枉法，死有应得。”魏牧冲冷漠道：“现在死的该是程当家了吧？”
<br/>　　“喔，你敢杀我？”
<br/>　　“为何不敢。杀你一人，能还帝国太平，便是陛下要我的脑袋我也敢杀。”
<br/>　　“有意思，真有些孤臣逆子的劲儿，但杀我总该给个理由吧？”
<br/>　　魏牧冲指着程大雷鼻子，破口大骂：“你这狗贼，祸害帝国，一件件一桩桩，办下的恶事罄竹难书，现在想要杀你，没有其他理由，只有四个字：为国为民。”
<br/>　　“好一个为国为民。”程大雷拍手道：“那么我倒是要问魏城主几件事。当初戎族卷土而来，帝国危在旦夕，城主在何处。诸侯纷争，祸及苍生，城主在何处，陛下被相府控制，整日以累洗面，我为国为民的魏大人呐，你到底在何处？”
<br/>　　魏牧冲一楞，竟有些无言以对，忽道：“你那时又在何处？”
<br/>　　“嘿，我还怕你不问呢。”程大雷道：“当初是谁以一己之力当戎族十万铁骑，又是谁殚精竭虑，保凉州平安，是谁智取宋伯康，陛下深夜闹长安，里应外合，帮助陛下一举夺回权柄？”
<br/>　　程大雷一拍桌子：“是谁，是谁，是谁？”
<br/>　　“当然是我们程当家。”刘发财与徐神机异口同声。
<br/>　　“然也。”程大雷轻轻点头，面向魏牧冲：“睁大你的眼睛瞧瞧，本当家身后背着响当当四个字：为国为民。”
<br/>(本章完)
