<br/>    一席酒宴吃完，刘玄刹拿出一副地图道：“这地图上共有三十八处禁地，有标记的便是已经有大君前去探索的地方，我们可以选择那些没有大君探索的地方前去，这里是关于这些禁地的介绍，诸位可以拿去观看一番，让我们决定先去哪一个禁地探索。”

<br/>        陈明也是接过了一份介绍，开始看了起来，不过看到的第一处禁地的介绍，就让陈明怦然心动，小心肝扑通扑通跳个不停。

<br/>        太初古林。此处为太初宗的宗门所在地，此宗，乃是最开始修行道经的宗门，不过有一日，天降流星雨，三日不息，此宗覆灭，便再无起势。
<br/>    
<br/>        太初宗蓄养古树，古树诞生了灵智，却不伤太初宗之人，流星雨之后，太初宗覆灭，古树却是扎根于地底，存活了下来，而后进入其中的修行者基本上都死在了里面，原因不明。可能是诞生了灵智的古树所为。
<br/>    
<br/>        一个修行道经的宗门，不是正好是凌仙现在所需要的吗？
<br/>    
<br/>        而且这个宗门可是最先开始修行道经的宗门，必定对于有着能够提升道经境界的宝物存在。
<br/>    
<br/>        其实陈明对于道经的调查也很多，但是就是没有调查出来，这让陈明也很奇怪，毕竟既然有这本经文存在，便一定有着它的历史，直到现在陈明才明白了过来，原来是一处禁地怪不得没人知道关于道经的消息。
<br/>    
<br/>        说实话，自己就好像是给这些主角开挂的，现在李素衣的挂是开满了，也应该给凌仙再开个挂了。

<br/>        刘玄刹看众人已经看了一会儿，问道：“你们有什么提议吗？”

<br/>        陈明直接开口道：“要不我们就去太初古林吧？”
<br/>    
<br/>        刘玄刹道：“太初古林太过危险，很久没有任何消息传出来了，还是不要去了。如果你坚持的话，至少要告诉我一个理由。”
<br/>    
<br/>        陈明继续道：“我修行道经已经修行到了悟道境，算不算理由。”
<br/>    
<br/>        刘玄刹继续道：“在太初古林之中，树木丛生，这些树木可不是普通的树木，都诞生了灵智，甚至有可能有着王级的修为，而且不在少数。”
<br/>    
<br/>        紫霞仙子道：“陈明在群战之中确实很强。”
<br/>    
<br/>        刘玄刹：......
<br/>    
<br/>        刘玄刹愣了片刻，然后道：“那好吧，就去太初古林了，对了，曙光大君，我给你一件东西，这是我们古仙门大君才能拥有的东西。”
<br/>    
<br/>        刘玄刹张开手，一个罗盘出现在手中，刘玄刹递到陈明的面前，然后道：“这是能够群星仙门群星弦盘的罗盘，你可以在里面发出你所知道的信息，求救或是告诉其他人别的什么消息，并且通过罗盘标注我们已经准备去探索的禁地。”
<br/>    
<br/>        陈明拿过来一看，意识沉浸在其中，可以看得到其他大君发出的消息，还有着一副地图，和刘玄刹拿出来的差不多，在地图上面，太初古林之上已经有了刘玄刹的名字。
<br/>    
<br/>        嗯，这是修仙界的朋友圈吗？
<br/>    
<br/>        还有这种黑科技？
<br/>    
<br/>        就是门槛有点高，大君才玩的起。
<br/>    
<br/>        陈明道：“我还准备带一个弟子过去历练一番，应该没事吧？”
<br/>    
<br/>        刘玄刹道：“你如果一定要带去的话，只能你自己保护。”
<br/>    
<br/>        陈明笑道：“没问题！”
<br/>    
<br/>        既然是去给；凌仙寻找造化，怎么也得把这个小家伙带去，毕竟那幸运光环可不是开玩笑的！

<br/>        陈明道：“我门下那个弟子，已经修行到了入道境巅峰了，你也知道，道经的修行，很慢的。”

<br/>        刘玄刹已经不想再纠结这个问题了，随他去吧，“带上你的弟子准备一番，我们准备出发了。”

<br/>        卓青瑶拱手道：“弟子明白了。”

<br/>        三位大妖王的额头滴下一滴滴冷汗，“我等，必定在三天之内，将雾飞花的首级送到大宗主面前！”
