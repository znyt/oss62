　　当他选择对帆系萝玛动手的时候，就代表已经彻底和刘哔绑在一起了。
<br/>　　“这样啊，那这个喰种，就交给你来处理吧。”
<br/>　　点了点头，刘哔没将此事放在心上，牵起什造的小手，一起走出了办公室的大门。
<br/>　　外面，拍卖会还在火热进行着，他们对接下来要面临的危险，根本就没有一点防备。
<br/>　　毕竟这是颇具盛名的小丑组织举办的拍卖会，往常也举行了很多次，也没有哪一次出过意外。
<br/>　　当第一声惨叫声出现在会场的时候，这些喰种们还不知道发生了什么事情，但随着屠杀开始，会场顿时混乱了起来。
<br/>　　刘哔和什造一起，将拍卖会变为了一场杀戮盛宴。
<br/>　　……
<br/>　　二十区，西尾锦的住宅中，刘哔嘴里叼着一根吸管，将腿搭在茶几上。
<br/>　　电视中正在放着新闻联播，报道着一件轰动整个东京的新闻。
<br/>　　“昨夜，在八区的宜家拍卖行中，发生了大规模的喰种组织火并，现场尸体确认均为喰种，离奇的是，所有尸体的死亡特征似乎都是出一种手段，进一步消息，正向CCG方面确认。”
<br/>　　“经统计，此次事件死亡喰种高达一千余名，是近年来，规模最大的一次喰种组织火并，望各位市民做好防范，减少外出，注意个人安全……”
<br/>　　将吸管插进杯子中，一股红色的液体随着吸管进入了刘哔嘴里，微咸，不过很鲜。
<br/>　　这是他回来的时候，顺手从外面带回来的零食，拍卖会有很多这样的东西。
<br/>　　昨天的收获很大，吞噬了那么多喰种，让他实力再次飞跃，并且本来已经不多的积分，又重新积累到了十多万点。
<br/>　　可惜，那些喰种的实力参差不齐，大部分也只是价值一百多积分的小喽啰而已。
<br/>　　有了积分，他自然在考虑怎么提升实力的问题。
<br/>　　本来按照他一开始的想法，是想要兑换一部功法，但随后想了想，放弃了这个打算。
<br/>　　有吞噬技能在手，他还用费什么力气进行修炼么。
<br/>　　最终，考虑再三之下，他花费了十万积分，用以兑换一件武器，SSS级武器，火之高兴。
<br/>　　虽然名字很让人吐槽，但是功能却还算对得起这个价格，除了超高的属性增幅之外，还额外有一个特殊技能，让他可以随意控制自身周围的火焰。
<br/>　　任何火焰，包括传说中的，心魔之火，三位真火，之类的诸多种种。
<br/>　　并且置身于火焰之中，他还可以大幅度提升自己肉体的恢复能力。
<br/>　　这是一个很强大的技能，可以说兑换了这件武器之后，他就免疫了一切火属性伤害。
<br/>　　十万积分，花的并不心疼。
<br/>　　反正等养猪场彻底建立起来了，他也就不必考虑积分这种东西了。
<br/>　　剩下的那些积分，他暂时都没有动用，喰种世界这边，他要返回的话，需要花费额外一千积分，况且白日庭这边，也需要筑基丹要维持。
<br/>　　猪饲料是不可以断的。
<br/>　　什造的话，他也要想个办法，带回他原本的世界里去。
<br/>　　这里太过于危险，把什造留在这，他不放心。
<br/>　　系统是无法直接带人回去的，但他从僵尸先生世界中获得了一副封印卷轴。
<br/>　　他可以暂时将什造封印，然后跟随卷轴一起带回去，这点可行，他已经得到了系统的确认。
<br/>　　为了防止意外，他直接就将什造封印进去了，反正按照时间来算，距离回归也只有不到半个月的时间了。
<br/>　　将手中的杯子放到一边，刘哔犹豫了一下，穿衣走出了大门。
<br/>　　他要在回归之前，去收集一些食物和生活用品，只是需要注意，一定不要让人发现他就可以了。
<br/>　　现在整个东京都在通缉他。
<br/>　　……
<br/>　　半个月的时间一晃而过，在最后一天的时候，他对西尾锦交代了一些事情，然后独自一人来到了郊区，准备即将到来的传送。
<br/>　　大概两三个小时之后，一股眩晕感袭来，等他清醒过来的时候，周围的场景已经变幻成一片残破不堪的建筑。
<br/>　　这是他前往喰种世界时，特意寻找的地方，这是一家生产手机零件的工厂，周围丧尸很多，也很少有幸存者来这里。
<br/>　　看到四周游荡的丧尸群，刘哔莫名感到一阵亲切。
<br/>　　心中记挂着家里，他也没有在这里逗留太长时间，直接从工厂中走出，寻找到了一个没有丧尸的地方。
<br/>　　随后，解开了卷轴上的封印，将什造给释放了出来。
<br/>　　这个世界的情况，他大致已经给什造交代清楚了，但她依旧选择跟随他一起过来。
<br/>　　对什造来说，或许在哪里也都没有什么区别吧。
<br/>　　“闷死了，在里面呆着这么长时间，什么都看不见！”
<br/>　　一出来，什造就嘟起小嘴，抱怨了起来。
<br/>　　为了减少不必要的麻烦，刘哔很少打开画卷，足足半个月的时间啊，什造都是呆在一片虚无之中。
<br/>　　这种感受刘哔经历过，自然知道是什么滋味。
<br/>　　“好了，这不是一回来，我就赶紧给你放出来了么。”
<br/>　　摸了摸什造的头发，刘哔笑着开口，一边寻找着还能发动的车子。
<br/>　　但遗憾的是，找了很久他都没有找到，毕竟这里不是城市中心，路上的车子，汽油也大都被抽走了。
<br/>　　被迫无奈，两人也只能步行，所幸的是，这里距离基地也不算太远。
<br/>　　大概半个多小时就能赶到吧，如果不考虑暴露速度的话，甚至能更快一些。
<br/>　　“诶，这个就是你说的丧尸吗？”
<br/>　　两人走在路上，什造对于所见的一切都很惊奇，从小在那种环境下成长，她也没怎么接受过电影之类的东西。
<br/>　　对于丧尸自然也是一无所知。
<br/>　　“是啊，可别小看这些东西，我如果不在的时候，一定不要被这些家伙碰到，不然的话，你也会失去自己的意识，变成和它们一样的家伙。”
<br/>　　拉着什造饶开了丧尸，刘哔有些担心的嘱咐道，毕竟这可不像喰种世界，被丧尸伤到的话，真的会死。
<br/>　　乖巧的点了点头，什造眨着眼睛，有些好奇的四处打量着。
<br/>　　这个世界中的很多东西，都超出了她的认知。
