　　“不过，此人能够来到归墟天，想必也不是什么弱者。”
<br/>　　“那又如何？我们三个人，就小子就是一个，而且生死境三重的境界，可以翻起什么风浪来。”
<br/>　　“这样落单的小子，最好对付了。”
<br/>　　“还是先问问这小子的来历再说。”
<br/>　　“要是没什么来历，就直接杀了他，看此人身上，似乎是有着一种龙威，显然是有着龙族血脉，或者是得到过什么龙族奇遇，这种奇遇正好给我们瓜分。”
<br/>　　“他的精血归我，我要练就一身魔功，正是需要武者的精血。”
<br/>　　“好，一会出手，干净利落，不要引来其他人。”
<br/>　　这三名炎阳门武者灵识交谈的一阵，突然一人开口道。
<br/>　　“不知道兄弟，是哪个门派的，我们炎阳门，跟不少宗门都有交好，这归墟天危险重重，也可以互相照应一下。”
<br/>　　“哦？”
<br/>　　听到这炎阳门弟子的话，楚风眠心中突然冷笑一声。
<br/>　　他们的灵识交谈，怎么可能瞒得过楚风眠，每一句话，如今都被楚风眠听得一清二楚。
<br/>　　不过楚风眠也没有揭穿，而是笑了笑开口道。
<br/>　　“我叫楚风眠，是北邙学院弟子，进入归墟天的时候，被传送到了一处偏远之地，如今才赶了过来，还没有联系到其他师兄弟。”
<br/>　　楚风眠笑着开口道，好像是如同那种毫无戒心，涉世不深的武者一般。
<br/>　　“北邙学院的弟子。”
<br/>　　“果然是孤身一人。”
<br/>　　“北邙学院，还不如我们炎阳门，我们杀了他，北邙学院也不敢来找我们的麻烦！”
<br/>　　那三尊炎阳门武者眼神对视了一眼，都是从对方的眼神之中，读懂了其中的意思，看向楚风眠，如同是看向了一只待宰的肥羊。
<br/>　　“原来是北邙学院弟子，北邙学院，跟我们炎阳门也是有着不少往来，看楚兄是独自以一人，来到归墟天，在这里危机重重，楚兄一人也太危险了，不如跟我们一起，来探索归墟天如何？”
<br/>　　一名炎阳门武者，笑着开口道。
<br/>　　“我们如今发现了一处远古遗迹中的洞府，还没有探索，本来我们三人正打算的探索的，没想到居然就碰到了楚兄，遇到及时缘分，不如楚兄跟我们一起去探索，所有好处，我们每人一份如何？”
<br/>　　“什么？远古洞府？你们居然在归墟天中，发现了远古洞府？”
<br/>　　楚风眠装出一副极为吃惊的样子来，不过这反应，倒是极为正常。
<br/>　　归墟天中，最为庞大的宝藏，便是无数远古洞府，归墟天中曾经有着不少的远古宗门。
<br/>　　在这些远古宗门中，也诞生有着不少的强者，他们在归墟天的角落中，开辟洞府，许多洞府，都不亚于颜古龙帝的洞府。
<br/>　　对任何武者来说，都是一个极大的机缘。
<br/>　　“不错，正是一处远古洞府，楚兄愿意前来，也正好可以帮我们对付其中的危险，得到的好处，我们都可以平分。”
<br/>　　那炎阳门弟子，一边开口道，一边直接转过头去，准备带路。
<br/>　　楚风眠站在原地，故作思考了一下，便是也化为了一道遁光，直接便是追了上去。
<br/>　　“蠢货。”
<br/>　　“这个废物，居然就这样跟了上来。”
<br/>　　“正好，一会等带他去准备好的地方，直接杀了他！”
<br/>　　那三名炎阳门弟子，对视一眼，都是露出了几分冷笑，眼神中已经是布满了杀意。
<br/>　　人无害虎心，虎有伤人意。
<br/>　　本来这些炎阳门弟子，突然出现，楚风眠也没有打算对他们出手。
<br/>　　毕竟楚风眠跟炎阳门，可是好无恩怨，楚风眠也不是滥杀无辜之人，没想到这些武者，居然是算计到了楚风眠的头上。
<br/>　　楚风眠要是这都能隐忍下去，那他就不是楚风眠了。
<br/>　　楚风眠的心神中，都是蕴含有着几分杀意，不过他也并不着急动手，他倒是要看看，这些炎阳门弟子，准备如何来对付他。
<br/>　　三名炎阳门弟子在前，化为一道遁光带路。
<br/>　　而楚风眠，则是在后面，紧跟其后，一共是过了一个时辰的时间，周围的地貌已经是变化为了无数次，突然在其中，出现了一座巨大无比的山谷。
<br/>　　这山谷，从外看去，倒是一副鸟语花香的样子，像是一处福地。
<br/>　　“诸位，这远古洞府，到底在哪里？”
<br/>　　楚风眠一边在后面跟随，一边不由有些着急的询问道，好像是已经有些对着远古洞府，迫不及待了。
<br/>　　看着楚风眠的反应，那些炎阳门弟子的心中，都是冷笑连连，脸上却是极为和善的开口道。
<br/>　　“楚兄不要着急，就在前方，在这一处山谷之中，便是那古洞府的所在。”
<br/>　　说着，这三名炎阳门弟子，便是一个加速，直接是进入到了其中。
<br/>　　楚风眠在后面，也是紧跟其后，突然遁光飞到其中。
<br/>　　就在楚风眠，踏入到了这山谷之中的一刻，周围的样子，却是突然之间，天旋地转。
<br/>　　整个山谷，便是变的鬼气森森起来，楚风眠的脚下，都是一处黑沼泽，其中还迷茫有着一股瘴气。
<br/>　　哪里还有刚刚一处福地的样子，现在看起来，这一处山谷，简直就是一处凶地，大凶之地。
<br/>　　“三位，这远古洞府，就在这里面？”
<br/>　　楚风眠都是不由的停下脚步，打量了周围一眼，故意的开口询问道。
<br/>　　“哈哈哈哈，远古洞府？什么远古洞府，小子，没想到你居然这么愚蠢，直接就是上钩了，不费我们一点功夫，现在乖乖将你身上得到的一切奇遇机缘，都交出来，我可以留你一个全尸。”
<br/>　　那刚刚开口的炎阳门武者，脸上已经是没有了刚刚的和善样子，如今已经是一脸冷笑，显得极为阴冷。
<br/>　　而在他的身边，那另外两名炎阳门武者，也是同样，一双双眼睛，看向楚风眠，满是贪婪，杀意。
<br/>　　“你们？你们算计我？炎阳门可是正道宗门，怎么会有你们这种弟子！我是北邙学院的弟子，你们敢来杀我？”
<br/>　　楚风眠扯着嗓子大喊道，似乎是极为吃惊，极为恼怒的样子。
