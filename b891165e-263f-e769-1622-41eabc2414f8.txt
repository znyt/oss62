        连华生表示，自己不会再跟他傅夜沉提童筱颖的任何事情。
<br/>
<br/>        只不过，作为朋友，他的初心，只是希望他傅夜沉能从这段爱而不得的感情里走出来。
<br/>
<br/>        对于一个失恋的男人来说，连华生还发现，自从童筱颖和傅昼景双宿双飞去了澳洲后，傅夜沉从此以后居然滴酒不沾。
<br/>
<br/>        私宅里珍藏的好酒，全都被他连华生一个人喝光了。
<br/>
<br/>        其实，不管是在家里，还是出门在外有应酬，傅夜沉都不再喝酒，应酬的时候，他让助理郑忠威替他请了公关一起陪同，酒桌上好代他喝酒。
<br/>
<br/>        连华生一直不太明白这是为什么，明明他傅夜沉是个能喝酒的人。
<br/>
<br/>        其实，不喝酒的原因，傅夜沉是因为害怕自己会醉，都说喝醉了可以麻木自己的心脏，让自己不再记得痛的滋味。
<br/>
<br/>        以他现在的身份，喝醉了，说不定还能邂逅一段“美丽”的艳遇。
<br/>
<br/>        可是，他宁可让自己痛着，好时刻提醒自己应该把所有的心思全花在工作上。
<br/>
<br/>        如果，他能真正走出来，接受另一段新感情的话……
<br/>
<br/>        傅夜沉不禁苦笑，那或许他就不用自我封闭这么久了。
<br/>
<br/>        忙碌了一天，也结束了一天的工作行程。
<br/>
<br/>        入夜后，“夜惑”里人声鼎沸，人满为患，形形色色的男男女女全聚集在此寻欢作乐，浪笑声不绝于耳。
<br/>
<br/>        这三年来，费尔南如果不是因为经常把童筱颖挂在嘴边，讨好傅夜沉，恐怕“夜惑”早就关门大吉了。
<br/>
<br/>        傅夜沉仍旧喜欢坐在清吧区的VIP雅座里，只喝鲜果茶，安静地听着台上的驻唱歌手唱歌。
<br/>
<br/>        其实，他很少来“夜惑”，或许是因为昨晚那个梦的缘故，今晚他竟然鬼使神差地跑来了“夜惑”听歌。
<br/>
<br/>        只可惜，物是人非。
<br/>
<br/>        “夜惑”里不再有“小谢”，不再有他爱的那个筱颖。
<br/>
<br/>        台上唱歌的那个女人，在他听来，永远都没有他的筱颖唱得好听。
<br/>
<br/>        哦，不！
<br/>
<br/>        已经不是他的筱颖了……
<br/>
<br/>        傅夜沉随之起身，一首歌还未听完，便离开了清吧区。
<br/>
<br/>        “嗨，美女，你一个人这是急着去哪儿啊？要不要哥哥我送你一程？”一个痞里痞气的公子哥，抓住了一个看起来有些焦虑的女孩的手腕。
<br/>
<br/>        “我……我走错地方了，对不起……”女孩仓促中又显得一副好害怕的样子，像是第一次来这里。
<br/>
<br/>        七、八个男人近乎着迷地望着她，一点也没有放她走的打算。
<br/>
<br/>        “都来这里玩了，还装什么纯啊！”他们朝她靠近，垂涎着她如凝脂般的肌肤，伸手摸着她的脸颊。
<br/>
<br/>        女孩抵死不从，挥手抵挡，但他们的手非常的不规矩，不禁让她恼羞成怒：“滚开！我不是来玩的！”
<br/>
<br/>        “呦，还是个‘小辣椒’啊！有意思！”
<br/>
<br/>        女孩惊惧地往墙角缩，她听不太清楚他们说了什么，这里的摇滚乐实在是太吵。
<br/>
<br/>        但是她却知道，他们准没好意，联想到刚才的画面，她胃里起了一阵痉挛，背脊泛起一阵冰寒。
<br/>
<br/>        “我要走了！”女孩皱眉，声音极度沮丧。“麻烦你们让一让！”
<br/>
<br/>        她的手心淌汗，内心害怕极了，面前这些人的平均身高都在一八○以上，而她的身高却只有一六二，夹在他们当中更显得弱小可欺。
<br/>
<br/>        他们朝她逼近，她已没有退路，向四周张望，期待找到能帮助她的人。
<br/>
<br/>        忽然，女孩的眼前一亮。
<br/>
<br/>        有位穿着白衬衫黑色西裤的英俊男子刚好路过。
<br/>
<br/>        他的气质与周遭格格不入，高贵得恍若出淤泥而不染。
<br/>
<br/>        有救了！
<br/>
<br/>        女孩兴奋地想着，不假思索地朝那男人冲了过去，一把抓住了那男人的手。
<br/>
<br/>        “大哥，能在这里碰见你真的是太好了！”女孩盯住男人，一声娇柔的呼唤后，才发现这个男人有双深邃且漂亮的黑眸。
<br/>
<br/>        傅夜沉身心一怔，眯起眼看着这个不知打哪里冒出来的女孩。
<br/>
<br/>        她的这一声“大哥”，叫得让他差点误以为是筱颖。
<br/>
<br/>        更重要的一点是，这女孩的行为举止，都与筱颖有几分神似。

<br/>        这女孩很漂亮吗？无庸置疑！
<br/>
<br/>        一头如瀑布般的黑亮发丝闪闪动人，随着她左张右望的动作，在香肩上甩动出诱人的线条。

<br/>        在酒吧内这彩光灯的照射下，她那双大眼乌溜而灵动，她的下巴十分小巧迷人，唇形完美，红扑扑的脸颊上还有一对迷人的小梨涡。
