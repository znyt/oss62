　　　　幻月洞府结界内部静得出奇，见无人应声，慕清霄望着远处的一颗有两人环抱粗的巨树，道：“来都来了，难道是想让我请你出来吗？”
<br/>　　“莎莎...”
<br/>　　草叶摩擦的声音响起，在周围寂静的环境中显得格外响亮，紧接着一道白衣身影自树杆后走了出来。
<br/>　　万剑一目光前所未有的凝重，紧紧盯着慕清霄，早就该想到，如此强者能看上的东西，青云门也就只有幻月洞府中的古剑‘诛仙’了。
<br/>　　慕清霄嘴角牵起一丝莫名的笑容，其实他出门的时候就知道万剑一跟在他的身后，他并没有收敛气息，只是没有点破罢了，现在看来，想要获取天书，必须搞定后者啊。
<br/>　　“阁下，诛仙剑乃是我青云门镇派宝剑......”
<br/>　　慕清霄微不可查的摇头，他自然知晓诛仙剑是什么，否则也不会来此，望着万剑一，瞳孔中出现漆黑色原子图案，永恒万花筒写轮眼现！
<br/>　　月读释放，恐怖的精神力量向万剑一席卷而去，后者措不及防，身躯一颤便没了动静，眼中同样有黑色原子图案！
<br/>　　慕清霄不再理会万剑一，转身向幻月洞府走去，他有必要抓紧时间，月读精神世界虽由自己控制，但万剑一这种强者，保不准什么时候就自己冲破开来。
<br/>　　等自己拿到诛仙剑，在解开月读，抹去万剑一脑海中的记忆也不迟。
<br/>　　来到洞口，慕清霄负手而立，静静的站在古洞入口处，头顶刻着四个大字“幻月洞府”，右侧有一石壁，平整的石壁之中，镶嵌着一块石板太极图案。
<br/>　　神识扩散而出，除了身后处于月读精神世界的万剑一，幻月洞府确实没有其他人镇守，想必道玄也想不到，会有人躲过他的耳目来到此处吧。
<br/>　　按道理来说，祖先祠堂是青云门禁地，有万剑一把守，可以说万无一失，而幻月洞府乃是青云门圣地，其中更存放着诛仙古剑，如此重要的圣地不可能无人看守。
<br/>　　然而，在慕清霄的神识范围内，千米范围内再无第三人，只是想到幻月洞府的特殊，也就释然了。
<br/>　　幻月洞府内幻境丛生，如同真实再现闯入者的生平，迷惑闯入者的心智，心智不坚者基本是有来无回。
<br/>　　加上万剑一镇守在右侧千米外的祖先祠堂，外有掌门道玄，谁能够无声无息的来到此处？
<br/>　　不是没有人镇守，而是没有镇守的必要。
<br/>　　慕清霄不再耽搁，走上前去，清楚剧情的他，自然知道幻月洞府开启的方法。
<br/>　　手放在了太极图案上，顿时泛起淡淡青光，身体经脉间流淌着太极玄清道的气息，太极图上同样亮起了青色的光芒。
<br/>　　与此同时，太极图案开始转动，片刻后在太极图案的右边，原本完整一块的石壁突然出现了一个圆环形状的裂缝，随即缓慢旋转着向旁边分开，露出了一个秘密的洞口。
<br/>　　只是这个洞口处却盘旋着一股灰白水雾模样的怪异事物，看过去如雾气，又似水波，旋转不停，里面朦胧不清，一点都看不真切。
<br/>　　望着灰白色的水雾，慕清霄没有丝毫的犹豫，迈起步伐，径直向内部走去。
<br/>　　水雾将他的身影吞没，在慕清霄的身影消失后，两侧石壁悄无声息的回转合拢，完全恢复原本的模样，任谁也看不出，此处已经被开启过。
<br/>　　在踏入水雾中后，慕清霄心弦一绷，脑海中变得一片空白，回过神来，目光不禁变得严肃起来，因为他已经站在了一条宽敞的马路上。
<br/>　　看着四周的建筑物，来往的人群，明显就是现代二十一世纪的地球，此处的景象也有些熟悉，难道......！
<br/>　　“清霄，今天在学校过的怎么样...晚上想吃什么菜，妈妈回家给你做...”
<br/>　　“昨天老师打电话过来，夸我们孩子聪明......”
<br/>　　人行道上，中年男子与妇女，一左一右牵着一名天真无邪的小孩，脸上挂着幸福的笑容。
<br/>　　“爸，妈...”
<br/>　　望着走在最中间的小孩，男子与妇女，慕清霄眼眶微红，低声呢喃，望着眼前熟悉的一幕，两行清泪不知何时自他的脸颊滑落。
<br/>　　一家三口其乐融融，绿灯亮起，走在斑马线上，忽然，卡车喇叭声疯狂的响起！
<br/>　　“滴滴......!”
<br/>　　走在中间的小孩仿若未觉，脸上依旧挂着天真幸福的笑容，紧接着整个幼小的身体，毫无预兆的翻滚出去，最后一股炙热的鲜血喷洒在他的脸上。
<br/>　　接下来的场面可想而知，周围的行人指指点点，有的放声尖叫，也有的拿出手机拨打救护车电话，唯有小孩趴在路边，望着躺在十几米外血泊中的两具尸体，目光呆泄！
<br/>　　慕清霄依稀记得，在最紧要的关头，是他的父母将他狠狠的推向人行道，而他们自己却没有逃过这场灾难。
<br/>　　“哎......”
<br/>　　心中千言万语化作一声长叹，慕清霄闭上双眼，再次睁开时，迷茫已经消散，坚定取而代之。
<br/>　　“爸，妈，愿你们在天堂一切安好。”
<br/>　　慕清霄收敛了下情绪，双膝跪地，额头猛地磕在地上，磕了三个头后，整个幻境空间产生一丝涟漪，紧接空间动荡，最终支离破碎。
<br/>　　当慕清霄再次睁开双眼，已经站在一片奇异的空间中，脚下漆黑一片，宛若噬人的巨口，头顶是一片奇异的苍穹，深紫色厚重的云层，仿佛要将这片天空都压垮下来。
<br/>　　云层之中，雷弧闪烁，不时有白色闪电，携带恐怖天威席卷而下，宛若要破开这片苍穹，倒是有点向强者渡劫时所引来的天地异象。
<br/>　　慕清霄收回目光，视线看向这片奇异空间的尽头，只见一座祭坛屹立在空间的最深处。
<br/>　　完全由白玉所砌成，搭建的祭坛，不管是正前方的台阶，又或者是分布在东南西北的柱子，都是由玉石砌成，说他是祭坛，倒不如说是一座豪华的小型宫殿。
<br/>　　而在祭坛的最中央，一柄长剑悬浮与空中，即便是站在百米开外，慕清霄也能清晰的察觉到，剑身上所散发出的煞气，丝毫不弱于当初的嗜魂魔棒，犹有过之！
