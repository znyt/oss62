　　“嗯！这样就顺眼多了，呵呵！”
<br/>　　一步踏出，元始就再次回到之前的山谷之前，只不过，因为先前的大战，山谷就算有先天阵法的守护，还是被破坏了许多。
<br/>　　就算元始之前修复了一些，但对比之前那种灵气四溢，云雾缭绕，仙音袅袅的环境，现在此地却是少了一种道韵自然的感觉。
<br/>　　这很正常，这大道自然可不是有神通就能恢复的，主要是此地道则破碎的原因。就算经过之前的改善，但想要恢复，还是需要不少时间的。
<br/>　　“唔？看来，此地不能待了，还是要换个地方啊！”元始看到这一幕，苦笑道。
<br/>　　“不过，去哪里好呢？”
<br/>　　元始想了想，突然心中想到一个地方，眼前一亮，道：
<br/>　　“不错，不错，那里有着大阵守护，应该还没有被发现。如今去那里再好不过了，说不定还有灵宝可以捡？”
<br/>　　想到这里，元始便不再耽搁，大手向着阵中一抓，瞬间就将那阵中的宫殿抓起。宫殿迅速变小，等到了身边，便只有巴掌般大小。元始大袖一刷，宫殿瞬间便消失不见。
<br/>　　不再停留，找好方向，元始一步踏出，便已消失原地。
<br/>　　……
<br/>　　世事难料，有时候你不欲招惹麻烦，但，麻烦却偏要前来找你，躲也躲不过。
<br/>　　这不，元始刚来到东海之滨，还没有进海，就遇到了麻烦。当然，这是他的麻烦还是别人的麻烦，就不好说了。
<br/>　　事情是这样的：
<br/>　　这一天，元始出现在东海海滨，刚想要入海寻山，但，就在这时，遇到了洪荒龙族。
<br/>　　东海为龙族的地盘，龙族当然不允许本地地盘上出现不明人物了，再加上龙族高傲的性格，视他人于无物，这样就惹起了麻烦。
<br/>　　元始在东海边转悠，远远地就看到十九位长着蛇头，身披鳞甲的大汉簇拥一位面部白皙，身材修长的年轻公子缓缓走来。只是那年轻公子鼻孔望天，目中无人，端是高傲无比。
<br/>　　元始没有在意，而是继续推算此行目的。
<br/>　　那年轻公子却是龙族子弟，名敖玉，作为龙祖的第四子，先天传承更是深厚无比，而且母亲又是洪荒第一条蛇龙，血脉高贵。综合了龙祖与蛇龙的血脉，就算他不刻意修炼，修为也在快速进步着，今年才出生五百年，修为就比那些修炼一元会的妖兽还强大，已经是玄仙境界了。
<br/>　　他一出生，就像被含金钥匙一般养着，整天下人对他毕恭毕敬，亲人更是对他百般恩宠呵护，这就养成了他高傲无比的性格。而且，作为龙二代，他的确有高傲的资格。
<br/>　　只不过，有些时候，这高傲也是要分对象的，不是什么人都会对你毕恭毕敬。
<br/>　　敖玉在龙宫中待烦了，想要出去玩玩，但，他的父皇母后担心他实力弱，会被人欺负，就一直不让他出去。
<br/>　　这天，他寻了一个机会，带着随从，躲开盘查，从龙宫之中跑了出来。
<br/>　　心情舒畅的他此处跑跑，那处跳跳，玩得不亦乐乎！
<br/>　　这天，他正想着还要去哪里玩呢，但就在这时，却看到了一个身穿白玉道袍，头顶白玉丝带，一声长发随风飘散的道人，那道人端是高贵无比，相貌也昳丽非常，一举一动，都无比和谐。
<br/>　　这道人竟然比本太子还漂亮，这简直不能忍，我要惩罚眼前这个道人，这是他敖玉的第一想法。
<br/>　　于是，他对着那道人便破口大喊道：
<br/>　　“呔！小子，见到本太子还不过来行礼，识相的，赶紧过来跪下，给本太子我磕个响头，若是我心情好，考虑放你一马，要是惹毛了爷，爷让你生不如死！”
<br/>　　元始正在演算此行目的地，他凭立虚空，御风而行，听到声音，斜眼一瞟，便无视那人的喊叫，想尽快赶往目的地。他有预感，自己经过上百年时间的演算推理，已经快要接近目的了。
<br/>　　“哼！混蛋，竟敢无视本太子，上，把他围住，在这东海之中，还没有任何人敢无视本太子。你一个小小的道人，竟敢无视我，那就让我看看，你到底有多少本事？！”
<br/>　　敖玉气急败坏，对着元始大声喝道。显然是一个被彻底宠坏的龙二代，彻头彻底的龙族纨绔。
<br/>　　“嗯？放肆！”
<br/>　　元始见此小白脸竟然如此不识趣，本座心情好，放过你一马，你竟然不知，还来找死，那我就成全你，你是龙族又如何？就算龙祖今日到此，也救不得你。
<br/>　　随着元始的一声大喝，十几个手抄兵器围上来的人便纷纷倒地不起。
<br/>　　“哼！废物！连一个道人都对付不了！”
<br/>　　敖玉见此情况，也顿时知道这情况不妙，但想了想自己有父皇赐予的天龙甲，顿时心中升起一股自信，对着倒地的几人大声骂道。接着又对着元始道：
<br/>　　“小子，你很不错嘛？可敢报上名来？”
<br/>　　“报名？哼，你算什么东西！披鳞戴甲之辈，也凭听到本座的名号。过来！”元始一阵鄙视，大手向着那敖玉一抓，顿时，敖玉不受控制的被元始抓到手中。
<br/>　　“吾乃龙族四太子，吾父皇可是龙族龙祖，识相的赶紧将本太子放了，要不然，让你难堪！”敖玉见这道人将自己抓住，顿时急了，搬出龙祖的大名威胁道。
<br/>　　“龙祖？哼？他算什么东西，你就算是天王老子的儿子，今天遇上本座，也休想如意。”
<br/>　　元始不顾敖玉威胁，将他全身上下乃至神魂法力都尽皆禁锢。之后扔到地上。
<br/>　　“本座《玄黄不灭》刚好到第四重龙脉融龙骨，就恰好遇到龙族，难不成是老天爷在帮我？不过，就眼前此龙身上的龙脉之气，还远远不够本座修炼，其他的又要如何？”元始默默思索道。
<br/>　　“嗯？有了！”元始思索一会儿，心道。
<br/>　　“度心诀！”
<br/>　　元始一声低喝，手捏印诀，一股缥缈无形的力量渗透到敖玉身体之中，不一会儿，原先还大声嚷嚷的敖玉瞬间就变得温顺无比。
<br/>　　元始打开敖玉身上都禁锢，敖玉站了起来，对着元始单膝跪地一拜，道：
<br/>　　“拜见主人！”
<br/>　　“嗯，不错，不错，度心诀效果还真是不错！不过，敖玉，本座不喜‘主人’这个称呼，今后就叫本座尊主。”元始对着敖玉吩咐道。
<br/>　　“是，尊主！”
