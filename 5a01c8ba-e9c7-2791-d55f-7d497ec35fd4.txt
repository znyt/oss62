<br/>　　“屠霄，你是否愿意和他们切磋？”龙腾转身问向苏莫，他知道今日若是不满足这两人，估计以后没完没了了！

<br/>　　因为，龙洌的实力实在太差了，他弹指之间就能击败，至于龙城也不会强到哪里去！
<br/>　　“哈哈！好！”

<br/>　　龙城闻言大怒，脸色铁青的怒吼道：“你一个奴仆，居然如此和我说话？”
<br/>　　龙城很愤怒，虽然这个屠霄的修为，远高于他，但只是一个奴仆，身份与他相比天差地别。
<br/>　　“呵呵，龙城，你别自作多情了，这里没有你的事！”龙洌笑着说道，既然屠霄答应了他的挑战，那就再好不过了。
<br/>　　至于龙城，想和他抢夺奴仆，简直是痴心妄想。
<br/>　　“龙城，我可没有说过，愿意把奴仆让给你！”龙腾沉声说道。
<br/>　　“你……！”龙城面色一滞，心中愤怒到了极点，但是现在，有龙腾和龙洌在此，他也没有办法。
<br/>　　“好，我就让你们先战一场！”龙城声音冷漠的说道，言罢，便闪身飞了回去。
<br/>　　他不得不暂时退却，毕竟，今日的比斗，本来就是龙洌的事情，他若是执意纠缠，反而落人口舌。

<br/>　　随后，龙腾和司空炎两人，纷纷闪身推开，给苏莫和龙洌让出地方。
<br/>　　现场寂静无声，众人都打起了精神，龙洌败给了炎空，会不会再败给屠霄？
<br/>　　众人不确定，毕竟，屠霄的真正战力，还没有表现出来。
<br/>　　场中，苏莫和龙洌相对千丈而立，目光碰撞在一起，虚空都在震颤。

<br/>　　据说天梦元神，能将人直接拉近梦境之中，所以，他直接严明领教天梦元神。
<br/>　　“好！”苏莫点了点头，陡然之间，他的身上绿光升腾，天梦之光弥漫八方，直接将龙洌笼罩在了其内。
<br/>　　下一刻，龙洌便被拉近了天梦元神早就的梦境之中。
<br/>　　梦中无岁月，龙洌的实力，给苏莫提鞋都不配，在梦境之中，苏莫幻化成洪荒猛兽，三两下便将龙洌击败了！
<br/>　　而在现实之中，只见龙洌伫立在虚空的身形，在刚刚进入梦境之后，便不断的颤抖，口中不停的喷出鲜血。
<br/>　　“这……！”
<br/>　　所有看到这一幕的人，无不瞪大了眼眸，不可思议的望着这一切。
<br/>　　不足一个呼吸！
<br/>　　龙洌和屠霄交手，不足一个呼吸的时间，便口吐鲜血，这是什么情况！
<br/>　　啊！
<br/>　　就在此刻，只见龙洌惨叫一声，口中一股血箭狂喷而出，身形倒飞了出去。
<br/>　　龙洌的脸色瞬间惨白，身上气息大减，明显遭受了重创。
<br/>　　“怎么可能？”所有看到这一幕的人，除了司空炎和龙腾之外，均是心中狂震。
<br/>　　败了！
<br/>　　龙洌败的奇快无比！
<br/>　　败的没有任何悬念！
<br/>　　众人心中惊骇不已，龙洌在梦境之中，到底遇到了什么，怎么可能败的这么快！

<br/>　　而在梦境之中遭受重创，连带着他的真身，也受到了极大的创伤。
