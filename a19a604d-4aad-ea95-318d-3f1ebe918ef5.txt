　　而开启天帝宝库的方式有两种，第一种也就是最简单的一种，常年的等待，宝库会在数十年开启一次，而另外一种，就是参悟另外四卷天书。
<br/>　　参悟另外四卷天书，同样也能开启天帝宝库，但这种可能性微乎其微，宝库主人何等修为，移山填海般的存在，他留下的天书岂是凡物。
<br/>　　许多人终其一生，都难以参悟一卷天书，更何况四卷，例如青云门青叶祖师，天音寺祖先，如此惊才艳艳的强者，连其一都参悟不透，更何况四卷。
<br/>　　然而，慕清霄就是个变数，以他的天赋，不仅参悟了其余四卷天书，并且还知晓天帝宝库所在。
<br/>　　天帝宝库的主人，恐怕做梦都想不到，在这个渺小的大陆上，竟然有人能够参悟天书古卷中的至理。
<br/>　　回过神来，慕清霄不禁有些感慨，自己的运气可不是一般的好，五卷天书就让自己的修为，从元婴二重暴涨到元婴六重中期。
<br/>　　元婴境界，每提升一个台阶，就需要更多的时间，更多的资源，但他的提升简直超乎常理，由此可见，天书至理的重要性。
<br/>　　拍了拍白衫上的灰尘，慕清霄转过头来，就看到一双温柔的美眸，道：“我参悟天书用了多久？”
<br/>　　陆雪琪语气平淡，道：“半个月的时间。”
<br/>　　慕清霄一愣，在他的思维中，不过弹指一瞬间，却没想到已经过了半个月，走到她的身旁，轻轻握住她的手腕，道：“金丹六重，真是不错。”
<br/>　　得到丈夫的夸奖，陆雪琪有些娇羞，同时心中有些苦涩，道：“比不得你，我是借万载灵液来突破修为，而你靠的只是参悟。”
<br/>　　慕清霄捏了捏她嫩滑的脸颊，不再说话，走到木台前，取出容器，将木杯中的灵液封入容器中，陆雪琪刚突破没多久，她也知道轻重。
<br/>　　没有迷失在灵液带来的恐怖效果中，继续借由灵液突破。
<br/>　　她从筑基突破金丹，然后又接连突破六个小境界，接下来恐怕需要不少时间来稳固修为，灵液暂时是用不上了，但也不可能便宜了别人。
<br/>　　木杯中的灵液不少，但长连累月蕴养神药，已经所剩无几，虽然也有不少，却也只能让他突破一个小境界而已，所以他并不打算使用。
<br/>　　不如带回去，用灵液让紫嫣她们全部突破金丹修为。
<br/>　　收好容器，慕清霄在天帝宝库中扫视一眼，除了一个空荡荡的木杯，再也没有其他的东西，天书也全部到手，那自然是没有必要留在这昏暗的地方。
<br/>　　这时，陆雪琪忍不住问道：“我们要如何离开此地？”
<br/>　　慕清霄微微一笑，道：“我们进来的时候，石门上有强大的禁制，需要漫长的等待，禁制自动消失，或者天书方可开启，开启后禁制消失，现在就只是一扇普通的石门，只要是强劲的攻击都能劈开。”
<br/>　　说着，慕清霄手中已经出现一柄由灵气所凝聚的魂剑，手起剑落，寒光闪烁了几下，石门上就出现几道裂痕，紧接着就向外倾倒下去。
<br/>　　随着暗淡的光线射入宝库中，慕清霄并没有收起长剑，将陆雪琪护在身后，眼神逐渐变得凝重起来。
<br/>　　黑夜之中，清啸凤鸣声响起，只见一身橙黄色羽毛的黄鸟，伸展着双翼，堵在了石门口，尖喙锐目，脚爪锋锐巨大，紧紧盯着慕清霄。
<br/>　　陆雪琪一惊，失声道：“难道是传说中，守护天帝宝库的九天神鸟？”
<br/>　　慕清霄嘴角抽了抽，道：“这杂毛畜生可不是守护在这里，而是觊觎宝库中的神药而已，雪琪，你就暂时躲在宝库中，等我解决了它，我们就离开此处。”
<br/>　　面前的这只黄鸟，气息在元婴三重，比无情海中遇到的黑水玄蛇稍微强大一些，但此一时彼一时，慕清霄已经元婴六重中期，杀它就跟杀鸡似得。
<br/>　　就在他准备出手的时候，黄鸟并没有发动攻击，黝黑的瞳孔中闪过一丝人性化的困惑，察觉到慕清霄身上的恐怖气息后，双翅一展，急速消失在两人的视线中。
<br/>　　慕清霄手中的灵剑散去，嘴角忍不住抽了抽，道：“鸟类果然天生比爬行动物聪明，黑水玄蛇不知死活，但它察觉到危险后，转身就跑，居然没有一丝犹豫。”
<br/>　　站在身后的陆雪琪，明显松了口气，虽然知道自己丈夫修为强大，可依旧忍不住会担忧。
<br/>　　转过身来，慕清霄道：“我们在宝库中待了半个月，这半个月都由黄鸟在看门，门派修士不敢贸然前来，但现在黄鸟离去，他们肯定会一拥而上。”
<br/>　　“恐怕已经有许多门派闻声来到死亡沼泽，当初的惊天异象太过骇人，异宝的吸引力足以让修士疯狂，我们还是尽快离开，心情不错，我也不想动手杀人。”
<br/>　　闻言，陆雪琪微微点头，她的修为也刚突破不久，不宜在此刻动手，而且她也不喜欢麻烦。
<br/>　　...
<br/>　　...
<br/>　　如同慕清霄所料的一样，此时，死亡沼泽的外围，已经聚集了各门各派的修士，其中有青云门，焚香谷，天音寺，合欢派，以及众多不知名的门派。
<br/>　　夜幕深深，森林里冷冽的夜风，突然见似乎也大了起来，声声凄厉，前方喧哗声音越来越大，但黑暗却如不可逾越的高墙，挡在了张小凡和林惊羽的身前。
<br/>　　死亡沼泽诡异无比，各门各派弟子都心生警惕。
<br/>　　就在林惊羽渐渐沉不住气，准备冲出的时候，前方突然响起一声锐啸，瞬间虫鸣寂灭，黑暗深处人影闪动，两个身影同时飘了回来。
<br/>　　两人正是焚香谷的李洵，天音寺的法相，两人身上衣服都有撕扯破开的地方，李洵身上隐隐有一些血迹。
<br/>　　只见他们身形极快，张小凡与林惊羽急忙上前接应，却见他们二人脸上都有几分疲倦之色。
<br/>　　也不等林惊羽他们问话，李洵眉头一皱，低喝道：“小心，是万毒门！”
<br/>　　众人皆是一惊，却只听见周围森林黑暗中，虫鸣声大作，从四面八方围了过来，密密麻麻，将他们包围在中间，散发着幽幽光芒，在黑暗中窥视着他们。
