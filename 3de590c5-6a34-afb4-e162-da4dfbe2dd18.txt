<br/>　　这一局比赛荣耀白队这边处在蓝色方，ST战队则是处在紫色方。

<br/>　　尤其是在那种顶尖辅助的手中，这样的英雄发挥出的威力可就要更加的大了，真的被他们游走起来的话，那这一把他们就没的玩了。
<br/>　　“ban盲僧！”另一边金在贤也是没有犹豫，直接指挥matin将夜雪最擅长的盲僧给ban掉了，虽然他们这边的选手小花生玩的也不错，但是不要忘了白队可是在蓝色方，他们拥有着先选人的权利的。
<br/>　　更重要的是，就是连金在贤在私底下也是承认夜雪的盲僧确实是当之无愧的世界第一盲僧，小花生跟他比起来还是要差了一些的。
<br/>　　双方飞快的ban人，叶帝他们这边主要针对的还是对方的打野和下路，又ban了蜘蛛和圣枪游侠这两个英雄。
<br/>　　而对方则是针对了一下香蕉接下来两手都是ban的上路英雄，ban了一个奶妈、又ban了一个鳄鱼。
<br/>　　奶妈这个是白队三保二战术的核心，只要破了这个点，那么白队的那个战术就不能够使用了，这一点ST早就已经研究过了，所以这个英雄他们自然是不会放给香蕉的。
<br/>　　至于鳄鱼这个英雄那就更简单了，这本来就是一个非常强力的英雄，而且香蕉在鳄鱼的造诣又这么高，他们也怕香蕉真的会在上路打穿。
<br/>　　“选什么？”眼看着对方将最后一个ban人名额用掉，轮到自己这边选人，香蕉连忙开口问道。
<br/>　　“流浪。”叶帝这边犹豫了一下后，开口说道。
<br/>　　......
<br/>　　“Wow！白队这边首选流浪，不知道这个他们是会拿来打上单还是打中单，之前香蕉可是用这个英雄打过上单的。”看着香蕉的选人，米粒这边顿时大声喊了起来。
<br/>　　“这个大魔王他们应该也是在犹豫，毕竟流浪这个英雄中上都可以打的，他们接下来肯定是要看对方的选人来做分配的，来针对以下ST这边。”林雅楠跟着开口说道。
<br/>　　而ST战队的比赛室，此刻金在贤在看到香蕉这第一手选人后，他的眉头也是有些皱了起来，之前他已经想到过这个英雄了，不过ban位只有三个，他自然是想将对方威胁最大的三个点给ban掉，所以倒是空下来个流浪。
<br/>　　“拿女警和扇子妈！”犹豫了一下后，金在贤开口说道。
<br/>　　“ST这边选择了在先手选下女警和扇子妈这两个英雄，扇子妈应该是要去打下路的，虽然这个英雄同样可以打中路，不过明显女警和扇子妈在下路搭配起来更强势一些，这两个英雄在一起无论是消耗还是持续伤害，可是都要比其他的下路组合强的。”米粒开口分析着。
<br/>　　场上叶帝这边自然也是看到了对面的选人。
<br/>　　“他们这是不想拿出中上这两路的英雄啊，还想要留一手。”夜雪狂歌开口说道。
<br/>　　“那咱们也不拿，下一手他们肯定要在中上选出一个英雄来，咱们这一手也拿下路。”说着叶帝他们这边便是确定了寒冰和布隆这两个英雄。
<br/>　　“白队这一手针对很好啊，布隆这个英雄可以利用护盾来抵挡对方的消耗的，而寒冰到了六级之后也是可以发挥出很强的作用，当然了这个还是要看萝卜大招的准确度了。”看着白队的选人，林雅楠跟着说道。
<br/>　　ST比赛室，
<br/>　　“拿冰女和螳螂。”在思索了这一阵子后，金在贤终于是决定拿出他们研究的一个算是针对流浪的大招。
<br/>　　冰女这个英雄同样也是可以中上两路都打的，而且论在线上的能力他是要强过流浪的，到了后期虽然没有流浪那么强势，但是作用依旧很大，毕竟她可是有着w和r双控的。

<br/>　　不过这个英雄按照金在贤的想法，他其实更像在比赛进入到后半程的时候拿出来的，在第一场小组赛的时候就放大招，这个其实是有点得不偿失。

<br/>　　所以如果能够用这个英雄来获取胜利，让的华夏人民将无尽的怒火发泄到白队身上，金在贤还是很乐意看到的，虽然私下里大家关系很好，但是这毕竟是赛场。
<br/>　　只不过他少算了一点就是，叶帝在看到这个英雄出现的一刹那，他便是已经猜到了对方的想法，还有什么套路能够强过地球那么多赛季积攒下来的呢。
<br/>　　上单冰女这样的套路，在叶帝的记忆中可是出现过了无数次了。
