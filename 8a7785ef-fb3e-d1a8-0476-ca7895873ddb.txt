　　楚风眠身上凝聚的远古战龙精血越多，他施展出的龙族武技，便是越发强大。
<br/>　　等到楚风眠将十二万九千六百滴远古战龙精血全部凝聚出来的一刻。
<br/>　　楚风眠施展出的龙族武技，只怕就跟真正远古战龙施展而出的，将会没有任何一点的区别。
<br/>　　巨大无比的战龙之爪，从天而降，这荒古门三人，都是脸色剧变，疯狂催动无数力量，前来抵挡这战龙之爪的轰击。
<br/>　　“轰隆！”
<br/>　　一爪落下，他们三人拼劲全力，都是被震的血脉翻涌。
<br/>　　远古战龙的力量有多大？只怕是群山，都可以一爪轰碎。
<br/>　　现在的楚风眠虽然做不到这一步，但是他的战龙决，也已经有所小成，远远不是这三人，可以抗衡的力量。
<br/>　　“怎么会这样？这三人想要救荣北，却被此人直接打飞了出去？”
<br/>　　“以一敌三，居然还占据上风！这三人可不是寻常人啊，可是荒古门弟子啊，我的眼睛真的没有眼花？”
<br/>　　一尊尊北域武者，现在已经都有些无法相信自己双眼中所见到的场景了。
<br/>　　先是那王智，被楚风眠一掌轰杀，接下来荣北，也在楚风眠的剑锋之下，节节败退。
<br/>　　现在就连这荒古门三人联手，都被楚风眠的战龙之爪，直接拍飞出去。
<br/>　　他们甚至不少人都以为自己是在做梦，这一切真的只是一个神力境四重的武者，可以做到的吗？
<br/>　　让一个生死境四重的武者前来，都做不到这一切吧。
<br/>　　“以我们三人的实力，居然都被直接拍飞，此人的力量怎么会如此强横？”
<br/>　　其中一尊荒古门弟子，被一掌拍飞出去，他浑身一震，脸色也满是震惊。
<br/>　　他们可都是荒古门弟子，来到北域，根本是没有将北域的任何人放在眼中。
<br/>　　在他们看来，在北域中就算是那些声名远扬的天才，也不过都是些寻常人，甚至是废物，蝼蚁。
<br/>　　若是说之前王智，荣北的失败，还有着巧合的可能性。
<br/>　　可如今他们三人联手，居然都被楚风眠直接拍飞了出去，这已经不是巧合可以说明的了。
<br/>　　令一尊荒古门弟子大声喊道。
<br/>　　“无论如何，也要救下荣北，王智已经死了，荣北在死，我们都要受到门派的制裁！”
<br/>　　荒古门弟子，每一位都是精英，死了一个拥有特殊血脉的王智，已经是一件不小的事了，若是荣北再死，才是大事。
<br/>　　他们三人对视一眼，同时点了，同时一股滂湃气势，猛然爆发出来。
<br/>　　“荒古大道！”
<br/>　　在这磅礴气势之下，更是一条巨大无比的大道，凝聚而出，在这大道面前，似乎是天地一切，都要被镇压。
<br/>　　“小子，现在给你一个机会，放了荣北，我们可以饶你不死，只是将你镇压，带到门派中去审判你！”
<br/>　　一尊荒古门弟子，站在这荒古大道上，高高在上，看着楚风眠俯视说道。
<br/>　　“这是你最后的机会，别再执迷不悟了，就算是你的实力在强，敢跟我们七大宗门为敌，也只有死路一条。”
<br/>　　“你们算是什么东西？七大宗门又算是什么东西！只有死路一条？今日楚某倒是要看看，到底是谁要死！”
<br/>　　楚风眠眼神中的杀意更浓。
<br/>　　敢拿七大宗门来威胁楚风眠，简直是触碰到到了楚风眠心中的逆鳞。
<br/>　　他对七大宗门，本身便是恨之入骨，如今这三人来拿七大宗门来压楚风眠，根本就是来触碰楚风眠心中的逆鳞。
<br/>　　“统统给我死！今日你们谁也别想活！”
<br/>　　楚风眠长啸一声，他心神一动，剑锋的威力更甚，直接席卷这无数力量，压迫这荣北最后一点空间。
<br/>　　“啊！我是七大宗门的人！我是荒古门弟子，谁敢杀我，荒古门定会永远灭杀了你！”
<br/>　　荣北看着楚风眠，大声咆哮道。
<br/>　　但是他的威胁，只能是让他死的更快。
<br/>　　“死！”
<br/>　　楚风眠冷笑一声。
<br/>　　楚风眠手中剑锋，一点点的落下，这荣北的最后一点躲避空间，都全部消失。
<br/>　　他的身躯，节节破碎，在这密室之中，所留下的，都只有一声声的惨叫，在不断回想。
<br/>　　“杀了！”
<br/>　　“真的杀了！”
<br/>　　这荣北的惨叫声，也在在场无数众人，一个个看向楚风眠，都如同看向一个真正的疯子一样。
<br/>　　当众斩杀两位荒古门弟子，能够做出这一切的人，只有疯子。
<br/>　　不就算是一个疯子，在听到七大宗门，荒古门的时候，只怕都会犹豫，不敢出手。
<br/>　　就算是地魔门的魔头，在听到七大宗门名字的一刻，也只能是安静奉承，奉为上宾。
<br/>　　七大宗门这等庞然大物，扎根九域，数万年的时间，到底有着多么庞大的势力，只怕是任何人都无法撼动。
<br/>　　任何势力，任何强者，就算是独来独往的许多强者，都要给七大宗门一个面子。
<br/>　　可现在，在这荒古门三人的威胁之下，楚风眠依然是毫不犹豫，直接斩杀了荣北。
<br/>　　这举动，便是从来没有将七大宗门，没有将荒古门，放在眼中过。
<br/>　　狂妄，嚣张，已经是不能形容现在的楚风眠。
<br/>　　只有两个字，可以形容楚风眠，就是疯狂，像是一个疯子一般的疯狂。
<br/>　　“该死！”
<br/>　　荒古门三人，听着荣北的惨叫声，脸上的愤怒，简直是要彻底爆发而出。
<br/>　　“小畜生，你做的很好，敢杀我们荒古门的弟子，蔑视我们荒古门的威严，不将我们荒古门放在眼中。”
<br/>　　“小畜生，你就等着我们荒古门无穷无尽的追杀吧！不，我们七大宗门，同仇敌忾！你要等这的，是我们七大宗门无穷无尽的追杀！”
<br/>　　一尊荒古门弟子，看着楚风眠怒声喝道。
<br/>　　“就算你是北邙学院的弟子又如何？北邙学院也护不住你！无论你是逃到天涯海角，就算是逃到任何地方去，你都要死！”
<br/>　　“我们走！”
<br/>　　说着，那荒古门弟子，便是转身，拂袖而去，准备离开。
<br/>　　“等等，楚某有让你们离开吗？”
<br/>　　一声冰冷的声音，突然响起。
<br/>　　只见楚风眠大手一挥，无数屏障，只见是将这密室空间完全封锁住了。
