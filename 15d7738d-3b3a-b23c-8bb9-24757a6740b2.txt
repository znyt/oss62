　　花甲老者的喝声，如同天地雷音，滚滚不休，震动了整个青俞皇城。
<br/>　　无数的人骇然不已，无数的人望着皇宫上空，那一群北海冰宫的强者。
<br/>　　北海冰宫绝世天才，万尊榜第六十二名的汪晟死于皇宫，此事早已传遍了皇城。
<br/>　　很多人知道北海冰宫的强者会来，也有很多人在等着看好戏。
<br/>　　花甲老者身上的气势极为恐怖，仅是其身上逸散的气息流落而下，都仿若九幽寒流，冰封天地。
<br/>　　很多皇城的低级武者，根本抵挡不住，有些离得较近的武者，刹那间便身体僵硬，便成了一块寒冰。
<br/>　　“天啊！是北海冰宫三长老亲临！”
<br/>　　“还有八长老，十五长老，大弟子慕冰雪，居然来了这么多强者！”
<br/>　　“皇室要完了，那名杀死汪晟的人也要完了！”
<br/>　　无数的人惊骇莫名，北海冰宫的三长老，那可是武皇境九重的超级强者。
<br/>　　这等修为的强者一怒，将尸横遍野，血流成河。
<br/>　　很多皇城的武者，立刻远离了皇宫，飞到高空之上，远远的观看。
<br/>　　金銮大殿之中，苏莫手抓屈覃国师，面色冷然，古井无波。
<br/>　　“你不用隐瞒了，告诉我一切，你才有活命的机会！”苏莫淡淡的向屈覃说道。
<br/>　　屈覃的脸色有些发白，不过他并未说话，因为现在的情况，他根本无需回答了。
<br/>　　青谷和青雅公主等皇室的强者，个个满脸愕然之色，北海冰宫的强者来临了。
<br/>　　而这位无名青年，居然还有心情逼问屈覃！
<br/>　　难道他不知道现在的情况？
<br/>　　难道他不知自己有多么的危险？
<br/>　　高空中，那名花甲老者，也就是北海冰宫三长老，见苏莫没有理会他，心中顿时大怒，眸中射出冰冷的杀机。
<br/>　　“简直找死，敢杀老夫弟子，无论你是什么人、什么身份，今日.你都难逃一死！”三长老冷喝一声，随即他的身形从高空中快速降落。
<br/>　　这时，伫身于大殿中的苏莫，缓缓的转过了脑袋，眸光望向了三长老。
<br/>　　“你，在和我说话吗？”
<br/>　　苏莫声音淡淡，但他的声音听在三长老的耳中，却无异于九天神雷，轰然炸响。
<br/>　　正在降落的三长老，身形猛然一震，立刻停在了半空中。
<br/>　　他的目光和苏莫对视，立刻双眸大睁，只感觉脑中狂震不已。
<br/>　　这是一双恐怖的眼眸，深邃、浩瀚、冰冷、无情、无敌、霸道。
<br/>　　“你……！”三长老顿时神情一滞，一时间有些呆愣。
<br/>　　“回答我！”苏莫再次开口，一声冷喝震动苍穹。
<br/>　　这道喝声霸道无情，冷漠无比，蕴含让人不容拒绝之意。
<br/>　　三长老神情怔怔，被苏莫的喝问，震的脑中有些短路，一时间沉默无声。
<br/>　　所有人都愣住了，目光呆滞，神情愕然。
<br/>　　青谷等青俞皇室的强者，简直不敢相信自己看到的一切、听到的一切。
<br/>　　这位无名青年，居然如此的霸气、如此的强势！
<br/>　　面对北海冰宫三长老这等超级强者，居然像是长辈在命令晚辈一般。
<br/>　　这是怎么回事？
<br/>　　此子疯了吗？
<br/>　　还是其自认为三长老不敢杀他？
<br/>　　青雅公主小嘴微张，水灵灵的大眼睛中，充满了震惊。
<br/>　　这位强大的过分的青年，居然如此强势的喝问北海冰宫三长老！
<br/>　　她简直被吓傻了，脑中一片空白。
<br/>　　高空中，跟随三长老而来的十几名北海冰宫的强者，都是一脸的懵逼，完全没想到会发生这种情况。
<br/>　　这十几名强者中，有一名身穿翡翠色长裙的女子，容颜绝美，姿色倾城。
<br/>　　此女，正是北海冰宫大弟子，万尊榜排名第二的慕冰雪。
<br/>　　慕冰雪的目光，紧紧的凝视着苏莫，她的柳眉微微皱起。
<br/>　　眼前的这名青年，给她一种很是熟悉的感觉，她曾经见过此人。
<br/>　　但一时之间，她却是想不起来，此人到底是谁？
<br/>　　整个皇城中都是寂静了下来，在苏莫的一声冷喝中，变得一片死寂。
<br/>　　“我……你……你到底是什么人？”
<br/>　　少倾之后，三长老反应了过来，顿时脸色难看，面带惊疑的问道。
<br/>　　他的心中震惊不已，眼前的青年居然给他一种无比危险的感觉，让他毛骨悚然。
<br/>　　要知道，就算是北海冰宫的宫主，准帝强者北海女皇，都不能让他生出这种感觉。
<br/>　　所以，三长老的心中充满了惊疑。
<br/>　　仔细的打量了一番苏莫的修为，武皇境四重巅峰！
<br/>　　这让三长老顿时脸色铁青，一个武皇境四重的蝼蚁，居然如此装腔作势！恐吓于他！
<br/>　　“你的回答，只有三息的时间！”苏莫没有理会三长老的询问，淡淡的说道。
<br/>　　苏莫心中也是有些温怒，这些人来的真是时候，不仅打扰他逼问屈覃，居然还敢如此强势！
<br/>　　当然，他也没有直接出手，毕竟这些人为门下弟子寻仇，也在情理之中。
<br/>　　不过，对方居然敢辱骂他，真是好胆色！
<br/>　　“装腔作势，简直不知死活！”三长老眸中杀机一闪，他堂堂武皇境九重强者，岂会被苏莫吓住！
<br/>　　体内寒冰玄力涌动，三长老手掌一抬，便欲强势镇杀苏莫。
<br/>　　“三长老快住手！”
<br/>　　就在此刻，蓦地，一声娇喝声陡然响起，只见高空上的慕冰雪，脸色猛然大变，身形一闪，便急速冲了下来。
<br/>　　三长老闻言一怔，但他还是暂时停止了出手。
<br/>　　“冰雪，怎么了？”转头望向飞到身边的慕冰雪，三长老皱眉问道。
<br/>　　其他北海冰宫的人，亦是心中不解，不明白慕冰雪是何意，为何不让三长老出手？
<br/>　　慕冰雪伫立在三长老身侧，面带惊疑的凝视着苏莫，随即她心中暗叹一声，对三长老道：“三长老，快道歉！”
<br/>　　“什么？”三长老闻言，顿时面色一滞，一脸的懵逼。
<br/>　　其他北海冰宫的强者、皇室的强者，包括皇宫之外高空中无数的观看之人，均是满脸愕然，不明所以。
<br/>　　慕冰雪居然让三长老道歉？
<br/>　　向那位无名青年道歉！
<br/>　　这是怎么回事？
<br/>　　难道慕冰雪认识此人，但就算如此，也不用道歉吧！
<br/>　　三长老是什么身份，北海冰宫的三长老，武皇境九重的超级强者。
<br/>　　这位无名青年，就算身份再不简单，也没有资格让三长老道歉吧！
<br/>　　可以这么说，整个苍穹世界，除非是那些超级势力的宗主，准帝级别的巅峰强者，才有资格让三长老道歉。
<br/>　　但面前的青年，不过二十出头，会是准帝强者？
<br/>　　这显然不可能！
