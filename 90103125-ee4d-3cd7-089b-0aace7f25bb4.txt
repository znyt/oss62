　　从京都到云州，一共是两天半的路程，
<br/>　　而在这两天半的时间里，叶尘待在林婉容的怀里，看似一直在熟睡，其实却在暗暗吸收灵气，强化着身体的经脉。
<br/>　　他现在身体太弱了，体内没有丝毫的真元，即便他已经参悟了四十八条法则之力，也根本无法发挥其作用，甚至现在一个孩童，都足以轻松打败他。
<br/>　　好在经过这两日的修炼，他体内脆弱不堪的经脉，已经强韧了十多倍，丝毫不弱于一个成年人，体型也大了将近一圈，但是肉身依旧脆弱。
<br/>　　一开始，林婉容察觉到叶尘，惊人的成长速度之后，着实吓了一大跳，反复检查了几次，发现叶尘一切正常，这才大大松了一口长气。
<br/>　　不过，最让叶尘感到尴尬的，还是莫过于喂奶的时候。
<br/>　　他现在还是个只有一月不到的婴儿，自然是要吃母乳的，虽然林婉容是自己的亲生母亲，这是天经地义的事情，却也让心智年龄已经超过千岁的叶尘，感觉到无比的蛋疼。
<br/>　　一开始，林婉容每次喂奶的时候，叶尘就将脑袋扭到一旁，坚决不吃。
<br/>　　可是后来，实在是饿得不行，也只好无比郁闷的妥协了。
<br/>　　等到火车抵达云州的时候，叶尘比两天前，已经明显健壮了许多，个头也高出了好几公分。
<br/>　　林婉容虽然也察觉到，自己的这个孩子，似乎有点非同寻常，甚至有些匪夷所思，可是见他脸色红润，中气十足，丝毫没有任何生病的征兆，最终也只好将心头的好奇心按耐了下去，打算以后有机会，给孩子去医院做一个检查。
<br/>　　而叶尘，待在林婉容的怀里，两耳不闻窗外事，已经完全进入修炼状态，暗暗运转神功，丹田之内，已经开始渐渐凝聚出了一丝真元。
<br/>　　他现在的当务之急，是要快点强大起来，才能保护自己和母亲的安全。
<br/>　　记得前世，在叶尘九岁那年，母亲便因为重病难医病逝了，在这个世界里，他绝对不能允许这种事情再发生。
<br/>　　而就在叶尘全心投入在修炼之中之时，突然一道暴喝声，将他从入静的状态中惊醒。
<br/>　　“丢人现眼的东西！你知不知道，你得罪了秦家的小姐，会给我们林家带来什么样的灾难？我林世骁怎么生出你这么没用的女儿！”
<br/>　　叶尘睁开眼，朝周围放眼一看，发现有些熟悉，原来林婉容已经带着他，回到了林家。
<br/>　　此时林家家主林世骁，也即是叶尘的外公，正满脸愤怒之色，指着林婉容一通大骂。
<br/>　　旁边林刚、林海和林牧三兄弟，以及林家的一些核心人物，也全部在场，一个个都满脸凝重之色。
<br/>　　林世骁的话才刚刚说完，一旁的林刚也跟着包含怒气道：
<br/>　　“小妹，你知不知道那秦家，在我们华夏国是何等崇高的地位？你如今得罪了秦家小姐，又被叶家赶出了京都，你觉得秦家会放过你，会放过我们林家吗？”
<br/>　　林海跟着嘟囔道：
<br/>　　“就是！原本还指望你能攀上叶家，没想到你不但被赶了回来，还带着这么一个拖油瓶，你可真够行的！”
<br/>　　一时间，林家众人纷纷对林婉容责骂起来，语气越来越难听，甚至有人还提出，为了避免林家被秦家报复，应该将眼前林婉容逐出林家！
<br/>　　众人之中，唯独叶尘的小舅林牧，满脸的不忍之色，为林婉容辩解了几句，可惜他人微言轻，根本起不到任何作用。
<br/>　　林婉容见到众人这般态度，则面如死灰，眼眸中透着一抹浓浓的绝望之色。
<br/>　　她本以为回到家，就可以让她原本已经千疮百孔的心灵，得到些许的慰藉，却怎么也没有想到，自己的家人，居然是这幅嘴脸，甚至连自己的父亲和亲哥哥，也全都容不得她！
<br/>　　叶尘看到众人如此羞辱自己和母亲，如果不是他现在还是一副婴儿的躯体，真恨不得将这些人全部暴打一顿。
<br/>　　而就在众人的骂声中，林婉容忽然冲着林世骁，缓缓跪倒了下去，绝美的脸庞上，露出一抹惨然笑容，
<br/>　　“父亲！请允许我，最后再这么称呼你一次！”
<br/>　　林世骁听到这话，顿时两眼一瞪，
<br/>　　“孽障！你这话是什么意思！”
<br/>　　林婉容缓缓道：
<br/>　　“女儿自知犯下大错，也不敢乞求父亲原谅，自今日之后，你我父女恩断义绝，我和林家再无任何瓜葛！”
<br/>　　“你！”
<br/>　　林世骁气得的身体乱颤，
<br/>　　说完这话，林婉容冲着林世骁恭恭敬敬磕了三个响头，然后站起身来，转身便朝外面走去。
<br/>　　“姐！你别走！”
<br/>　　林牧大喊了一声，便要追上来阻挡。
<br/>　　不料，林世骁一声暴喝，
<br/>　　“不许追！让她走！这个不孝女，从今往后就不再是林家的人，谁也不许替她求情！”
<br/>　　……
<br/>　　从林家出来，雪依旧在下。
<br/>　　叶尘能明显感觉到，母亲的身体冰冷无比，而且一直都在微微颤动。
<br/>　　虽然之前，他曾不止一次想象过，母亲被人从京都赶出来的场景，可是真当他如今亲眼目睹了这一切，才知道母亲当年，承受了多么大的痛苦！
<br/>　　先是被情敌逼着离开京都，然后带着襁褓中的孩子，好不容易回到家中，却又被自己的家人责骂、厌弃。
<br/>　　估计如果不是因为还有孩子要照顾，恐怕她都难以支撑活下去了吧？
<br/>　　叶尘眼眶微红，挣扎着从林婉容的怀里爬起，张开细小的双臂，轻轻搂住了林婉容的脖子，将自己的小脸，紧紧凑到了母亲的脸颊上。
<br/>　　林婉容明显为之一愣，继而她那原本已经绝望的眼眸之中，猛然又重新燃起了一丝火焰，将叶尘紧紧搂在怀里，喃喃道：
<br/>　　“好孩子，你这是在心疼妈妈吗？你放心，就算是为了你，妈妈也一定会好好的活下去的！”
<br/>　　林婉容抱着叶尘，冒着风雪，走了半个多小时，在一栋好算不错的别墅前停了下来。
<br/>　　门铃按响，过了片刻之后，别墅的大门被打开，
<br/>　　这是一名约莫二十六七岁的美貌女子，身后还跟着一名约莫三、四岁的小女孩，
<br/>　　看到这两个人，叶尘顿时两眼一亮，
<br/>　　“岚姨！这小女孩，难道是蔓姐？”
<br/>(本章完)
