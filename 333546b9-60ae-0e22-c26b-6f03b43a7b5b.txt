<br/>　　无数齐林山弟子，都是疯狂的怒骂这林书古帝。

<br/>　　这一名长老有些不忍的说道。
<br/>　　他的许多弟子，都是死在了这血祭之中，这林书古帝做事的疯狂，让他都有些来不及阻止。
<br/>　　“后继无人？怕什么！今日只要是能杀了这小子，从此之后，大罗天星域都会归我们齐林山掌控，死几个弟子怕什么，几百年时间就可以恢复了。”
<br/>　　那林书古帝冰冷的开口道，血祭数十万弟子，他的脸色都没有任何的变化，彻底的疯魔了。

<br/>　　“可是……”
<br/>　　一些长老，还是有些不忍。
<br/>　　“做大事者，不拘小节，这些弟子受到我们齐林山的恩惠，就应该想到这一天，你们既然是不忍，那就跟他们一起死吧！正好化为护山大阵的力量！”
<br/>　　那林书古帝，大手一挥，突然那几名出言反对他的齐林山长老轰击而去。
<br/>　　砰！
<br/>　　这几名齐林山长老，都想不到林书古帝居然是突然出手，他们都没有来得及抵挡，便是都死在了这林书古帝的手心之中。
<br/>　　他们的精血，也被吸入到了祭坛之中去，化为了这护山大阵的力量。
<br/>　　数十万齐林山弟子的精血，凝聚在了一起，令这每一道阵眼的力量都强横了数十倍的地步。
<br/>　　虽然说这齐林大陆之上的祭坛，已经是被楚风眠摧毁了三分之一了，可现在这林书古帝的实力，还是强横了无数倍。

<br/>　　一个大大的齐字，突然从天凝聚，融入到了这林书古帝的身躯之中，林书古帝身上的气息，突然变换着。
<br/>　　如今随着这一个齐字融入身躯之中，林书古帝身上的气势已经是跟之前截然不同，一种无边的霸气，散发而出。
<br/>　　整个人，都如同换了一个人一样。

<br/>　　楚风眠目光看向这林书古帝，他察觉到，这血祭之力，似乎是唤醒了一些东西，附身在了林书古帝的身上。
<br/>　　“血祭苍生，使我苏醒，齐林山居然被逼到这个地步了吗？”
<br/>　　这陌生的林书古帝，发出了古老的声音，似乎是一尊古老的存在，突然苏醒了。
<br/>　　这林书古帝的目光，突然看向了楚风眠的方向，露出了无边深邃的眼神。

<br/>　　林书古帝盯着楚风眠，突然大笑起来道。
<br/>　　“天下间，居然真的有这样的存在，好，好一个天地异种，吞噬他，我的实力都可以再进一步！甚至独立重生！”
<br/>　　这林书古帝看向楚风眠的一刻，突然出手了，施展的依然是齐林山的武技，可是要比起那林书古帝施展出的齐林山武技，要熟练的多。
<br/>　　这齐林山武技，仿佛便是如同就是他所创造的一般。
<br/>　　“这是什么人物？这林书古帝被什么东西附身了？”
<br/>　　青梦在一旁也是观察到了这一点，惊声说道。
<br/>　　“齐霸神拳！这是齐林山的创始人才会的招数，据说已经是在齐林山之中失传了，此人怎么可能会？”
<br/>　　看到这林书古帝，突然凝聚灵力在拳风之上，一拳轰击过来，青梦震惊无比道。
