　　通过这玉符之中的记载，楚风眠也大约的了解到。
<br/>　　这佛寺，便是曾经九域之中赫赫有名的佛门宗门，净土宗的宗门遗迹。
<br/>　　也是曾经这净土宗的老巢。
<br/>　　楚风眠是第一个踏入这佛寺之中的，不过随着楚风眠冲了进来，这少帝跟五行刀圣，也放弃了争执，冲入到了佛寺之中。
<br/>　　随即便是各方势力，涌入到了这佛寺之中，圣龙秘境的四宗，五行天，齐林山，沧海都，甚至是净土星域，阎罗星域等等的势力。
<br/>　　也都尽数进入到了这佛寺之中。
<br/>　　这一点也算是在楚风眠的意料之内，因为之前在这佛寺还未开启的时候。
<br/>　　在外面，就已经是聚集了无数的势力，大量的强者。
<br/>　　他们都在等待着佛寺开启的一刻，现在佛寺开启，自然是会全部进入其中来。
<br/>　　而且随着佛寺开启的动静，只怕是整个赤红大陆，甚至是归墟天的强者，都在往这佛寺赶来，前来争夺一番。
<br/>　　不过现在佛寺之中虽然开启，但是各方势力还在探索之中，所以并没有起太大的冲突。
<br/>　　如今楚风眠所在这一处山谷所处的地方，是一片藏地，属于曾经净土宗的藏地。
<br/>　　传闻之中，无数净土宗历代之中的强者，皆是被埋葬在了这藏地之中。
<br/>　　净土宗向来对强者死后，给予厚葬，所以在这藏地之中，不知道是埋藏了多少的宝物。
<br/>　　这一处藏地，位于佛寺最为中心的位置，无论是从佛寺之中如何行走，走到尽头，便是都会来到这一处藏地所在。
<br/>　　像是楚风眠跟这一群五灵宗圣子前来的方向，并不一样，但是最终却都是来到了这一处山谷的附近。
<br/>　　这些五灵宗圣子，便是就地驻扎了下来，搭建营地，去探索周围的宝藏。
<br/>　　那小炎帝，已经是在这一处藏地之中，得到过其中两处宝藏了。
<br/>　　这两处宝藏之中的宝物，都是被小炎帝一人自己独吞了下去。
<br/>　　可惜小炎帝都没有来得及将这两处宝藏之中的宝物炼化收取，便是就被楚风眠所轰杀。
<br/>　　“这就是万念佛珠？”
<br/>　　楚风眠打开小炎帝的空戒，从其中拿出来一串翠绿色的佛珠来。
<br/>　　之前在楚风眠轰杀了小炎帝，打开小炎帝空戒之中的一刻，就注意到了这一串佛珠的存在。
<br/>　　只是没想到，居然是赫赫有名的万念佛珠。
<br/>　　万念佛尊，也算是净土宗之中一位名气极大的强者了，他算是净土宗末代衰败之后，少有是一尊强者了。
<br/>　　一身实力，达到半步天人境的巅峰，几乎是当时天人境之下的第一人。
<br/>　　所以是在古籍之中，对着万念佛尊的记载并不少。
<br/>　　在古籍的记载之中，万念佛尊最强大的灵器，便是他手中的一串佛珠，这一串万念佛珠，一共十二枚佛珠组成。
<br/>　　这十二枚佛珠，都是世世代代受到佛门贡祭，蕴含精粹的佛门之力。
<br/>　　一旦是注入灵力，催动起来，足以是让一名佛门强者的实力，提升数成。
<br/>　　不过楚风眠并非是佛门武者，这一点对他倒是没有什么用处，不过这其中的佛门之力，却是足以帮助楚风眠来抵挡佛门禁制的威力。
<br/>　　楚风眠拿出这万念佛珠，待在手腕上，再度催动灵识。
<br/>　　突然只见周围的迷雾，被驱散了不少，楚风眠灵识所受到的限制，减弱了几分。
<br/>　　起码方圆三十里左右的区域，现在楚风眠的灵识，都足以探察的到了。
<br/>　　这万念佛珠的效果，无比惊人。
<br/>　　“果然，这万念佛珠对于佛门禁制，有着一定的克制性，在外对我没有什么用处，但是在这佛寺之中，绝对是人人想要得到的至宝。”
<br/>　　楚风眠感受到万念佛珠效果，心中不由的喜悦起来。
<br/>　　在这佛寺之中，楚风眠最为担心的，便是遭遇偷袭。
<br/>　　楚风眠的灵识，在这佛寺之中，受到禁制的力量所压制，只能够探察方圆十里。
<br/>　　十里的距离，对九劫古帝，甚至是半步天人境来说，刹那之间就可以到来的距离。
<br/>　　一旦是楚风眠被人偷袭，只怕是会危机无比，像是那五行刀圣，再度偷袭楚风眠，在这佛寺之中楚风眠就连反应过来的机会都没有。
<br/>　　现在有了这万念佛珠的加持，探察周围三十里，楚风眠也足以放心许多了。
<br/>　　而且在这佛寺之中，在这一片藏地之中，其中开启的许多宝藏，其中都有着大量的佛门禁制。
<br/>　　这些佛门禁制，许多都足以用万念佛珠来化解。
<br/>　　这小炎帝，看来还真的是送了楚风眠一份大礼。
<br/>　　“楚兄。”
<br/>　　就在楚风眠继续看向这佛寺之中，其他的几条情报的时候，炎撼世突然走了过来。
<br/>　　“什么事？”
<br/>　　楚风眠看到炎撼世走来，也收起了玉符，关于这佛寺，这一片藏地，楚风眠已经是有着几分了解了。
<br/>　　当然纵然是五灵宗，对着藏地之中的了解，依然不足百分之一，这藏地的神秘，还没有人可以探察清楚。
<br/>　　“楚兄，圣院宗的圣子领袖之一的寒夜，突然发来消息，说要邀请我五灵宗跟去圣院宗一起，联手开启一处宝藏。”
<br/>　　炎撼世看着楚风眠恭敬的说道。
<br/>　　“联手开启宝藏？圣院宗，寒夜？圣院宗的圣子首领，不是少宗主无道吗？”
<br/>　　楚风眠听到寒夜这个名字，顿时感觉到几分陌生，不由的看向炎撼世询问起来。
<br/>　　“楚兄，是这样的，圣院宗内部，也并非是贴吧一片，。无道虽然是少宗主，但是并非是所有圣院宗的圣子，都愿意跟随无道。”
<br/>　　“这寒夜便是他们其中的佼佼者，据说这寒夜，曾经得到过大机缘，得到过古老一尊强者，耀王的传承，也比称为小耀王。”
<br/>　　“不过此人，并没有得到力量传承，只是得到了耀王留下的功法，前途无量，所以此人不甘于在无道之下。”
<br/>　　炎撼世对着楚风眠详细的解释道。
<br/>　　圣龙秘境，四宗八堂之中，都无比复杂，没有任何一宗一堂，是铁板一片。
