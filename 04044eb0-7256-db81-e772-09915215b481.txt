　　百年之后，元始睁开眼睛，他的眼中闪过一丝精光，宛若星辰，明亮无比。
<br/>　　在他的眼眸深处，好似能够看到宇宙开辟，生死轮回，无尽生灵，幻灭重生，天地崩裂，演化混沌，混沌炸裂，世界开辟等各种惊人的恐怖异象。
<br/>　　良久，那些异象才缓缓消失，元始轻轻呼出一口气，那气如同一道白色匹练，冲出很远，飘荡在空间中，良久才消散开来。
<br/>　　“吾之推演已成功，接下来，便要看能不能实行了。”
<br/>　　元始缓缓站立起来，抬头望着虚空，开口说道。
<br/>　　元始一步踏出，走出昆仑山地界，举步凌空，一步步踏上天空，一步步地走向天外。
<br/>　　无尽道韵流转，道则震动，大道的气息在弥漫，与道韵相互交织，彼此交融。
<br/>　　一道道霞光在他周身绽放，祥云万朵，瑞气千万，无尽紫气浩荡而来，如同铺天盖地一般，遮蔽大半个洪荒。
<br/>　　每一步踏出，他脚下都会有紫光凝结，如同一朵朵紫色的莲花在虚空绽放。
<br/>　　紫气纵横，步步生莲。
<br/>　　飘然若仙帝之姿，有霞光万道之象，一如主宰临尘。
<br/>　　“轰隆隆！”
<br/>　　天地在巨震，好似在畏惧，在颤抖，元始出手，大道都在震动，整个世界变得不稳定了，随时可能要炸开。
<br/>　　元始的气息不再隐瞒，泄露了出来，无尽的气机洪荒世界之中横扫，蔓延到世界各地，天地在颤栗，种种异象环绕，仿佛都在恭迎一位无上存在的觉醒。
<br/>　　整个洪荒世界，无尽生灵都感应到一种至强的压迫了，自身的肉身都难以承受了，颤抖着向元始所在的方向跪拜下来。
<br/>　　“天啊，这是怎么回事！”
<br/>　　“这种威压，比起圣人来还要恐怖啊？难道天尊要灭世吗？”
<br/>　　“元始圣主，他到底要干什么？”
<br/>　　“……”
<br/>　　洪荒大地上的生灵都感到灵魂在颤栗，洪荒之中，除了圣人之外，其他的生物，不管是心甘情愿，还是不愿意，都在此刻跪拜了下去。
<br/>　　“嘶！元始竟如此恐怖，只怕是圣人后期了，嘶！”
<br/>　　“元始，你到底在干什么？竟有如此威压？”
<br/>　　“难道是有人招惹到了元始？”
<br/>　　几个圣人大惊，感受着元始身上那股恐怖的圣威，直让他们眼皮狂跳，觉得不可思议。
<br/>　　虚空之上的元始对此毫无感觉，他抬头望向虚空，眼中射出两道恐怖的光芒射出，洞穿周天，与之前的温和淡然截然不同了，此刻的他，像是从沉睡之中醒来，不再安安静静，而是化成一尊魔神。
<br/>　　“以元始大道为基，鸿蒙灵魂为心，梦幻虚实为法，混沌物质为料，太初能量为源，万般大道为火，创世天地，一念开天！”
<br/>　　元始双手交叠，缓缓抬起，而后在虚空之中划动，施他展盖世手段，引动天地大道纹路，交织着恐怖的力量，席卷整个洪荒大地。
<br/>　　万道为火，天地为炉，锻造元始大道，引发各种大道共鸣，各种秩序神链交织。元始在虚空之中刻画各种神秘符文，他的速度很快，天地间顿时便布满各种符文，那是复杂莫名的道痕，是元始所创世界的诸般大道，令它更加贴近这浩瀚的世界本源。
<br/>　　大道本源原本就是一方宇宙的一种本源重宝，与天地万道有着玄妙的联系，元始所创世界的重点，便是要借着这种联系，令世界借此扩张，成为天地的一部分，直接蔓延到整个宇宙。
<br/>　　“轰隆隆！”
<br/>　　“一元！”
<br/>　　元始一指点出，一道无比闪亮的光点从他手指间点亮，紧接着，亮点一分为二，二分为四，四分为八，八分十六……
<br/>　　一瞬间，无量光辉，照耀大千，点亮大道之光，映照诸天万界，道则闪动，一道朦朦胧胧，虚虚实实，宛如梦幻般的神秘空间出现在虚空之中，它绽放出耀眼的光芒，无尽迷雾笼罩，让人看不清虚实。
<br/>　　“两仪！”
<br/>　　元始一挥大手，整个世界顿时变成黑白两色，白色在上面，宛若天，黑色在下方，宛若地。黑白涌动，两色交织，缓缓旋转，到最后越来越快，宛若两条鱼，在追逐打闹。不知过了多久，黑白两色缓缓分开，此刻黑中有白，白中有黑，相互交织，衍化天地。
<br/>　　“三才！”
<br/>　　日月星开始演化，天地人开始出现，草木虫鱼，山川河流，世界开始演变，但如今的世界，还是没有显化出来，依旧只是一种朦朦胧胧的现象，让人看之不清，摸之不透。
<br/>　　“四象！”
<br/>　　地水火风涌动，翻滚不息，天地大道在轰鸣，数不尽的大道碎片飞出，烙印在天地之中，有无数神光涌动，一道炽盛的光团，闪烁在天地之间……
<br/>　　龙凤和鸣，白虎跃天，玄武出海，神鸟翱翔于九天，人形生物举霞飞升……霞光一片片，瑞气一道道，完全淹没了天地，簇拥着那道神秘的空间。
<br/>　　“五行！”
<br/>　　白、青、黑、赤、黄五道耀眼的光芒旋转着飞了出来，金、木、水、火、土出现，相生相克，演化天地万物。巨木遮天，大河涛涛，神焰焚天，大地震动，金气纵横，没入天地之间……万物开始演化，天地初成。
<br/>　　“六合！”
<br/>　　“七星！”
<br/>　　“八卦！”
<br/>　　紧接着，又有数道不同颜色的光彩出现在天地间，虚空在颤栗，大道在轰鸣，虚空深处那神秘的空间若隐若现，但却巨大无比，炽盛夺目，压得苍穹都倒塌了。它非常炫烂，像是一个太阳横在虚空，悬在那里，周围无尽仙气袅绕，它在飞速转动，洒下飞仙般的光雨，极度灿烂。
<br/>　　“九宫！”
<br/>　　“十方！”
<br/>　　紧接着，世界不停，继续颤动，天地开始崩塌，按照九宫十方的格形在演化，世界开始演化成一种九天十地的格式。各方天地之间互有联系，形成一方神秘的世界。天地在轰动，虚空在坍塌，苍穹在颤栗，好似承受不住这般恐怖的威力。
<br/>　　许久之后，当一切都停了下来，一切归于平静，整个光团散去，一道朦朦胧胧，虚虚实实的巨大浩瀚的天地横在虚空之中，里面的东西一切都是那么的神秘，让人看之不清，神念也不能触及到。
<br/>　　之后，虚空震动，那略显虚幻的世界才缓缓隐去，消失于虚空，它就仿佛从来没有出现过一般，人们谁也不能发现他，就算圣人也不能。
<br/>　　元始一阵感应，一道朦胧的世界出现在他心神之中，世界之中的一草一木，一沙一石，每一个生灵，每一缕空气，都在他的掌控之中。世界内的一切，都在他的一念之间。
<br/>　　“嗯？这种感觉……好神奇！”
<br/>　　元始仔细的感应着这方世界，世界初开，一切尽在他的掌控之中，他就仿佛是天地间的唯一主宰。一切都是那么的让人痴迷，让人疯狂。
<br/>　　元始微微一笑，开口说道：
<br/>　　“很好，从此，这方世界便名：灵神界！”
<br/>　　……
