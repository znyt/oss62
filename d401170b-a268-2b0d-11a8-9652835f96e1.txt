        &lt;h3&gt;第三卷第二百一十五章  无能为力&lt;/h3&gt;
<br/>
<br/>        新年的几天假期让大家都放松了心情，只有张家的人列外。
<br/>
<br/>        张函一家三口新年的时候和陈子昂在一起度过的，虽然简单，但是很温馨，有俩个孩子的存在，陈子昂的家里是欢声笑语，但是分别之后，他们都限于一种沉闷的无可奈何之中。
<br/>
<br/>        如果说世界上有后悔药可以吃，如果说他们能够回到过去重新选择，那么他们是不会选择和陆家走得那么近的。
<br/>
<br/>        从知道了陈子昂的孩子是张函的之后，弥青云就不想见到蔡少芬，期间蔡少芬约了她几次，她都以生病为借口推脱了。
<br/>
<br/>        说实话，后悔莫及的心情让她真的和生病了一样，每天感觉浑身无力，病泱泱的。
<br/>
<br/>        新年从陈子昂的家里回来她就睡倒了，连着几天都是高烧不退，有可能是在外面冻着了，也有可能是心病。
<br/>
<br/>        弥青云睡倒了，但是张函和他的父亲张沛尧一刻也没有闲着，年前因为去了二爷家，他们都心思重重，连江俞轩和张倩楠的婚礼都不让举办，让他们感觉到了危机。
<br/>
<br/>        陆玉的姑姑嫁给了吴家的人，吴家的吴润竹嫁给了陆宇霆，这两家是亲上加亲，虽然陆玉嫁给了张函，但是陆玉的心思从来都没有在张家，他们两家只是维持着面子上的和谐。
<br/>
<br/>        “小函，我觉得还是尽早将孩子送走。”张沛尧将张函叫到书房。
<br/>
<br/>        “什么时候将孩子送走，我说了不算，子昂说了算。”张函确实是说了不算。
<br/>
<br/>        “你去和陈子昂商量一下，我觉得越早越好。”
<br/>
<br/>        “我现在觉得不光光是要将孩子送走，爸，我要和陆玉离婚！”
<br/>
<br/>        “离婚，你也得等陆玉同意，等她回来吧？”
<br/>
<br/>        “我可以去甸城找她，另外我怀疑陆家和甸城有着不可告人的秘密。”张函沉默良久才终于鼓起勇气。
<br/>
<br/>        “嗯？”张沛尧看着他的儿子，等着他儿子的下文。
<br/>
<br/>        “还记得闫欢吧？你们告诉了我闫欢的下落，我找了两年，利用我自己手中的权利将闫欢找到，但是我将闫欢找到的时候，闫欢在那个地方实际上是离不开那个村子的，说白了，那么些年闫欢是被软禁的。而你们知道闫欢是怎么到的那个地方，只是你们装死而已！你们的良心不疼吗？”张函说到这里有些激动，他捂住了心口。
<br/>
<br/>        “这么些年，你们也肯定知道一些什么，但是你们都装作视而不见听而不闻，难道不是为虎作伥？”
<br/>
<br/>        “我们没有证据！”张沛尧无力的辩解。
<br/>
<br/>        “证据？这个世界上没有什么是不会留下痕迹的。只是看你愿不愿意去找。我不愿意苟且偷生，但是因为我的爱人和孩子我不得不苟且偷生，曾经你教育我做人要头顶天脚踏地，上对得起大众，下对得起父母和爱人。呵，你做到了吗？”
<br/>
<br/>        “张函，你放肆，有这样和你父亲说话的吗？”
<br/>
<br/>        “别恼羞成怒，我理解你，因为你也是为了你妻儿，为了自己能够安宁的升迁，能够有一番报复，可惜你遇人不淑，断送了自己的前程，夹着尾巴做人，这么些年，你也曾经彻夜不眠，也曾经煎熬过吧？”张函笑着说。
<br/>
<br/>        “你！”张沛尧指着张函的手在颤抖。
<br/>
<br/>        “父亲，你应该觉悟了，因为你没有太多的机会，而你的儿子我也没有太多的机会了，如果陈子昂母子有什么问题的话，这个世界上也就没有张函的存在了。”
<br/>
<br/>        “小函，是我们对不起你，你现在想去干什么就干什么吧，我们不阻止你。”弥青云从外面走进书房，她已经是泪流满面，她儿子的幸福是葬送在她的手中，如果能让他安心的话，如果能挽回一点点错误的话，她为什么不支持他的儿子，为什么要绑架她的儿子呢。
<br/>
<br/>        “沛尧，我们已经错了，我们不能在错下去了，我不能看着陈子昂母子受欺负。”弥青云哭着对张沛尧说。
<br/>
<br/>        “事情还没有坏到你们想象的那种地步，陆家也不知道陈子昂有孩子的事情，最起码目前还不清楚，我们从长计议。”张沛尧冷冰冰的说。
<br/>
<br/>        “小函没有必要去甸城！你干好你的本职工作就好，我们想办法将孩子送出去，找机会和陆家摊牌，反正陆玉这几年也不能生养！”
<br/>
<br/>        “你是不到黄河心不死！”张函摔门而去。
<br/>
<br/>        弥青云看着张沛尧：“沛尧，你变了！”
<br/>
<br/>        张沛尧：“我没有变，甸城是个什么样子我是清楚的，秦家就是陆家一手提报起来的，小函在这里怎么折腾都行，到了甸城，就不是你我说了算！”
<br/>
<br/>        弥青云：“这么说你是有打算的了？”
<br/>
<br/>        张沛尧：“什么打算都没有，只能走一步看一步。”
<br/>
<br/>        弥青云：“那小函怎么办？”
<br/>
<br/>        张沛尧：“随他折腾吧，我阻止不了他，我们能做的是悄悄的给他提供资然，但是你还得和陆家维持好关系。”
<br/>
<br/>        弥青云：“蔡少芬约了我好多次，我一直都没有出去，你这样一说，我还真得见见她。”
<br/>
<br/>        张沛尧：“如果蔡少芬问起孩子的事情，你只能推到江家那个小子的身上。至于怎么说，你看着办。”
<br/>
<br/>        弥青云长长的叹了一口气，俩个人一时沉默，相顾无言。
<br/>
<br/>        张函从张沛尧的书房出来心里悲愤不已，他说服不了他的父亲和他站在一条线上，那么就让他单打独斗吧！
<br/>
<br/>        拿起手机给陈子昂打电话，问问陈子昂在不在北京，他想见她们母子了。
<br/>
<br/>        电话响了好久，陈子昂才接起电话：“你好！”客气而疏离的语气让张函心里莫名其妙的一慌。
<br/>
<br/>        “子昂，我想见见你们。”
<br/>
<br/>        “张总，如果你真的为我们母子着想的话，请不要再来骚扰我们了！我们原本很安宁的生活着，可是你利用江俞轩一步一步的接近我，如今我和我的孩子不得不分开，你又何苦呢？你觉得我和你还有将来吗？请张总自重！孩子并不是你的，你不要自作多情了！”陈子昂那冰冷的语气直击张函的心脏，张函捂着电话，心口处疼得无法呼吸，他跌跌撞撞的走到书房，倒了一把药和着泪水塞进嘴里。
<br/>
<br/>        房间的床上都是摆满着他偷拍陈子昂和孩子的照片，陈子昂和孩子的笑容是多么的美丽啊！他们娘三个一直都是他支撑下去的信念，无论他们见不见他，他们都在他的心中！
<br/>
<br/>        这个世界应该是晴朗的光明的，他要和陈子寒一起去奋斗，他要揭掉那些虚伪的面孔隐藏着的可恶嘴脸！
