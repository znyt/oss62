　　“你认为这东西有问题？”
<br/>　　赵天宇拿着平板电脑，手指滑动着快速浏览着关于“crystal-ball”和“维伦恩”生物制药公司的信息。
<br/>　　网页上的信息写得很清楚，“crystal-ball”是“维伦恩”生物制药公司历时多年研制出来，用以解决男人难言之隐的保健品，其效果优于其他同类商品，并且已经通过了相关检测，获得了在北美洲销售资格。
<br/>　　从网页上看，根本看不出有什么问题。
<br/>　　“杭山市刑侦队那边认为，这个‘crystal-ball’可能是一种全新的，我们所未知的合成型致幻剂。”慕容飞说道：“目前为止，只有李霄力在杭山市的‘碧海潮生’夜总会以拍卖的形式在销售，场面异常的火爆，知道这一粒小珠子最高的拍卖价格是多少吗？55000块！很难以想象是吧？但事实就是发生了。”
<br/>　　“这的确是有些异常。”罗昊皱眉道：“就算这东西的效果再神奇，市场上还是有同类商品存在，就算是效果不如‘crystal-ball’，但也不至于不纳入人的考虑中，而只是一味追求‘crystal-ball’，不惜以高价竞拍！”
<br/>　　“杭山市警方那边也是同样的意思，杭山市缉毒处觉得这些疯狂竞拍‘crystal-ball’的人就像是犯了毒瘾的人。”
<br/>　　慕容飞点了点头，说道：“本来我也没放在心上，回去后，跟千雪说起这件事情，她也觉得不可思议，她认为这样一种可有可无的保健品能让人表现的如此疯狂，实在是难以想象！所以我在网上就查了关于‘crystal-ball’的消息，知道了这家‘维伦恩’生物制药公司。”
<br/>　　“玛丽科娃在北美洲，而‘维伦恩’生物制药公司也在北美洲。”云熙黛眉轻蹙，对慕容飞说道：“你是认为这家‘维伦恩’生物制药公司跟玛丽科娃有关系？”
<br/>　　“玛丽科娃有着成熟的制毒、贩毒、运输网络，但是到目前为止却没有人知道玛丽科娃的制毒工场在什么地方，如果把一家制毒工场伪装成正规的，生产保健品的生物制药公司，是不是可以很好的掩人耳目？”慕容飞说出自己的猜测和想法，道：“如果这个‘crystal-ball’是一种全新的，未知的合成型致幻剂，那么整条线就可以串联起来，这个‘维伦恩’生物制药公司就是一个被伪装起来的大型制毒工场，他们有专业的设备和团队，集研发、生产制作、销售于一体，而杭山市的李霄力则是这家‘维伦恩’生物制药公司的下家，大胆猜测一下，李霄力手中的‘crystal-ball’只是一批试验品，是用来测试这个市场的，如果市场反响好，就会大批量的生产并且销售。”
<br/>　　“而最重要的，就是这个‘维伦恩’生物制药公司的实际控制人很有可能就是玛丽科娃！”罗昊补充道。
<br/>　　“没错！”
<br/>　　“也就是说，这个李霄力现在是我们的突破口！”云岳站起身来说道：“只要确定他这个人身上不干净，就能联合警方采取行动，进而从他身上打开突破口！”
<br/>　　“我会跟杭山市警方进行联系，了解详细情况，你们随时做好行动准备，很有可能要去杭山市一趟了。”
<br/>　　赵天宇对罗昊、慕容飞和云熙下达了做好行动准备的命令。
<br/>　　……
<br/>　　南美洲，“cocaine”王国，布克尔特所控制的木料厂。
<br/>　　这里是布克尔特所控制的，用来运送高纯度cocaine的地点之一。生产包装好的高纯度cocaine会被运到这个木材厂，这里的工作人员会把圆木中间挖空，然后把一块一块cocaine装进这些圆木中，然后通过远洋货轮，以出口木材的名义，运送到世界各地。
<br/>　　这次被装运出去的cocaine中，有十公斤是要送到杭山市，交给李霄力的，这是李霄力提供关于佟千雪的消息，布克尔特给他的报酬。
<br/>　　布克尔特看着把四块高纯度cocaine装入到圆木中，然后把圆木上缺口封盖起来的工人，拔出挂在自己腰间的兰博军刀，在那根圆木上刻了一个倒三角记号，随后对站在一旁的Tim招了招手，说道：“通过普提查把消息透露给中国内地的警方。”
<br/>　　Tim跟在布克尔特身边多年，自然是明白布克尔特的用意，“你是想要李霄力在接货的时候，被警察人赃并获？”
<br/>　　“李霄力有句话说得不错，蛋糕就这么大，想要自己吃多吃饱，只有去把别人手里的蛋糕抢过来。”布克尔特眼睛中寒芒闪烁，说道：“那么什么样的办法才能一劳永逸？就是把这块蛋糕原来的主人干掉，这样他就不会再回来把被抢走的蛋糕再抢回去。”
<br/>　　“可是‘毒皇后’那边？”Tim有些迟疑地问道。
<br/>　　“据说他们内部发生了一些事情，那个女人现在自己的事情都够她麻烦的，她现在可没有心思去管李霄力。”布克尔特显然是得到了一些消息，说道：“而且对李霄力动手的是中国警方，玛丽科娃胆子再大，也不敢去撸老虎的胡须。更何况，玛丽科娃的下线可不止李霄力一个，少了他一个，并不会对玛丽科娃所控制的市场造成多大的影响，最多再扶植一个就是了。等着吧，玛丽科娃那女人的麻烦还没开始呢。”
<br/>　　“我知道该怎么做了，我立刻就去安排。”Tim点头应下，转身离开去安排相关事宜。
<br/>　　……
<br/>　　杭山市市公安局，局长办公室。
<br/>　　刑侦队队长周宋捷和缉毒队队长宋冬飞，正在向省公安厅的领导汇报有关于李霄力，还有李霄力拍卖“crystal-ball”的事情。
<br/>　　龙隐部队不能直接插手地方上的事情，所以赵天宇只能通过其他途径去获取消息，正好省公安厅厅长是赵天宇以前在东南军区时的战友，对方在问清楚了一些情况后，立刻就安排人员下去了解情况。
<br/>　　“情况我们已经了解了。”省公安厅的领导合上笔记本，对周宋捷、宋冬飞还有市公安局局长唐宇说道：“这个李霄力很有可能跟军方正在追查的一个神秘组织有关系，有新的情况请立刻汇报，不要私自行动！”
