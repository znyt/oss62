　　难道这才是最贵重的东西？
<br/>　　梧桐很好奇里面机会有什么，于是抓住卷轴的一头，想要拿出来，发现卷轴被卡得很紧，于是用力去拉，只听咔嚓一声，好像里面有什么东西被拉动了，卷轴总算被拉了出来。
<br/>　　梧桐也是一头雾水，刚要把油纸包打开，突然听见一阵轰隆隆的响动，从身后石门的方向传来，紧接着就是哐当一声巨响，震得房间里一阵晃动，同时产生了一股强大的气流，将两盏鲛油灯也吹灭了。
<br/>　　眼前一片漆黑。几秒钟之后，叶少阳才回过神来，黑暗中一个人都看不见，于是大声喊道：“你们都还好吗？”
<br/>　　“我没事。”
<br/>　　“我也没事。”
<br/>　　是梧桐和妙心的声音。
<br/>　　叶少阳把手中的鲛油灯点起来，朝周围的人影照过去，这才发现大伙都在，很快另一盏鲛油灯也被点亮，石室里这才明亮起来。
<br/>　　“刚才发生什么事了？”大伙面面相觑。
<br/>　　妙心拿过一盏鲛油灯，走出外间，朝之前那声巨响发出的方向走过去。叶少阳等人追出去的时候，她已经站在石门前，呆呆地一动不动。
<br/>　　“怎么了？”梧桐忙问。
<br/>　　“石门关上了……”妙心喃喃说道。
<br/>　　大伙震惊，纷纷冲过去，果然，之前打开的石门在完全关闭了。
<br/>　　愣了一下之后，大伙开始检查之前的门锁，于是发现了一个令人震惊的事实：门锁不见了！
<br/>　　之前被他们挖出来的缺口还在，但是不管是往外还是往里推，石门都纹丝不动……叶少阳把手插进石槽里，摸索了半天，发现石门好像比之前厚多了，但是……为什么中间多了一道缝隙？
<br/>　　很快，叶少阳触电般的想到了什么，把手缩回来，回头望着众人，用一种震惊而无奈的口吻说道：“多了一道门！”
<br/>　　“什么意思？”毛小方问。
<br/>　　“多了一道门，在之前那道门的后面，又多了一道门，这道门没有门锁，完全动不了……”
<br/>　　大伙一听，都愣住了，挨个上去检查了一下，的确跟叶少阳说的一样，也就是说，现在将他们困在这里的门并不是之前那个，而是凭空多出来的。
<br/>　　一番检查之后，大伙发现，里面这道石门很可能是一直悬在空中的，通过某种机关控制，现在放了下来，挡住了石室唯一的出口。
<br/>　　“可能是从上面落下来的，这叫悬门，是古墓里比较罕见的机关。”妙心说道，“不过这机关也很实用，靠外力很难弄得开。”
<br/>　　卢晓清一听，皱眉说道：“我们也没碰什么机关吧，为什么这悬门就突然放下了，之前为什么没落下来。”
<br/>　　梧桐低头，咬着嘴唇说道：“是我的缘故……我刚才拉这幅卷轴，明显感到有什么东西连在卷轴上，我一用力，那个东西好像就断了，然后……石门就掉下来了。”
<br/>　　众人愕然。
<br/>　　妙心叹了口气道：“怪不得了，我明白了……那卷轴就是控制悬门的机关，一旦被拉开，悬门就会落下……”
<br/>　　大伙怔了怔，卢晓清道：“那些东西，都是墓主人给他后人留的吧，这石碑上也写了，既然是这样，为什么还要留下这东西？难道不怕把他子孙后代也害死在这里？”
<br/>　　叶少阳接过话头，说道：“跟你说的恰恰相反……既然墓主人将钥匙也留给他后人了，不可能没有别的交代，或者已经特意嘱咐过，那个石槽里的卷轴中，有一个不能动，如果动了，石门就会掉下来，把人封在石室里……这么做是为了防止盗墓贼。”
<br/>　　说到这里，毛小方已经听懂了，抢着说道：“你是说，他祖先给他留下钥匙的同时，肯定也留下了这个提示，但是盗墓贼不知道，就算是通过别的方法创进来，最后也会被这道石门困死，最后什么东西都带不走……”
<br/>　　“没错，那个石槽里有那么多卷轴，拿到前面的，全都是价值连城的字画，最后一个当然也不会放过，再说前面那些都没事，人早就放松警惕了，谁会想到，最后一个卷轴联通着机关……”
<br/>　　说到这，转头对梧桐说道：“这件事真不怪你，换成谁看到那个卷轴，结果都会跟你一样的。”
<br/>　　大伙暗地想了下，假如换成自己，一定也会毫不犹豫地把卷轴拉出来……这是修墓之人钻研人心的结果，又巧妙得设计了机关，想不中招都不可能。
<br/>　　“那我们现在怎么办，这里没有别的出口，我们会被活活憋死在这里的！”陈晓宇有些激动地嚷起来。
<br/>　　他这么一嚷，也在大伙心中造成了阴影，一时间面面相觑，最后叶少阳重新来到石门前，看到上面被挖出来的石槽，顿时心安，说道：“这里有孔可以透气，不会被憋死。”
<br/>　　“那又怎么样，我们出不去，还不是会憋死在里面！”陈晓宇吼起来。
<br/>　　叶少阳也无言以对。
<br/>　　大伙一起来到石门前，再次尝试用力推石门，结果自然还是纹丝不动，其实根本不用这么尝试：当初修墓的人费尽心思，弄了这么一个机关，如果石门能轻易打开，那人家真是白玩了。
<br/>　　叶少阳尝试继续抠那个之前挖出来的缺口，但是周边的岩石都很结实，用手根本就抠不开。大伙急得不行，最后没办法，拿着鲛油灯，趴在墙上，几乎是一寸一寸地把两间石室的墙壁摸索了一遍，结果自然是没有一个地方能出去的。
<br/>　　“没办法了，看来要饿死在这里了。”毛小方泄气地往地上一躺，无心朝天，自从下墓之后一直折腾，大家都很累了。
<br/>　　叶少阳等人也都坐下来，开始思考还有什么办法能够出去。陈晓宇情绪有些激动，绕着石室走来走去，大喊大叫的，也没人去理会。
<br/>　　要是有炸药就好了……
<br/>　　叶少阳不禁产生了浮想，只需要一丁点炸药，塞到之前挖出的缺口里，定点爆破，很容易就能炸出一个供人出去的缺口……
<br/>　　“我有一个办法！”妙心突然开口，打断了叶少阳的胡思乱想，一时间大伙都看着她。
<br/>(本章完)
